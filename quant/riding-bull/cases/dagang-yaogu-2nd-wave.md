# 大港股份——妖股二波实战案例

> 妖股二波启动的完整跟踪案例。核心看**集合竞价盘口**和**放量承接**两个信号。

---

## 案例背景

大港股份在第一波拉升后回调，通过盘口语言和二波启动信号确认第二波行情。

---

## 关键技术点

### 1. 集合竞价盘口语言

| 位置 | 含义 |
|:----:|:------|
| **上方红柱** | 想买但**未成交**的人——越多越好 |
| **上方绿柱** | 想卖但**未成交**的人 |
| **下方红柱** | 已经**成交**的买入 |
| **下方绿柱** | 已经**成交**的卖出 |

**判定**：下方已成交量大、上方未成交红柱多 = 强势开盘。

### 2. 9:33开盘定式

早盘9:33分的关键位置：低开高走摸2-1位才有生命力。

### 3. 放量承接信号

第一波拉升后出现**放量不跌破** = 有第二波。

### 4. 妖股启动形态

```
第一波拉升 → 回调（放量不破底）
  → 出现放量承接K线（画龙点睛）
  → 缩量涨停 → 妖性启动
```

### 5. 风控位

18.10元——跌破则妖性消失。

---

## 操作流程

```
① 复盘发现：在妖股池中发现大港股份
② 确认：前4个涨停板不看→只看成交量对等（1=2，有第二波）
③ 放量承接K线=画龙点睛
④ 缩量涨停=妖性启动
⑤ 风控=18.10
⑥ 预期：挑战前高，创新高
```

---

## 结果

- 妖股二波启动确认
- 空间技术测量目标位：21.8 / 24.4 / 27
