# 骑牛登山量化框架设计方案

> 基于40+篇技术手册，将骑牛登山的信号体系量化为一套可回测、可实时选股、可择时的工程化框架。

---

## 一、设计原则

### 1.1 框架定位

```
量价时空传承（骑牛登山技术体系）
         ↓
    因子化 → 可回测
    规则化 → 可编程
    分级化 → 可择时
```

### 1.2 三种使用场景

| 场景 | 触发 | 数据范围 | 输出 |
|:----:|:----|:---------|:-----|
| **回测** | 研究某个信号的历史表现 | 全历史（2010-至今） | 收益/胜率/最大回撤等 |
| **选股**（日频） | 每日收盘后 | 全市场当天 | 分级股票池（精选/核心/基础） |
| **择时**（盘中/日频） | 盘后或盘前 | 大盘指数 + 板块 | 仓位建议 / 市场状态分类 |

### 1.3 技术栈
- 数据：现有邢不行股票日线CSV `/mnt/c/b/stockdata/stock-trading-data/`
- 计算：Python pandas/numpy，向量化优先
- 因子：复用现有 `select-stock-pro` 框架中的因子函数
- 部署：Cron 每日盘后自动运行

## 二、信号/指标层（可量化的一切骑牛技术）

### 2.1 均线系统（对应5篇手册）

| 指标 | 计算方式 | 用途 |
|:----:|:---------|:-----|
| MA方向 | 144日线斜率（线性回归/差分），共3档：向上>0.5%/走平±0.5%/向下<-0.5% | 方向判定（第一层） |
| 均线粘合度 | 5/10/25日线间距归一化 = (max-min)/close | 收拢/发散判定（第二层） |
| 均线排列 | 5>10>25>60>144=多头，反向=空头，交叉=纠缠 | 趋势强度 |
| MA25乖离 | (close-MA25)/MA25 | 线上阴线买/线下阳线抛 |
| 一阳穿多线 | 单日K线同时穿透[5,10,25,60]中至少2条 | 第四层价格反转信号 |

### 2.2 MACD系统（对应3篇手册）

| 指标 | 计算方式 | 用途 |
|:----:|:---------|:-----|
| MACD(12,26,9) | 标准EMA差离值 | 基础指标 |
| MACD柱级别 | 归一化MACD柱 / close × 100，全市场分位数：大绿(<33%)/中绿/小绿/小红/中红/大红(>66%) | 八字要诀级别判定 |
| MACD浪型 | MACD柱连续N根上升=红浪，连续N根下降=绿浪 | 三浪/五浪位置 |
| MACD区间 | MACD柱高度分3档：第一区间(大红)/第二(中红)/第三(小红) | 选股强度 |
| 零轴位置 | DIF >0 为水上，DIF<0 为水下 | 趋势性质 |
| 金叉/死叉 | DIF上穿/下穿DEA | 买卖点信号 |
| 底背离 | 价格创新低 + MACD柱(或DIF)未创新低 | 经典买入信号 |
| 顶背离 | 价格创新高 + MACD柱(或DIF)未创新高 | 卖出信号 |
| 上水回踩 | DIF>0后回踩不破0轴 = 三浪买点 | 第三层级别判定 |

### 2.3 成交量系统（对应3篇手册）

| 指标 | 计算方式 | 用途 |
|:----:|:---------|:-----|
| 倍量 | vol / vol[-1] ≥ 2.0 | 异动量基本形态 |
| 脉冲量 | vol > MA(vol,20) × 3.0，且前20天无类似放量 | 异动量（常态跌） |
| 组合量/移动量 | 连续3天vol递增，且最后一天vol > vol[-1] × 1.5 | 最可靠异动量 |
| 右量过左量 | 对比左方最近一个天量的水平线，当前量能突破该线 | 主升浪信号 |
| 量承接 | 放量阴后放量阳收复 = 增量资金入场 | 半山腰买入法 |
| 价承接 | 缩量至地量后不破前低 | 底部买入法 |
| 瀑布杀 | 盘中急跌7%+后收长下影，且分时无该价格 | 老鼠仓买点 |
| 底部尖角波 | 分时突然拉升后回落，底部附近 | 主力进场信号 |

### 2.4 K线形态系统（对应10篇手册）

| 指标 | 定义 | 用途 |
|:----:|::----|:-----|
| 长阴加长腿 | 阴线实体≥5% + 下影线≥实体1/2 | 下跌到位信号 |
| 阳包阴 | 阳线实体完全覆盖前一根阴线实体 | 反转K线 |
| 次阳过阴半 | 阳线收盘站上前阴线2分之1位 | 企稳信号 |
| 一阳穿多线 | 单阳线同时穿透多条均线 | 强势反转 |
| 阴线平台突破 | 突破最近最高阴线的开盘价 | 支撑压力判定 |
| ABC结构 | 下跌→横盘→下跌完整3段 | 底部结构判定 |
| 黄金坑 | 跌破前低后快速放量站回 | V型反转信号 |
| 斜横线突破 | 突破两个高点连线的下降趋势线 | 趋势反转 |
| 平行线突破 | 突破阴实顶水平压力线 | 突破确认 |
| l联动效应 | 板块内龙头连板身位判定 | 短线情绪 |

### 2.5 综合指标体系（五层架构）

```
方向因子 = MA144方向（向上=1/走平=0/向下=-1）
粘合因子 = 均线间距归一化（越小越收拢）
级别因子 = MACD月/周/日金叉状态（3级权重）
价格因子 = 阳价>阴价天数占比（最近5天）
量能因子 = 右量/左量比值
```

## 三、选股体系（每日盘后）

### 3.1 三级股票池架构（复用现有骑牛登山池）

```
基础池（~800只）
  条件：
    月线MACD红柱 AND 周线MACD红柱
    AND MA25 > MA60
    AND MA60向上（ma60[-1] > ma60[-5]）
    AND 上市满1年
  含义：大方向没问题，可操作池

核心池（~400只）
  条件：
    基础池基础上
    AND 月+周MACD均为中红或大红
    AND MA25斜率 > 0（25日线向上）
  含义：趋势明确，优先关注

精选池（~200只）
  条件：
    核心池基础上
    AND 距MA200 > 20%（长期趋势确立）
    AND MA25/MA60间距 > 5%（均线敞口够大）
    AND 月线MACD大红（大级别动量最强）
  含义：最可能跑赢大盘

妖股池（<50只，新增）
  条件：
    连板+换手板+流通盘小
    + 月线红肥绿瘦（阳量高阴量缩）
    + 有中线翻倍基础或历史天量
  含义：短线打板候选
```

### 3.2 买入信号评分卡

对精选池每只股票逐项打分（每项0或1分，共10项）：

```
□ MA144向上（方向正确）
□ 均线收拢（粘合度为近60日最低20%分位）
□ MACD为月线级别红柱（级别大）
□ 出现阳包阴（价格反转）
□ 右量过左量（量能突破）
□ 异动量出现（倍量/移动量）
□ 长阴加长腿（下跌到位）
□ 次阳过阴半（企稳确认）
□ ABC下跌结构完成
□ 周线金叉支撑
```

**评分结果**：8-10分=强推，5-7分=关注，0-4分=放弃

## 四、择时体系（每日盘前）

### 4.1 大盘状态分类

```
使用上证指数 和 国证2000：

      MA60方向    MA25位置    MACD          状态
上涨     ↑          线上       水上金叉      强牛
震荡     →          线附近     零轴附近      震荡
下跌     ↓          线下       水下死叉      熊市
反弹     ↑          线下       水下金叉      反弹
回调     ↓          线上       水上死叉      回调
底部     →          线下深     水下大绿      底部区域
```

### 4.2 仓位建议

| 大盘状态 | 建议仓位 | 池子 |
|:--------:|:--------:|:----:|
| 强牛 | 80-100% | 精选池+妖股池 |
| 震荡 | 40-60% | 核心池 |
| 反弹 | 30-50% | 精选池 |
| 回调 | 20-30% | 精选池中距MA200>20%的 |
| 熊市 | 0-10% | 观望，等底背离信号 |
| 底部区域 | 50-70% | 核心池（抄底） |

### 4.3 每日信号汇总

```
盘后（15:00收盘后）：
  ① 更新MACD/均线/量指标
  ② 运行三级筛选池
  ③ 运行买入评分卡
  ④ 输出评级列表（强推/关注/放弃）

盘前（9:00开盘前）：
  ① 更新大盘状态
  ② 输出仓位建议
  ③ 输出当日关注清单
```

## 五、回测体系

### 5.1 回测信号类型

| 类型 | 买入条件 | 卖出条件 | 风控 |
|:----:|:---------|:---------|:----:|
| **ABC反弹** | ABC下跌完成+阳包阴+异动量 | 空间到位 (0.5位) 或顶背离 | 阴实底 |
| **144翻倍** | MA144向上+均线收拢+过左峰 | 跌破MA144或顶背离 | MA144 |
| **异动量买入** | 异动量出现+回踩不破+阳包阴 | 跌破异动量最低价 | 异动量最低价 |
| **黄金坑** | 跌破后放量站回 | 空间到位或二次破位 | 黄金坑边缘 |
| **妖股二波** | 中线翻倍+调整+板块共振 | 放天量 | 起涨点 |
| **MACD第一区间** | 大红+增量资金+阳包阴 | 顶背离或天量 | 买入K线最低价 |

### 5.2 回测框架设计

```python
# 伪代码结构
def backtest_signal(signal_config, start_date, end_date):
    # 1. 加载全市场日线数据
    # 2. 计算所有基础指标（MA/MACD/Volume）
    # 3. 检测所有信号
    # 4. 计算每个信号的forward_returns (1d/3d/5d/10d/20d)
    # 5. 统计汇总（胜率/平均收益/最大回撤/夏普）
    # 6. 分层分析（A层→B层→C层）
```

### 5.3 评价指标

| 指标 | 计算方式 |
|:----:|:---------|
| 胜率 | 正收益信号数 / 总信号数 |
| 平均收益 | 所有信号的平均return |
| 最大回撤 | 信号组合的峰值到谷值跌幅 |
| 盈亏比 | 平均盈利 / 平均亏损 |
| Sharpe | (平均收益 - 无风险) / 标准差 |
| 分箱单调性 | Q10-Q1差距，Q1→Q10单调趋势 |

## 六、工程架构

### 6.1 目录结构

```
riding-bull-quant/
├── data/
│   └── loaders.py            # 数据加载（复用现有load_stock_csv）
├── factors/                   # 因子计算
│   ├── ma.py                 # 均线系统
│   ├── macd.py               # MACD系统
│   ├── volume.py             # 成交量系统
│   ├── patterns.py           # K线形态
│   └── comprehensive.py      # 综合信号（评分卡）
├── screening/                 # 选股
│   ├── pool_basic.py         # 基础池
│   ├── pool_core.py          # 核心池
│   ├── pool_elite.py         # 精选池
│   └── pool_yaogu.py         # 妖股池
├── timing/                    # 择时
│   └── regime.py             # 大盘状态 & 仓位建议
├── backtesting/               # 回测
│   ├── backtest_runner.py    # 回测引擎
│   ├── signals/              # 各信号检测脚本
│   │   ├── abc_reversal.py   # ABC反弹信号
│   │   ├── abnormal_volume.py# 异动量信号
│   │   ├── golden_pit.py     # 黄金坑信号
│   │   ├── golden_hook.py    # 黄金钩信号
│   │   └── first_zone.py     # MACD第一区间信号
│   └── evaluation.py         # 评价指标
├── daily/                     # 每日运行
│   ├── daily_report.py       # 每日报告生成
│   └── cron_jobs.py          # Cron任务定义
├── scripts/                   # 辅助脚本
└── config.py                  # 全局配置
```

### 6.2 运行模式

```
# 盘后选股
python -m daily.daily_report --mode pool

# 信号回测
python -m backtesting.backtest_runner --signal abc_reversal --start 2018-01-01

# 研究一个因子
python -m backtesting.factor_analysis --factor ma_convergence --bins 10

# 实时大盘状态
python -m timing.regime --update
```
