# 资金箱体（MACD资金箱体/起爆线）

> 基于MACD白色线（DIF）数值绘制的资金箱体，因精准定位主升浪启动点（起爆线）而闻名。由资金（成交量）堆出来的箱体，非自然箱体。

---

## 核心定义

**前提条件**：找到一波**圆滑**的MACD走势

- "圆滑" = MACD白色线（DIF）走势清晰，**中间没有明显波动打断**
- 上涨或下跌均可
- 只看白色线（DIF）的数值，不看黄色线（DEA）或红绿柱
- 白色线数值会随着光标在软件中显示

---

## 取点规则（决定箱体上下边界）

### 上位（起爆线）— 取最高值

在一波圆滑上涨中，找到白色线数值最高的那根K线：

| 最高值对应K线 | 取价方式 |
|:---:|:---:|
| **阴线**（收盘价 < 开盘价） | 取**开盘价** |
| **阳线**（收盘价 ≥ 开盘价） | 取**收盘价** |

> ⚠️ **必须取第一次出现的最大值**。同一数值出现多次，取第一次。

### 下位（箱底线）— 取最低值

在同一波圆滑下跌（或紧邻的下跌段）中，找到白色线数值最低的那根K线：

| 最低值对应K线 | 取价方式 |
|:---:|:---:|
| **阴线** | 取**收盘价** |
| **阳线** | 取**开盘价** |

> ⚠️ 阴线/阳线的取价规则上下**相反**，不可记混。

### 取点口诀

| 位置 | K线形态 | 取值 |
|------|---------|------|
| 顶部（最大） | 阴线 | 开盘价 |
| 顶部（最大） | 阳线 | 收盘价 |
| 底部（最小） | 阴线 | 收盘价 |
| 底部（最小） | 阳线 | 开盘价 |

---

## 起爆线（核心）

**起爆线 = 箱体上边界**，是抓主升浪启动点的关键。

当股价运行到起爆线位置时，会出现以下四种突破形态之一：

| # | 突破形态 | 说明 |
|:-:|---------|------|
| 1️⃣ | **跳空高开** | 直接跳空过起爆线 |
| 2️⃣ | **涨停板** | 涨停板突破 |
| 3️⃣ | **倍量** | 当日成交量是昨日的2倍以上 |
| 4️⃣ | **线上阴线买** | 回踩起爆线收阴线，阴线买入 |

> 四种形态出现任一即可买入。**最常见的是起爆线位置直接跳空高开+涨停。**

---

## 箱体测幅

将箱体上下边界之间的空间设为 **1格**（100%）：

```
箱体 = 起爆线（上）—— 箱底线（下）
       ↓
用百分比线将箱体一分为二
       ↓
第一格 = 原始箱体高度（100%）
第二格 = 箱体高度×1.5（一又二分之一）
第三格 = 箱体高度×2（200%）
```

- 股价突破起爆线后，第一目标位 = 起爆线 + 一整个箱体高度
- 存在密集压力区，需配合MACD顶背离判断卖点

---

## 配合MACD使用

资金箱体买点 + MACD十大理论卖点 = **完整波段操作体系**

| 阶段 | 技术 | 动作 |
|------|------|------|
| 买入 | 资金箱体起爆线突破四种形态 | 买入 |
| 持仓 | MACD三浪/五浪结构判断 | 持筹 |
| 卖出 | MACD顶背离（A点→B点→C点/死亡勾） | 卖出 |

### 识别卖点示例

```
起爆线买入后 → 股价运行至箱体第二格附近
  → MACD出现：
    ① A1: 白色线高点 → A2: 白色线次高点
    ② B点: 白色线低于A点（线型顶背离）
    ③ C点/死亡勾: 确认顶背离
  → 卖出
```

---

## 灵活用法

当MACD走势不够圆滑、无法清晰取点时：

- **只取起爆线（上位）**，不要箱底线
- 用起爆线判断突破买点，用**自然箱体**判断目标位
- 阳线突破时要求**一半以上**实体越过起爆线——仅冒头不算有效突破

---

## 实战案例

### 辽宁成大
- 圆滑上涨段 → 取白色线最大值：0.11（阳线）→ 起爆线 = 收盘价12.27
- 股价到起爆线位置 → 跳空高开涨停突破
- 箱体底部取最低值 → 百分比测幅 → 精确涨到箱体1.5倍位置见顶

### 传媒股
- 白线最高0.10（阳线）→ 起爆线 = 收盘价
- 精准回踩起爆线 → 涨停板突破（线上阴线买）
- 完美回踩后上攻

### 山西焦化
- 三浪/五浪结构的最后一浪用箱体测幅
- 起爆线突破→涨到箱体第二格→MACD顶背离→ABC死亡勾→卖出
- 单波盈利约24%
