# 黄金线/定海神针（超短线反包战法）

> 强势股急跌后的超短线反包技术。三种形态：**黄金线（连板后跌停反包）**、**定海神针（下跌趋势中的放量阳线）**、**补量型（缩量跌停放量未跌停+阳包阴）**。持股2-3天，风控明确。

---

## 三种形态

### 形态一：黄金线（连板后跌停反包）⭐⭐⭐

```
连板上涨 → 缩量跌停 → 放量低开高走阳线（黄金线）
```

**特征**：
- 左边必须有连续涨停板（强势股）
- 跌停板**缩量**（无量跌停）
- 黄金线**放量**（量明显大于跌停板）
- 低开高走带上影线或实体阳线

### 形态二：定海神针（下跌趋势放量阳线）⭐⭐⭐

```
下跌趋势 → 持续下跌 → 低开高走放量阳K线（定海神针）
```

**特征**：
- 可以是长上影，也可以是实体阳线
- 必须是**放量的**
- 最低价未来不能被跌破

### 形态三：补量型（连板后缩量跌停放量未跌停+阳包阴）⭐⭐⭐⭐

```
连板上涨 → 缩量跌停 → 第二天放量未跌停（低开高走）
            → 第三天阳包阴（也是放量）
```

**特征**：
- 跌停板缩量
- 第二天不跌停但放量
- 第三天阳包阴（量可等于或大于前一天）
- 妖股接力中常见这种形态

---

## 操作规则

### 买入

**买入时机**：黄金线/定海神针出现的**次日**买入。

买入日不论收阴收阳都行，关键是未来收盘价**不跌破黄金线的最低价**。

### 风控

```
黄金线/定海神针那根K线的最低价 = 风控位
收盘价跌破 → 无条件卖出
```

### 卖出

- 持股 **2-3天**，短线操作
- 涨停或冲高就可以卖
- 结合空间技术（自然箱体/对偶）精确卖点

---

## 配合条件

| 条件 | 说明 |
|:----:|:-----|
| **MACD** | 最好在**第一区间**（零轴上强势区） |
| **周线** | 周线金叉/红柱有推升作用 |
| **量能** | 黄金线量必须 > 跌停板量 |
| **位置** | 越低越安全，下跌中继要小心 |

### 下跌中继的警告

如果黄金线出现在**下跌中继**位置（非底部）：
- 仓位一定要小
- 持股不超过2天
- 跌破风控立刻走
