# 筹码峰ABC（解放战法/低开阴线定位）

> 用筹码峰的移动判断主力资金动向。**低开阴线**位置 = 筹码峰最大的压力位。**右侧放量 > 左侧** = 解放完成，主升浪启动（不超过5天）。

---

## 一、核心法则

### 低开阴线 = 最强压力位

> 涨了一波后出现的**低开阴线**，其开盘价/收盘价 = 未来最强的压力位。

未来股价到此位置的两种走法：
- **高开** → 突破（强势）
- **低开** → 回调（弱势）

### 筹码峰最大 = 最大压力

筹码峰最尖（最高）的位置 = 套牢盘最集中的位置 = 最强压力。
解放这个位置 = 主力真金白银买入。

---

## 二、解放信号——右侧量 > 左侧量

> **所有V型反转，右侧拉升量必须大于左侧下跌量。**

这是判断解放是否真实的关键指标。

| 现象 | 含义 |
|:----:|:------|
| 右侧量 > 左侧量 | ✅ 真解放，主力进场 |
| 右侧量 < 左侧量 | ❌ 假反弹，还会跌回去 |

---

## 三、三种筹码峰形态

### 形态一：逐步解放型（慢做底）

```
高位筹码峰最大 → 下跌
  → 第1次解放（回抽到某一峰）
  → 回落 → 第2次解放（更高）
  → 回落 → 第3次解放（突破）
  → 回踩 → 主升浪
```

**特征**：一个峰一个峰解放，逐步推进。底部构建时间长。

### 形态二：V型直接突破型

```
高位筹码峰最大 → 快速下跌
  → V型反转 → 一口气突破所有筹码峰
  → 回踩不破 → 主升浪
```

**特征**：右侧量必须明显大于左侧。一口气解放多峰。

### 形态三：两波解放+回踩型

```
高位筹码峰 → 下跌
  → 一波拉升解放两个峰
  → 回踩（缩量不破）
  → 再次上行 → 主升浪
```

**特征**：一波解放两峰，量是关键牵引车。跳空高开启动。

## 四、操作流程

### 看筹码峰三板斧

```
第一步：找到当前最大的筹码峰（最尖的位置）
第二步：画出该峰对应的价格线
第三步：观察股价向这个位置的运行
  → 低开 → 压力（先卖出）
  → 高开 → 突破（买入）
```

### 解放完成的判断

```
右侧放量 > 左侧量
  + 股价站上筹码峰最高价
  + 回踩不破
  = 解放完成 → 买入

买入后 → 一般不超过5天主升浪启动
```

## 五、实战案例

### 海马汽车（逐步解放型）
- 高位筹码峰最大 → 每次反弹解放一个峰
- 低开阴线位置 = 每次的压力位
- 最终放量突破 → 主升浪

### 环宇数控（V型直接突破型）
- 筹码峰最大 → V反 → 右侧量 > 左侧
- 低开阴线定位压力
- 一口气突破回踩不破 → 主升浪

### 传智教育（两波解放+回踩型）
- 一次拉升解放两个峰
- 量大是牵引车
- 回踩后再上 → 主升浪

## 六、配合阴线平台

技巧：把所有**低开阴线**的位置画出来：
- 这些位置是天然的压力位
- 未来股价到此必定有反应（高开/低开）
- 结合筹码峰大小判断突破概率
