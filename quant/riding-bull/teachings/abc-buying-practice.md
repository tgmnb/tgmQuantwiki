# ABC买入法实战推演（静态交易练习）

> 静态交易练习专题：在通达信训练模式中随机选股，用收盘价逐根K线走，综合运用ABC判定、空间测量、MACD抓大放小、八字要诀抓小放大、筹码分析、DMI、自然箱体50位等全部技术做买卖决策。
>
> **实盘收益率：125%（龙舟股份，多波段操作）**

---

## 一、训练模式基础设置

### 激活训练模式
- 按 `47` 键进入通达信训练模式
- 点 **随机** 按钮，再点 **隐藏当前品种**、**隐藏日期**
- 用画线工具盖住股票名称（监控精灵→缩小盖住）

### 随机选股（防作弊）
- 按 `60` 键 → 回车 → 沪深A股涨幅排名榜（4000+只股票，无法作假）
- 随便翻翻停停 → 再点 **随机**（取中间段K线，看不到左边历史）
- ⚠️ 不要用自选股练习！对方可能只放几只股票，结果失真

### 交易规则
- **只用收盘价做买卖**（不是盘中决策）
- 2:50~2:55 或早盘9:30 看收盘决定买卖
- 模拟盈亏比计算

## 二、K线行走前的分析框架

拿到一只随机股票后，先看三线（周线→月线→日线），建立全局认知：

### Step 1：数字操盘看大顶大底
- 查看当前价格数字特征（367/147/258等规律）
- 示例：龙舟股份底11.81（含1→147），顶16.62（含2→22做头），底6.74（含6→369）
- 数字特征辅助判断当前是反弹还是反转级别

### Step 2：月线/周线定方向
| 组合 | 含义 |
|:----:|:------|
| 月线绿柱 + 周线红柱 | 日线只能做**反弹**，不是反转主升浪 |
| 月线红柱 + 周线金叉 | 日线可以做**主升浪** |
| 周线零轴下绿柱 | 行情级别小，慎入 |
| 周线零轴上二次金叉 | 大行情基础 |
| 月线DMI见信号 | 至少先横盘不跌，非立即见底 |

### Step 3：日线看当前状态
1. 144均线方向（向上/走平/向下）
2. 是否站上25日线（第二生命线）
3. 是否出现完整**下跌→横盘→下跌**图形
4. 当前MACD柱是大绿/中绿/小绿/小红/中红/大红？

## 三、ABC买入法的实战操作流程

### 核心口诀
```
大绿小绿是同一波
小绿小红是反弹
大红中红是主升
大小小红是还债（最后一波）

长阴加长腿，下跌已到位
到位不到位？阳包阴定位
次阳过阴半，次日高开为强
低开高走，放量为强
```

### 买点判定流程

```
下跌→横盘→下跌 图形出现
    ↓
大绿柱确立（绿柱第三四根最高）
    ↓
MACD绿柱缩短 → 出现小红
    ↓
等待阳包阴（放量/缩量都可）
    ↓
判定ABC位置：
  A点 = 第一根底背离阳包阴
  B点 = 突破A点后的回踩确认
  C点 = 突破B点后的加速（常一字板买不到）
    ↓
买入，设风控（阴实底/阴虚底）
```

### 具体K线品种的ABC分类

| 级别 | MACD柱 | 含义 | 操作策略 |
|:----:|:------:|:----:|:---------|
| **大绿** | 绿柱最高/次高 | 下跌主浪 | 观望，不买 |
| **小绿** | 绿柱缩短变矮 | 下跌衰竭 | 准备观察阳包阴 |
| **小红** | 红柱刚出但低于前一红 | **反弹级别** | 快进快出，空间受限 |
| **中红** | 红柱高于前一红但低于大红 | 中期上涨 | 可波段持有 |
| **大红** | 红柱创新高 | 主升浪性质 | 重点持有，关注顶背离 |

⚠️ **关键判断**：月线绿柱时，日线出小红只能是反弹（不能当反转做）！只有月线红柱+周线金叉支撑，日线的大红才是主升浪。

### A/B/C三点具体判定（量化）

**A点（第一买点）：**
- 图形：大绿→小红的第一根阳包阴
- 条件：长阴加长腿 + 阳包阴放量（或缩量但站稳阴二一位）
- 风控：跌破阴实底（前一根阴线的最低价）无条件卖出
- 预期：反弹至自然箱体50位（乒乓球理论：跌下去反弹一半）

**B点（第二买点/加仓点）：**
- 图形：A点上冲后回踩，不破A点起涨的阴实底
- 条件：25日线上阴线买（或者次阳过阴半）
- 风控：跌破A点起涨阴实底
- 预期：突破前高走主升

**C点（第三买点）：**
- 图形：B点突破后的加速段
- 特点：常以一字板或跳空涨停出现，实际买不到
- ⚠️ 放弃C点，不追高

### 卖点判定（同样重要！）

```
创新高 → 往往是卖出点，不是买入点！
脉冲量 → 常态是跌，变态是涨
阴包阳 → 减仓/卖出
破阴实底 → 无条件卖出
破阴二一位 → 卖出
大红变小绿 → 卖出离场
```

**三个必须卖出的情况：**
1. **空间到位**：反弹到自然箱体50位附近爆量 → 卖
2. **顶背离**：价格创新高但MACD红柱比上一波低 → 卖
3. **破位**：跌破设定的风控阴实底/阴虚底 → 卖

## 四、风险控制（风控至上）

### 风控位设置规则

| 买入级别 | 风控位 | 止损幅度 |
|:--------:|:------:|:--------:|
| A点（底部反弹） | 阴实底（买入阴线最低价） | 约7-8% |
| A点（底部反弹） | 阴虚底（买入阴线下影线最低） | 约10-12% |
| B点（突破回踩） | 起涨K线的阴实底 | 约5-7% |
| B点（突破回踩） | 二一位（前阴线二分之一） | 约4-5% |

### 风控铁律
```
买进需要1万个理由，卖出只需要破位
大风控大利润，小风险小利润
跌破阴实底 → 时刻盯着
跌破阴虚底 → 无条件清仓
```

### 特殊场景处理
- **小红阳包阴买入**：判定为反弹，空间有限，到达50位附近就卖
- **周线红柱支撑**：日线回调不太可能直接死叉，可以多持有一段时间
- **月线绿柱+日线反弹**：只能做反弹不能做反转，不要预期主升浪
- **144横着走**：股价会上下串，买在下位（靠近144）不买在上位

## 五、实战案例：龙舟股份完整推演

**股票**：龙舟股份
**日期**：2021-10-10（静态交易练习）
**总收益**：125%

### 第一波段：底背离A点买入 → 反弹50位卖出（+33%）

```
看盘初始：日线顶背离，月线绿柱，周线零轴下
→ 确认只能做反弹，不做反转

红柱出现 → 出现放量阳包阴 → 判定为A点买入
→ 风控：跌破6.73阴实底卖出
→ 目标：自然箱体50位（7.95）

涨到7.95附近爆量（脉冲量69天最大）
→ 判定：脉冲量常态是跌 + 空间到位
→ 卖出，锁定33%
```

### 第二波段：下跌横盘下跌 → B点买入 → 卖出（+60%）

```
A点卖出后下跌 → 又走完一轮大绿小绿
→ 下跌50%（从12→6元），符合乒乓球反弹条件

收到筹码上移 + 下横下图形完成
→ 找到阳包阴买入B点
→ 放量过左方筹码密集区

持有直到出现小红顶背离
→ 分两次卖出（第一次减仓在通道上沿，第二次破位清仓）
→ 锁定60%
```

### 第三波段：周线金叉回踩买入 → 卖出（+2%）

```
周线走金叉，月线也红了
→ 日线买了个B2位置

结果是弱势反弹，只赚了2%
→ 破位卖出，没等来主升
```

### 关键教训
1. **月线绿柱时不要期待大行情** — 只能做反弹，反弹到位必须卖
2. **空间测量永远优先于量价** — 先算空间再看量价
3. **熊市股票做不到主升浪** — 如果月线亮绿灯，日线怎么走都只是反弹
4. **不是每个ABC都有C** — C点可能直接破损，不猜顶

## 七、风控铁律：移动量破位斩立决

> **买入必须有风控，破移动量最低价斩立决。** 这是操盘手技术中最核心的风控口诀，没有任何例外。

### 移动量风控口诀

```
买入必须有风控
破位斩立决
  → 破 = 跌破移动量最低价
  → 斩 = 无条件卖出，不犹豫
  → 立决 = 当即决定，不拖泥带水

没有风控的买入 = 赌博
设了风控不执行 = 自杀
```

### 移动量风控的操作规则

| 步骤 | 操作 |
|:----:|------|
| ① 买入时 | 确定移动量K线的最低价作为风控位 |
| ② 持仓中 | 每日收盘确认是否跌破风控位 |
| ③ 破位时 | 次日开盘无条件卖出，不找借口 |
| ④ 未破位 | 持有，等待新移动量上移风控位 |

### 实战点评案例

以下为骑牛登山实盘操作中的风控执行案例：

**浙江永强（两次操作）**：
- 第一次：阳包阴放量买入 → 设移动量最低价风控 → 破位斩立决 → 小亏离场
- 第二次：再次出现移动量 → 阳包阴买入 → 不破风控 → 持有获利

> 第一次斩立决虽然小亏，但保住了本金。第二次才能抓住机会。

**中华岩土**：
- 股价站上移动量 → 移动量最低价不破 → 风控有效 → 持股待涨
- 移动量是判断"能不能拿"的唯一标准

**金龙鱼**：
- 右量过左量（量能突破左侧高点）→ 确认资金进场
- 设好移动量风控 → 量在价先，风控在先

### 风控执行的三个层次

```
初级：设了风控，但破位时犹豫 → 被套
中级：设了风控，破位时执行 → 保本
高级：设了风控，破位前预判减仓 → 主动

目标：至少做到中级，追求高级。
```

### 风控与仓位

```
移动量风控位距离买入价：
  < 3%   → 正常仓位
  3-5%   → 半仓
  5-7%   → 轻仓
  > 7%   → 放弃此买点（风控太远，不划算）
```

> **核心认知**：风控不是限制你赚钱，是保护你不亏大钱。破位斩立决 = 亏小钱等大机会。
