# 周线翻倍技术（180周均线/大悬空/脚踩巨量）

> 适合大资金（千万级以上）的稳健翻倍体系。以**180周均线**为核心，结合基本面、题材风口、巨量战法。一旦突破，至少翻一倍，不是翻一倍，是**至少翻一倍**。

---

## 四大要素

### ① 大悬空

```
股价长时间在180周均线下方运行
  → 跌幅够大（空间）
  → 时间够长（>1年，3-5年更佳）
```

大悬空 = 下跌的空间 + 震荡的时间 两大要素缺一不可。

### ② 脚踩巨量

股价出大悬空后，必须放出**相对区间巨量**或**历史巨量**。

| 类型 | 说明 |
|:----:|:------|
| **相对区间巨量** | 比前后一段时间的量都大 |
| **历史巨量** | 上市以来最高量（最佳） |

> 脚踩巨量 = 股价站在了巨量K线的**上方**（回头看：K线在脚下）

### ③ 平台（关键买点）

平台包含两个子条件：

**子条件1：站上180周均线**
- 股价必须翻上180周线
- 有两种方式：
  - A型：一触180回踩再突破
  - B型：直接突破后回踩不破

**子条件2：立标杆（MACD确认）**
```
MACD白色线（DIF）的高度必须高于前方一切
  → 这代表有大资金进场了
  → 这叫"立标杆"
```

**子条件3：突破阴线平台**
```
涨了一波最高阴线开盘价 = 左峰线
突破左峰线 = 买点
```

### ④ 越走越强

```
A点（最低）→ B点（高于A）→ C点（高于B）
每次回落的低点越来越高
  → 趋势已经改变 → 向好信号
```

---

## 完整买点流程

```
① 大悬空确认：跌幅够 + 时间够（>1年）
② 脚踩巨量确认：相对或历史巨量在脚下
③ 立标杆确认：MACD白线高于前方一切
④ 站上180周线
⑤ 突破阴线平台（涨一波最高阴线开盘价）
⑥ 买入！

口诀：
  一悬空、二巨量、三平台、四越强
  突破买进，至少翻一倍
```

---

## 买入时机

| 方式 | 操作 | 时间成本 | 收益 |
|:----:|:------|:--------:|:----:|
| **突破买入** | 周五收盘确认过左峰线 | 短 | 翻倍 |
| **潜伏买入** | 在底部回踩时提前介入 | 长（等风口） | 3-4倍 |

> 潜伏消耗时间，但成本更低。突破买进不耗时间，但赚的相对少一些。

### 关键时间点

- **周五收盘**买进（周线定盘）
- 周三突破加入自选，周五确认过左峰再动手

## 选股方法

每天按 `60` 看涨停板排行榜：
```
① 看到涨停的股票
② 打开周线 → 设180均线
③ 检查是否符合四大要素
④ 符合→加入自选，等周五确认买入
```
