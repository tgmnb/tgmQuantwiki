# 静态交易练习方法论（训练模式高级运用）

> 在通达信训练模式中，随机选股、逐根K线推演，综合运用所有技术做买卖决策。本篇侧重**训练模式的进阶设置、自动播放推演、数字操盘辅助判定、以及实盘中的买卖点执行细节**。
>
> **实盘收益率：65%（多波段，同一只股票完成完整交易循环）**

---

## 一、训练模式进阶设置（比基础设置更细）

### 坐标与复权设置

**核心问题**：量价不匹配会导致技术分析失误。

| 设置项 | 操作 | 说明 |
|:------:|:----|:-----|
| 复权模式 | **前复权** | 不能用不复权，否则价格历史不连续 |
| 量赋权 | 必须勾选 | 价格复权了量能也必须复权，否则量价不匹配 |
| 设置三→主图指标不影响坐标 | **不打勾** | 打勾会压缩空间，影响百分比排比判断 |
| 分时移动显示行情小窗口 | 打勾 | 屏蔽小窗口，避免显示股票名和时间 |
| 十字光标 | **必须显示** | 用于看量值和坐标位置 |

### 群内PK防作弊

当别人分享屏幕给你练习时：
1. **必须全屏分享** — 局部分享可能只显示名字，对方朋友能看见股票名作弊
2. **必须按60键+回车** — 从沪深A股涨幅榜翻进去，确保是4000+只随机，不是自选股里的几只
3. **名字必须盖掉** — 用画线工具或监控精灵缩小盖住

### 全屏练习显示布局

```
全屏时应看到：
┌────────────────────────────────────┐
│  年月日  周期切换（日/周/60分）    │
│                                    │
│      K线主图（MACD副图）           │
│                                    │
│  左边周期导航：月/周/日/60分/30分  │
└────────────────────────────────────┘
```

只留**价格、量、MACD**三个最核心指标，其他全部屏蔽。

## 二、自动播放推演（逐根K线行走法）

### 设置方法
```
按47进入训练模式
→ 随机选股
→ 点"自动播放"
→ 设置3秒/根（省去手动翻看，让图形自己走出来）
→ 随时暂停，放大/缩小看细节
→ 继续自动播放
```

### 推演原则

```
先看空间 → 再看时间 → 再说量价
```

**具体流程：**

1. **月线定大方向**：红柱？绿柱？大绿小绿？离零轴多远？
   - 月线绿柱（含大绿/中绿）→ 日线只能做反弹
   - 月线红柱 → 可做日线主升浪

2. **周线定操作级别**：金叉？死叉？零轴上/下？
   - 周线零轴上金叉 → 大行情级别
   - 周线零轴下小红 → 反弹级别

3. **日线逐根走**：144方向 → 站上25线？→ MACD柱形态？→ 寻找ABC买点

### 核心原则：先让K线走出来

> **不走出来你不需要看**。让自动播放自己走，直到144走平上翘、ABC图形清晰、出现明确的买点信号，再下手。

不要急于判断，不要预设方向，让K线自己告诉你它要干什么。

## 三、数字操盘辅助判定

数字操盘在本篇练习中频繁使用，作为辅助判大顶大底的工具。

### 核心规律

| 数字 | 规律 | 示例 |
|:----:|:----|:----:|
| **3/6/9** | 相同数（大顶大底常见） | 底6.69顶9.96 |
| **1/4/7** | 倍数关系（1->4->7或7->4->1） | 底5.39顶6.89（差1.5→4的倍数） |
| **2/5/8** | 倍数关系 | 底11.81顶16.62（含1->4->7->?） |

### 实战用法
1. 看当前价格是否落在关键数字附近
2. 历史顶部底部是否出现相同数字（369）
3. 当前浪形的起点/终点是否符合数字规律
4. **辅助但不主导**—以图形分析为主，数字只是辅助

**示例**：某一波起点5.39（含3/6/9的3），顶6.89（含9），底又回到5.39 → 数字策略验证此处是重要支撑。

## 四、实盘中的买卖点执行细节

### 买点执行

#### 1）144方向先看，再谈买点

| 144方向 | 含义 | 操作 |
|:-------:|:----|:----|
| 向下 | 不能操作 | 等待，走一段再说 |
| 走平 | 股价上蹿下跳 | 买在下位（靠近144），不买在上位 |
| 向上 | 可操作 | 回踩144可买 |

#### 2）阴实底 vs 阴虚底（精确风控位）

```
阴实底 = 阴线实体最低价（收盘价或开盘价中较低的那个）
阴虚底 = 阴线下影线的最低价（包含影线）

买点买入后，以阴虚底为主要风控位
跌破阴虚底 → 无条件清仓
```

#### 3）A里边的A和B里边的A（ABC嵌套）

实战中ABC是嵌套的——一个大级别的A里面可以有小级别的ABC：

```
大绿（级别A）
  ├── 小红（A里边的A点）
  │     ├── 小绿（A里边的B点）  
  │     └── 再阳包阴（A里边的C点）
  └── 大绿最后一跌
      └── 小红（B里边的A点） ← 真正的买点
```

**实战口诀**：
- **A点只能做反弹**（空间有限，到了50位就走）
- **B点可以做波段**（突破确认后空间更大）  
- **C点往往买不到**（一字板/跳空加速）

#### 4）买点口诀全表

| 场景 | 条件 | 动作 |
|:----:|:----|:----:|
| 长阴加长腿 | 出现阳包阴 | **买入**，阴虚底做风控 |
| 次阳过阴半 | 次日高开/低开高走放量 | **买入** |
| 25日线上阴线 | 阳包阴确认 | **买入** |
| 大绿小绿走完 | 出小红+阳包阴 | **买入**（反弹级别） |
| 大绿直接 | 无阳包阴 | **不买** |
| 量太小 | 即使阳包阴 | **不买**（量能不足以解放左方套牢盘） |

### 卖点执行

#### 1）放量即是卖出信号

> 技术就是要说**在哪个位置一定会爆量**。这是自然规律。

每次画好目标位（如6.4、7.25、9.3等），当K线走到该位置**爆量**时：
- 放量阳线 → 持有看下一根是否缩量
- **放量阴线或放量星线** → 卖出
- **脉冲量（近60-70天最大量）** → 卖出（常态是跌）

#### 2）上涨横盘上涨走完 = 卖出

```
上涨→横盘→上涨 图形完成
    ↓
跌破上一笔起涨的阴实底
    ↓
卖出
```

判断一笔是否走完的方法：
- 从A点买入开始，数浪型
- 每笔走完，自然箱体目标到位
- 跌破风控位 → 无论盈亏都出

#### 3）创新高是卖点！

> **记住：创新高是个卖出点，不是买入点。**

因为：
- 创新高意味着顶背离可能性大
- 创新高起多诱多
- 创新高后铁律是大阴线回踩

#### 4）卖出后不要立即反手买

```
大阴线下来 → 等它形成
下跌→横盘→下跌 需要时间走完
小绿出现了 → 等阳包阴
```

急跌急买是亏损最大来源。

## 五、风控体系（本篇核心）

### 双重风控：阴实底 + 阴虚底

```
阴实底 = 关键K线实体最低价（稍宽松）
阴虚底 = 关键K线下影线最低价（最严格）

买入后先看阴虚底：
  - 未跌破 → 持有
  - 似破非破 → 时刻盯着（盘中破不算，收盘破才算）
  - 明确跌破 → 无条件卖出

某一段的阴虚底破了 → 该段操作结束
```

### 级别递进风控

| 风控级别 | 标准 | 动作 |
|:--------:|:----|:----:|
| 一级 | 跌破本次买入的阴实底 | 减仓1/2 |
| 二级 | 跌破本次买入的阴虚底 | 清仓 |
| 三级 | 跌破起涨浪型的前低 | 合约结束，不再参与 |
| 四级 | 月线绿柱+周线死叉 | 不参与该股任何操作 |

### 大/小风控

```
大风控 = 不跌破A点起涨的阴实底 → 大利润（可能翻倍）
小风控 = 不跌破B点买入的阴虚底 → 小利润（10-20%）
```

## 六、实战案例完整推演

**股票**：某A股（练习时随机选中）
**日期**：2021-11-07
**总收益**：65%

### 第一波段：144走平→买在下位→反弹50位卖出（+10%）

```
初始：月线大绿（但左侧未知真假），周线零轴下悬空（无量）
→ 144走平 → 股价上下串，等待买在下位

144走平中突破前高 → 买点出现（波段级别）
→ 风控：突破阴的阴虚底

目标位计算：6.2附近（自然箱体50位）
→ 涨到6.88爆量 → 卖出
→ 赚约10%
```

### 第二波段：144走平+倍量阳→周线金叉潜力→卖出（波段）

```
跌下来后：144还是走平的，买在下位
→ 出现倍量阳K线（比前震荡区最高量还大）
→ 判定：B点的A类买点

如果涨上前 = 周线金叉 → 走周线级别行情
→ 持有到上升横盘上升完成
→ 跌破二一位 → 卖出

60分钟MACD顶背离确认 → 清仓
→ 赚约（波段，累计到20-30%）
```

### 第三波段：月线金叉→周线零轴上二次金叉→大行情（+35%）

```
月线走金叉了！周线也零轴上金叉了
→ 日线级别可以做主升浪了

回调下横下 → 买B点
→ 风控：阴虚底

爆量在9.3（画线目标位）→ 先没卖
→ 出现阳线星（放量不涨）→ 警觉
→ 又出一根阴线 → 跌破→ 卖
→ 赚约35%

后边直接跌停 → 卖对了
```

### 关键教训
1. **144走到中线方向** — 从走平到上翘需要时间，在这之前不要期待大行情
2. **月线走金叉才是做多的门票** — 只有月线红了，才有主升浪
3. **爆量位置精确到分** — 6.4、7.25、9.3这些位置都是画线算出来的，不是拍脑袋
4. **第一次突破往往是假的** — 均线发散得太开，第一次突破大概率会回踩

## 七、训练模式练习要点总结

### 练习的目标
- 不是预测涨跌（没人能预测）
- 是练习**看到信号后执行**的肌肉记忆
- 在真实交易前，用模拟把错误犯光

### 练习的质量标准

- 是否每一笔买入都有明确的风控位？
- 是否每一笔卖出都有明确的破位信号？
- 是否遵循了"先看空间，再看时间，再说量价"的顺序？
- 月线绿柱时，是否严格只做反弹不做反转？
- 爆量时，是否理性执行卖出而不是幻想"还能涨"？

### 练习频率建议
- 每天至少走1-2只股票
- 每只股票从60键随机开始，走到自然结束
- 记录每笔交易盈亏和原因
- 重点练习**不买**的耐心（大部分时间应该空仓等待）
