# ABC点判定法（量价ABC + 周期ABC）

> 用A/B/C三个关键点定位波段的底和顶。分为**量价的ABC**（成交量+价格柱体）和**时间周期的ABC**（MACD黄白线+红绿柱）。两种ABC共振时信号最强。

---

## 一、下跌中的ABC结构

### 前提条件

必须出现完整图形：**下跌→横盘→下跌**

没有这个结构，ABC判定无效。

### 两种ABC的分类

| 类型 | 依据 | 含义 |
|:----:|------|------|
| **量价的ABC** | 成交量柱体 + K线价格 | 看量能的衰竭和价位的承接 |
| **时间周期的ABC** | MACD白色线 + 红绿柱高度 | 看动能的衰竭（背离） |

> 量价看的是"实"（资金），周期看的是"虚"（动能），虚实结合才是完整判断。

---

## 二、量价的ABC

### 定义

| 点 | 特征 |
|:--:|------|
| **A点** | 第一个下跌低点，**放量大阴**量，下跌初始 |
| **B点** | 第二个下跌低点，**缩量小阴**，下跌动能减弱 |
| **C点** | 第三段下跌末端 — 出现**阳包阴**的买入信号 |

### 买点判定

**最佳买点 = B点**（比A点好）

| 买点 | 特征 | 预期 |
|:----:|------|------|
| A点 | 第一个阳包阴，量能不一定够 | 反弹为主 |
| **B点** | 阳包阴+缩量+MACD底背离共振 | 一波拉升 |
| C点 | 进一步确认 | 延续上涨 |

### 阳包阴的三种强度

```
① 中阳包阴（实体大阴线被全包）⭐⭐⭐⭐⭐ 最佳
② 阳线收盘价包阴线最高价 ⭐⭐⭐⭐  完美
③ 阳线收盘价包阴线开盘价 ⭐⭐⭐   一般
```

### 量价B点的买入流程

```
第一步：下跌横盘下跌结构完成
第二步：出现阳包阴（阳量+阳价）
第三步：检查量能——阳量需大于阴量
第四步：次日——高开补量或继续放量（确认信号）
第五步：尾盘2:50买入
```

## 三、时间周期（MACD）的ABC

### 定义

| 点 | MACD特征 |
|:--:|---------|
| **A点** | MACD白色线最高点 / 红柱最高（第一波拉升） |
| **B点** | 白色线低于A点 / 红柱低于A点（顶背离/底背离） |
| **C点** | 白色线低于B点 → 死亡钩或黄金钩确认 |

### 买点配合

```
时间周期C点（MACD底背离走完）
  + 量价B点（阳包阴出现）
  = 共振买点 ⭐⭐⭐⭐⭐
```

### 卖点配合

```
时间周期A→B点（顶背离）
  + 量价小红（八字要诀）
  = 卖点 ⭐⭐⭐⭐⭐
```

## 四、风险控制

| 条件 | 操作 |
|------|------|
| **买入后，以阳包阴那根阳线的最低价为风控** | 跌破 = 无条件离场 |
| 出现新的倍量阳线 | 风控位上移到新阳线最低价 |
| 连续两根阳线包阴（第二天补量） | 两根视为一根，风控以最低价为基准 |

> 不懂风控 = 必套牢。

## 五、量线（填满理论）

> 成交量必须在**60日量线**之上才有行情——叫"填满"。

### 量线体系

| 量线 | 位置 | 含义 |
|:----:|:----:|------|
| 5日量线 | 短期 | 5日均量线 |
| 10日量线 | 中期 | 10日均量线 |
| **60日量线** | **关键** | **填满=有行情，不填满=没行情** |

- 成交量在60量线之下 → **大悬空** → 不涨
- 成交量在5/10量线之间 → **小悬空** → 不涨
- 成交量 > 60量线 → **填满** → 行情启动

## 六、脉冲量与妖股

> 脉冲量 = 突然放出的巨量（半年/三个月内最高量）。

**脉冲量有两种结局**：

| 结局 | 特征 | 操作 |
|:----:|------|------|
| **妖（涨）** | 脉冲量后不破底 + 高开高走 | 持有，成妖 |
| **货（跌）** | 脉冲量后跌破底 | 无条件卖出 |

**判定**：脉冲量之后，如果次日**高开**→ 成妖可能性大（让所有买入的人都赚钱）；如果低开低走→ 出货。

## 七、实战总结表

```
下跌横盘下跌（前提结构必须有）
    ↓
量价A → 第1个底（放量，不买）
量价B → 第2个底（缩量+阳包阴=买点）
    ↓
确认：次日高开+补量/放量
    ↓
风控：阳线最低价
    ↓
卖出：时间周期ABC出现顶背离 → 小红 → 系数到位
```

## 八、中枢与拐点

> **上行找大阴，下行找拐点。** 中枢是价格震荡区间的顶和底，拐点就是中枢的边界。

### 核心口诀

```
上行找大阴 → 上涨过程中，先找前方大阴线压力位
下行找拐点 → 下跌过程中，找拐点/中枢支撑位
```

### 中枢的定义

```
中枢 = 价格横盘震荡区间
  中枢的顶 = 震荡区间的上沿（压力位）
  中枢的底 = 震荡区间的下沿（支撑位）

拐点 = 中枢的顶或底
  价格突破中枢顶 → 向上拐点
  价格跌破中枢底 → 向下拐点
```

### 中枢的实战运用

| 方向 | 找什么 | 作用 |
|:----:|:------:|:----:|
| **上行**（上涨） | 前方大阴线 | 判断压力位，到压力位附近注意减仓 |
| **下行**（下跌） | 中枢底部 | 判断支撑位，到支撑位附近准备抄底 |
| **震荡** | 中枢顶底 | 高抛低吸，箱体操作 |

### 大阴线的压力判断

```
上涨途中遇到前方大阴线：
  ① 标记大阴线的最高价和最低价
  ② 大阴线实体二分之一位 = 强压力位
  ③ 股价到达大阴线底部 → 开始承压
  ④ 放量突破大阴线顶部 → 压力解除，看高一线
```

### 破位后的超短线买点

```
破位（跌破中枢支撑）后：
  第二天 或 第四天 = 超短线买点
  
条件：
  股价从破位点继续下跌 4.3% 以上
  → 超跌反弹概率极高
  → 买入后等反弹获利

原理：
  破位后恐慌盘涌出
  第二天（或第四天）恐慌盘耗尽
  出现超跌反弹
```

| 时间窗口 | 操作 |
|:--------:|:----:|
| 破位第1天 | 不买，等恐慌释放 |
| **破位第2天** | ⭐ 超短线买点（跌4.3%以上） |
| 破位第3天 | 观察，可能延续 |
| **破位第4天** | ⭐ 超短线买点（如第二天未反弹） |

### 中枢与ABC点的关系

```
下跌→横盘（形成中枢）→ 下跌（破中枢底）
  ↓
中枢底部被跌破 → A点区域
  ↓
反弹回中枢 → B点确认（站上中枢底）
  ↓
突破中枢顶 → C点加速

中枢 = ABC结构的"横盘"部分
拐点 = ABC的买卖信号触发点
```

### 操作要点

1. **上行时**：先画前方大阴线压力位，到了就警惕
2. **下行时**：等中枢底部出现拐点信号（下影线/阳包阴）
3. **破位时**：不急抄底，等第2天或第4天的超跌机会
4. **反弹时**：回到中枢内部再考虑加仓
5. **风控**：破中枢底后反弹不回中枢 = 彻底走弱，清仓
