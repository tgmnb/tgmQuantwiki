# 5/10均线细节 + 分钟突破 + 9:33开盘定式

> 分钟级别的全套短线技术：5/10均线叉上阴叉下阳、5/15分钟放量突破买入法、9:33开盘定式（开盘三分钟箱体）。

---

## 一、5日与10日均线细节

### 叉上阴（买入信号）

```
5日线与10日线金叉向上
股价在均线上方收阴线（叉上阴）
  → 买点（线上阴线买）
```

**最佳**：高开低走放量阴线，未跌破5日与10日线。

### 叉下阳（卖出信号）

```
5日线与10日线死叉向下
股价在均线下方收阳线（叉下阳）
  → 卖点（线下阳线抛）
```

### 与25日线的对应关系

```
25日线上阴线买 = 叉上阴（同方向）
25日线下阳线抛 = 叉下阳（同方向）
```

### 5日死叉10日的死叉点 = 波段压力位

```
5日和10日的死叉点画一条水平线
  → 未来反弹到这个位置 = 波段压力位
  → 结合MACD三浪判断是否能突破
```

## 二、5/15/60分钟突破买入法

### 核心原理

在一波日线调整的末端，用分钟级别捕捉**放量突破前日最高价**的瞬间。

### 操作流程

```
① 日线找到一波调整形态（波浪末端）
② 确定前一日（或前一波）的最高价（不分阴阳）
③ 当日盘中用5分钟/15分钟观察：
     股价突破前日最高价 + 放量（倍量）
     5分钟量 ≈ 前一日60分钟量
   → 买入
④ 次日开盘价卖出 或 盘中冲高卖出
```

### 关键条件

| 条件 | 要求 |
|:----:|:------|
| **K线** | 突破前日最高价 |
| **量能** | **5分钟量 ≥ 前一日60分钟量** |
| **形态** | 前日分时最好是阴线或假阳线 |

### 超短线 vs 波段

| 模式 | 持有时间 | 操作 |
|:----:|:--------:|:----:|
| **超短线** | 1天 | 次日开盘卖/冲高卖 |
| **波段** | 视波浪形态 | 可持有至主升浪 |

### 成功率

> 约**90%**——突破+放量是标准信号。

## 三、15分钟任意超短买入法

### 适用范围

- **下跌类**：A/B/C点（配合MACD大绿小绿），买在反弹的阴线最高价被突破时
- **上涨类**：25日线上V型回踩，买在假阳线最高价被突破时（往往变主升浪）

### 操作流程

```
① 找到一波波浪形态（下跌一波→反弹→回调）
② 在回调末端找到一根阴线（或假阳线）
③ 记下这根K线的最高价（不分阴阳）
④ 次日盘中15分钟：
    放量（相对区间天量）突破该最高价
  → 买入
⑤ 次日开盘卖出（超短线）或持有（波段）
```

### 风控

```
风控位 = 那根阴线/假阳线的最低价
跌破 → 止损
```

## 四、9:33开盘定式（开盘三分钟箱体）

### 核心原理

9:30开盘后**三分钟内**（9:30-9:33）形成一个箱体（最高-最低）。这个箱体决定了当天的强弱定式。

### 使用场景

| # | 场景 | 说明 |
|:-:|:----|:------|
| 1️⃣ | **移动量破位** | 跌破移动量最低价时 |
| 2️⃣ | **阴实顶/阴虚底破位** | 跌破/突破关键阴线位置 |
| 3️⃣ | **死亡模式** | 25日线下+MACD大绿 |
| 4️⃣ | **阴排/关键支撑** | 重要均线/平台破位时 |

### 开盘定式判定

| 开盘走势 | 判定 | 操作 |
|:--------:|:----:|:----:|
| 高开+向上突破箱顶 | 强势 | 可买 |
| 高开+跌破箱底 | 转弱 | 观望 |
| **低开+放量跌破箱底** | **极弱** | **斩立决（卖出）** |
| 低开+向上突破箱顶+爆量 | 弱转强 | 可关注 |

### 最强信号

```
低开 + 三分钟箱体 + 放量跌破箱底
  → 反抽确认不过
  → 斩立决（立即卖出）
```

### 与15分钟突破的关系

```
9:33箱体突破（开盘定式）
  + 15分钟放量突破前日最高价
  = 最强买入信号 ⭐⭐⭐⭐⭐
```

## 五、标志性K线（板后站稳均线）

> 涨停板后急跌，出现一根K线**重新站稳5日线与10日线**——这根K线叫**标志性K线**。

### 形态定义

```
连板上涨 → 急跌回踩 → 某天出现一根K线
  → 同时站在5日均线和10日均线上方
  → 这根K线 = 标志性K线 ✅
```

### 核心规则

| 规则 | 说明 |
|:---:|:-----|
| **站稳双线** | K线收盘价必须同时 > 5日线和10日线 |
| **最低价是风控** | 标志性K线的**最低价**不能被跌破 |
| **建立在周线月线完美基础上** | 周线和月线走势必须处于多头或上升趋势中，否则标志性K线无效 |

### 操作

- 标志性K线出现后，次日可买入
- 止损设在标志性K线的最低价
- 结合MACD、量能确认——放量标志性K线比缩量更有力
