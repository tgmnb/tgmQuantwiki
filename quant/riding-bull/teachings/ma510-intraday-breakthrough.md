# 5/10均线细节 + 5/15分钟突破买入法

> 两个小细节：均线叉上阴/叉下阳的操作口诀，以及分钟级别的放量突破买入法。

---

## 一、5日与10日均线细节

### 叉上阴（买入信号）

```
5日线与10日线金叉向上
股价在均线上方收阴线（叉上阴）
  → 买点（线上阴线买）
```

**最佳**：高开低走放量阴线，未跌破5日与10日线。

### 叉下阳（卖出信号）

```
5日线与10日线死叉向下
股价在均线下方收阳线（叉下阳）
  → 卖点（线下阳线抛）
```

### 与25日线的对应关系

```
25日线上阴线买 = 叉上阴（同方向）
25日线下阳线抛 = 叉下阳（同方向）
```

### 5日死叉10日的死叉点 = 波段压力位

```
5日和10日的死叉点画一条水平线
  → 未来反弹到这个位置 = 波段压力位
  → 结合MACD三浪判断是否能突破
```

## 二、5/15/60分钟突破买入法

### 核心原理

在一波日线调整的末端，用分钟级别捕捉**放量突破前日最高价**的瞬间。

### 操作流程

```
① 日线找到一波调整形态（波浪末端）
② 确定前一日（或前一波）的最高价（不分阴阳）
③ 当日盘中用5分钟/15分钟观察：
     股价突破前日最高价 + 放量（倍量）
     5分钟量 ≈ 前一日60分钟量
   → 买入
④ 次日开盘价卖出 或 盘中冲高卖出
```

### 关键条件

| 条件 | 要求 |
|:----:|:------|
| **K线** | 突破前日最高价 |
| **量能** | **5分钟量 ≥ 前一日60分钟量** |
| **形态** | 前日分时最好是阴线或假阳线 |

### 超短线 vs 波段

| 模式 | 持有时间 | 操作 |
|:----:|:--------:|:----:|
| **超短线** | 1天 | 次日开盘卖/冲高卖 |
| **波段** | 视波浪形态 | 可持有至主升浪 |

### 成功率

> 约**90%**——突破+放量是标准信号。
