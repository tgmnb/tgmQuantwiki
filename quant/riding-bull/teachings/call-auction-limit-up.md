# 集合竞价抓涨停

> 早盘9:15-9:25集合竞价期间，利用量价关系和盘口挂单识别当天可能涨停的个股。两种方法：**放量阴线法**（看日线+60分钟量价）和**盘口语言法**（看买卖盘口挂单）。

---

## 一、放量阴线法（日线量+60分钟价定位）

### 选股基础（前一日收盘后）

按 `60` → 沪深涨幅排名榜 → 翻**跌幅榜**，筛选满足条件的个股加入自选。

**核心条件：前一天收放量阴线**
- 阴线成交量 > 前一日成交量的 **1/3**（3分之1）
- 最佳：> **1/2**（2分之1），即两倍量更优
- 放量阴线代表恐慌盘/获利盘涌出

> 如果会编程，可以写选股公式：`今日量 > 昨日量 × 1.33`

### 买入确认（次日9:25）

**第一步：确认高开**
- 次日集合竞价结束后必须是 **高开**

**第二步：看60分钟线**
- 找到昨日放量阴线在60分钟图中的**最大成交量那根K线**
- 查看这根K线的**最高价**
- 条件：**集合竞价的开盘价 > 昨日60分钟最大成交量K线的最高价**

**满足 → 直接挂单买入**（挂买1/买2/买3都可以，不差几分钱）

### 三种形态

| 形态 | 特征 | 操作 |
|------|------|------|
| **标准型** | 日线放量阴线 + 次日直接高开 + 60分钟最高量价直接被跳空 | 开盘价直接买 |
| **突破型** | 日线放量阴线 + 平开/低开 + 盘中拉升 | 突破60分钟最高量对应的最高价时买入，突破量需大于前量；若不足则等第二次突破追入 |
| **阳量跳空型** | 前日**放量阳线** + 次日跳空高开 | 开盘价直接买，往往演变成波段主升浪 |

### 操作流程总结

```
前一日收盘：按60→跌幅榜→找放量阴线（>昨日1/3量）→加入自选
次日9:25：  按60→涨幅榜→看自选股中哪些高开
        → 打开60分钟→看昨日放量阴线最大量的最高价
        → 开盘价>该最高价 → 买入
```

> 短线操作，次日冲高卖出。不贪，吃8-9%即可。

## 二、盘口语言法（看买盘/卖盘挂单）

### 时间窗口

| 时间 | 可撤单？ | 说明 |
|------|---------|------|
| 9:15-9:19 | ✅ 可撤单 | 9:19前可撤，单子可能有水分 |
| 9:20-9:25 | ❌ 不可撤 | **9:20后的单子更真实** |
| 9:25 | 集合竞价结束 | 按60看涨幅榜 |

### 操作方法

**9:25 按60键 → 沪深涨幅排名榜**
- 去掉涨幅太高(>8%)和太低的(<0%)
- 主要看 **5%以下，最好是4%以下**的高开股
- 越低越好——低开利润空间更大
- 用鼠标滚动翻看

**看什么：盘口挂单**

```
卖5: xxx手
卖4: xxx手
卖3: xxx手
卖2: xxx手
卖1: xxx手  ← 这里的卖盘越小越好
────────────────
买1: xxx手
买2: xxx手  ← 这里的买盘越大越好
买3: xxx手
买4: xxx手
买5: xxx手
```

### 判定标准

**买单必须占绝对优势：**
1. 看**买1-买5**有连续的大买单（特别是前4档）
2. 卖1-卖5**不能有大卖单**（最好是十位数的单子：22手、50手、33手等）
3. 具体多少才算"大"？——**与流通盘对比**
   - 1亿流通盘 → 挂单几千手（1000-2000手）算大
   - 10亿流通盘 → 挂单1万-2万手才算大

**理想形态（最强）：**
```
卖5:   2手
卖4:   5手
卖3:   3手
卖2:   1手
卖1:  10手    ← 几乎没有卖单
────────────────
买1: 3000手   ← 大买单
买2: 1000手
买3:  666手
买4:  200手
买5:  100手   ← 连续的大买单
```

### 如何买入

- **技术成熟**：直接挂涨停价买入（以最低价成交）
- **保守**：挂卖5价格买入

### 判定信号强度

| 盘口特征 | 信号 | 预期 |
|---------|------|------|
| 买盘密集大单 + 卖盘稀疏小单 | ⭐⭐⭐⭐⭐ 最强 | 当天涨停概率高 |
| 只有一单大买（买1很大，买2-买5薄弱） | ⭐⭐⭐ 中等 | 有冲高，可能不封板，冲高出货 |
| 买盘卖盘差不多 | ⭐ 弱 | 观望 |

> 配合技术图形（MACD浪形/阴线平台/均线模式）使用效果更佳。

## 三、辅助：缩量涨停 + 分时图经典形态

### 缩量涨停

在上涨初段出现**缩量涨停** = 强势信号。
- 次日必须**高开**（大幅高开最强，小高开凑合）
- 出现时机：阳量已覆盖前方缺口/压力位

### 分时图经典开盘形态

特定的分时图开盘形态会在不同股票上**重复出现**：
- 形态1：早盘某个特定时间点的放量拉升
- 形态2：另一种早盘走势

> 同一个分时图形态出现在不同股票相同位置 → 后市走势大概率一致。
> 案例：中通客车、大港股份出现完全相同分时图形态。

|------|------|---------|------|
| 南风股份 | 2020-05-07 | 买1-买5密集大单，卖盘零散 | 开盘即涨停 |

| 歌尔股份 | 2020-06-30 | 5000-6000手买单 vs 1700手卖单 | 开盘冲涨停回封 |
| 龙蟠科技 | 2020-06-30 | 全白手，上方无卖单 | 涨停+次日再涨停（共十几个点） |
| 巨人网络 | 2020-07-03 | 买2-买5均有单，挂单厚重 | 涨停不费力 |
| 太极集团 | 2020-07-03 | 卖盘稀疏，买盘厚重 | 封板不开 |
| 鼎捷软件 | 2019-06-28 | 放量阴线+次日跳空高开破60分钟最高量价 | 涨停，单次盈利8%+ |
