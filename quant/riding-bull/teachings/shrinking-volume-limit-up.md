# 缩量涨停板技术

> 涨停板当日的成交量比前一日缩量，代表筹码锁定良好，主力控盘度高。超短线/短线技术，最佳买点是缩量次日开盘。

---

## 核心定义

**缩量涨停板** = 涨停板当日的成交量 < 前一日的成交量（不论前一日是阴是阳）。

**关键要求**：
- 缩量不能低于前一日成交量的 **1/2**
- 最佳缩量区间：前一日量的 **1/3~1/2**
- 低于1/2 = 缩得太狠 → 如弹簧压死，往往只是一冲就完

---

## 三种趋势类型

| 类型 | 均线状态 | 预期 | 策略 |
|:---:|---------|------|------|
| **① 上行趋势** ⭐⭐⭐ | 144均线向上 | 大级别行情 | 可持股，最安全 |
| **② 横盘震荡** ⭐⭐ | 均线粘合初发散 | 中级行情 | 短线做，注意风控 |
| **③ 下跌趋势** ⭐ | 均线向下 | 短线反弹 | 极短，快进快出 |

> **首选第一种**：144均线向上 + 缩量板 + 过左峰形态 = 最佳。

---

## 最佳形态

### 最强组合（首选）

```
前一日：放量阴线 ← 假跌，洗盘
当日：   缩量涨停 ← 筹码锁定，主力控盘
```

### 次强组合

```
前一日：放量阳线
当日：   缩量涨停
```

### 需要回避

- 缩量低于前一日 **1/2以下** → 弹簧压死 → 冲一下就完
- **MACD顶背离**的缩量板 → 不能买
- 高位远离均线缩量板 → 风险大

---

## 买入方法

| 开盘形态 | 操作 |
|---------|------|
| **高开6-8%** | 最佳！直接挂涨停价买入（以开盘价成交） |
| **一字板** | 排队，但观察封单量 |
| **低开** | 谨慎，可能转弱，用板后系数操作 |

### 关键配合条件

```
缩量板买入需同时满足：
  ① 25/60/144均线附近（不远离）
  ② MACD正向（非顶背离）
  ③ 放量阴+缩量阳组合为佳
  ④ 最好是三浪位置
```

---

## 风控

| 条件 | 动作 |
|------|------|
| **次日不涨停** | ✅ 卖出（不板就走） |
| **跌破放量阴线的收盘价** | ✅ 无条件卖出 |
| **收盘价跌破阴线底** | ✅ 无条件离场 |

> 缩量板是**超短线/短线**技术，仓位要控制，不是重仓操作。

---

## 分时细节

### 封单判断

- 封单手数 ≈ 流通盘（如15.7亿流通→16万手封单）
- 封单在**逐渐缩小**时 → 提前低挂卖出
- 集合竞价量是昨日的**3倍以上** → 容易被核按钮

### 高开 vs 低开

| 位置 | 最佳开盘 |
|------|---------|
| **低位/底部** | 低开高走最佳 |
| **中位/高位** | 高开最佳（6-8%） |

### 一字板放巨量

一字板如果放出超级巨量（远超流通盘对应手数）→ 容易核按钮 → 次日低开 → 卖出。

---

## 缩量板买点总结

```
第一步：看趋势 → 144向上（首选）
第二步：看MACD → 正向，非顶背离
第三步：看均线位置 → 25/60/144附近
第四步：看组合 → 放量阴+缩量阳（最佳）
第五步：看开盘 → 高开6-8%（买）
第六步：次日 → 不涨停就卖
```

## 实战案例

| 股票 | 特征 | 结果 |
|------|------|------|
| 星期六 | 放量阴+缩量涨停 → 144上行 | 连续涨停 |
| 博天环境 | 横盘震荡+双阴+缩量板 | 多个涨停 |
| 宝塔实业 | 横盘+一字板+缩量板 | 短线盈利 |
| 廊坊发展 | 放量→缩量→再缩量涨停 | 3-4个板 |
| 龙升股份 | 一字板+巨量阴+缩量板 → 三浪位置 | 6个板 |
| 大连友谊 | 周线翻倍基础+缩量板 | 空间可期 |
