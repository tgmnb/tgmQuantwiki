# 日线翻倍技术（144均线+过左峰）

> 完整的波段/翻倍操作体系：以**144均线**定方向，以**MACD双线上零轴**确认多头，以**过左峰**作为买入信号。一波操作可赚30%-100%。

---

## 一、三种走势——只做第三种

| 走势 | 144均线状态 | 操作 |
|:----:|:----------:|:----:|
| ① 下跌 | 明显向下 | ❌ 回避 |
| ② 震荡 | 走平 | ⚠️ 容易买就跌卖就涨 |
| ③ **上涨** | **走平上翘/明显上翘** | ✅ **只做这一种** |

> 方向错了，南辕北辙。得趋势者赚大钱。

---

## 二、口诀（完整买点体系）

```
144均线走平上翘（趋势确立）
股价冲高回落，144正负10%以内（回踩）
MACD双线上零轴（进入多方区）
过左峰买进（突破信号）
```

---

## 三、左峰定义

**左峰 = 涨了一波最高的阴线的开盘价**

> 注意：不是最高价，是**开盘价**最高的那根阴线。

```
K线最高价很高，但是开盘价不是最高 → 不取这根
开盘价最高即使K线实体不大 → 取这根
```

---

## 四、过左峰的四种买入条件

股价运行到左峰线时，必须出现以下四种形态之一才能买入：

| # | 形态 | 强度 |
|:-:|:----|:----:|
| 1️⃣ | **涨停板过左峰** | ⭐⭐⭐⭐⭐ 最强 |
| 2️⃣ | **跳空高开过左峰** | ⭐⭐⭐⭐ |
| 3️⃣ | **倍量过左峰** | ⭐⭐⭐⭐ |
| 4️⃣ | **线上阴线买**（回踩左峰线收阴不破） | ⭐⭐⭐ |

> 四种形态出现任一即可买进。没有这四种信号 → 不过左峰→不买。

---

## 五、操作流程

### 买入

```
第一步：确认144均线走平上翘（用眼睛看是否明显高了）
第二步：股价冲高回落，回踩144附近（正负10%）
第三步：MACD双线已上零轴（多方区）
第四步：画出左峰线（涨一波最高阴线开盘价）
第五步：出现四种过峰信号之一 → 买入
```

### 卖出（风控）

```
跌了一波最低的阴线的收盘价 = 风控位
收盘价跌破 → 卖出
```

### 级别区分

| 类型 | 均线 | 行情规模 |
|:----:|:----:|:--------:|
| **日线翻倍** | 144日线 | 一波30%-100% |
| **波段** | 较短周期 | 较小级别行情 |

## 六、实战案例

### 华伍股份
- 144均线从向下→走平→明显上翘
- 股价回踩144正负10% → MACD上零轴
- 左峰线突破 → 买入 → 一波上涨

### 长春经开
- 横有多长竖有多高
- 144走平上翘 + MACD零轴上
- 左峰6.26 → 倍量过左峰 → 三浪买点 → 翻倍

### 豫金刚石
- 144明显上翘 + MACD上水
- 左峰线 → 线上阴线买 → 一波拉升

---

## 七、144均线的意义

- 144均线只给**方向**，不给买点
- 144均线走平上翘 = 可以关注
- 144均线走平下压 = 回避
- 一只股票16年只有3波144级别行情（约5年一波）
