# 必胜模式与死亡模式（25日线/三线红/梅开二度）

> 以**25日线**为生命线的交易系统。股价在25线上方 + MACD红柱 = **必胜模式**（做多）。股价在25线下方 + MACD绿柱 = **死亡模式**（做空/回避）。级别由周线和月线的MACD状态决定。

---

## 一、核心定义

### 25日线 = 生命线

操盘手系统中，25日线（25MA）是核心均线。适用于任何周期（日线/周线/月线）。

### 两种模式

| 模式 | 位置 | MACD | 方向 |
|:----:|:----:|:----:|:----:|
| **必胜模式** ✅ | 股价**在**25线上方 | 红柱 | 做多/持股 |
| **死亡模式** ❌ | 股价**在**25线下方 | 绿柱 | 做空/回避 |

### 判定规则

```
必胜模式：
  收盘价在25线上方 + 日线MACD红柱 = 必胜

死亡模式：
  收盘价在25线下方 + 日线MACD绿柱 = 死亡
```

> 看收盘价（2:50后确认），盘中不算。

---

## 二、小必胜 vs 大必胜（三线红）

### 小必胜模式

只有日线MACD红柱，周线和月线可能是绿的。

→ 预期：**小涨或反弹**，难有大作为。

### 大必胜模式（三线红）⭐⭐⭐⭐⭐

| 周期 | MACD状态 |
|:----:|:--------:|
| **日线** | ✅ **红柱** |
| **周线** | ✅ **红柱** |
| **月线** | ✅ **红柱** |

→ 预期：**大级别主升浪**，翻倍起步。

> 周线和月线的红柱是推动力，买点从日线上找。

---

## 三、两种必胜形态

### 初级必胜

```
股价温和翻上25日线 → 回踩25日线
  → 线上收阴线（回踩不破25）
  → 下方MACD红柱
  = 买点（线上阴线买）
```

### 梅开二度 ⭐⭐⭐⭐⭐（最强）

```
股价站稳25后 → 大阴线故意跌破25（洗盘）
  → 放量大阳线快速翻上25（收回）
  → 在25线上方开阴线
  → 下方MACD红柱
  = 买点（洗盘后拉升）
```

**特征**：大幅跌破+快速收回+放量。

> 梅开二度比初级必胜更强，往往是一波主升浪的起点。

---

## 四、操作口诀

### 买入

```
线上阴线买（股价在25线上方收阴线 + MACD红柱）
买错也要买（因为大势向上，回踩就是买点）
```

### 卖出

```
线下阳线抛（股价在25线下方收阳线 + MACD绿柱）
抛错也要抛（因为大势向下，反弹就是卖点）
```

---

## 五、死亡模式详解

### 识别

```
股价跌破25日线 → 反抽25日线
  → 收阳线（线下阳线）
  → MACD绿柱
  = 卖出点（死亡模式）
```

### 死亡模式的杀伤力

```
大级别均线在头顶上压着
  + 25线下阳线
  + MACD绿柱
  = 大跌前兆
```

---

## 六、操作流程

### 买入（大必胜模式）

```
第一步：确认三线红——日/周/月MACD都是红柱
第二步：股价在25日线上方
第三步：出现线上阴线（回踩不破25）
  或：梅开二度（跌破→快速翻上→开阴线）
第四步：倍量切出现（可选，更强信号）
第五步：买入

风控：阴线的最低价或缺口支撑
```

### 卖出（死亡模式出现时）

```
第一步：确认股价跌破25日线
第二步：确认日线MACD转绿柱
第三步：线下阳线（反抽）出现
  → 卖出
```
