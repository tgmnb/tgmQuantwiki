# 妖股与龙头战法（板块定龙术+连板见顶判定）

> 妖股每年诞生5-10只，集中在7月底证券暴动后的1-3个月内。不分行业板块，但必须具备**中线翻倍基础**或**月线/周线历史天量**两大基因之一。
>
> **认知核心**：妖股是在板块搭好台子后唱戏的。证券集体涨停搭台→妖股上台表演。

---

## 一、妖股的诞生时机

### 核心信号：证券集体涨停

```
2023年7月28日：12家证券涨停板
        ↓
信号含义：
  证券集体涨停 = 至少1-3个月行情
  随后的1-3个月 = 妖股诞生的黄金窗口
        ↓
今年（2023）妖股窗口：8月 - 10月底
预计诞生5-10倍的个股
```

### 为什么是证券集体涨停？

证券涨停代表**政策东风已起，台子已经搭好**。大盘接下来的调整（春雨）只是短期现象——"一场春雨一场暖，十场春雨到夏天"。不要因为中间几根阴线就认为熊市来了。

### 历史规律

- 每年大约诞生5-10只五倍、十倍的妖股
- 上半年一批，下半年一批（7-10月）
- 妖股诞生不区分行业板块
- **中国石油、中国石化、四大行等巨型股除外**

---

## 二、妖股的两大基因

妖股必须具备**至少其中之一**：

### 基因1：中线翻倍基础

```
特征：第一波已经走出中线翻倍
      第一波结束后调整
      板块调整完毕重新走好
      妖股启动第二波（1.5-2波）

核心逻辑：
  板块刚刚启动触底，里面1-2只个股提前走连板（3-4板）
  板块开始走好时，这只个股调整完毕
  板块+个股共振 → 第二波主升 → 妖股诞生
```

**实战案例**：恒银科技
- 板块未动时已走3连板
- 板块走好时它调整完毕
- 预期走5板以上

### 基因2：月线/周线历史天量

```
特征：不是日线历史天量
      是月线或周线级别的历史天量
      直接从底部爆量拉升（没有中线基础）
      无端放量，直接翻5-10倍

核心口诀：
  天量见天价 → 见了天量见天价，后边就要回老家
  但是！如果高开高走不跌破 → 妖股形态确认
```

### 两种基因的盘面表现

| 基因类型 | 表现方式 | 例子 |
|:--------:|:---------|:----:|
| 中线翻倍型 | 有基础→调整→二波主升 | 恒银科技、中马传动、浙江世宝 |
| 天量暴起型 | 无基础→直接放历史天量拉升 | 蒙古栎（301487，10倍） |

## 三、板块定龙术（如何定龙一龙二龙三）

### 工具：通达信细分行业

```
步骤：
1. 按60→涨幅榜 → 点板块指数看细分行业
2. 找到涨停家数最多的行业（环保/软件服务等）
3. 点开该行业 → 查看板块内所有个股
4. 点"几天几板"排序 → 自动按连板天数排序
5. 点"连板天数"排序 → 看连板高度
```

### 龙一判定标准

```
龙一 = 本板块内连板天数最多的那只

10%板龙一 vs 20%板龙一：
  两者是独立计算身位的
  10%的龙一和20%的龙一要分开看
  20%的龙一虽然涨停板数少，但涨幅大
```

**身位概念**：

```
身位 = 领先几个板
  大哥4板 → 小弟2板 → 落后2个身位
  大哥倒下了，小弟不可能撑住

规则：
  龙头不倒，板块不倒
  龙头倒了，小弟全部完蛋
  龙头首阴后可以反包，反包要卖
```

### 龙一的具体判定流程

```
① 打开通达信→行业板块→涨停家数最多
② 打开该行业→点"几天几板"
③ 连板最高的 = 龙一
④ 连板第二 = 龙二（但和龙一可能差好几个身位）
⑤ 剩下的全是跟随者，不要碰
```

**补充规则**：当手板（首板）中有次日开一字板的，会推着现有的二板股晋级成新龙头。

```
手板中多个一字板 → 推动二板晋级三板
→ 二板中可能有一个晋级成功变成新龙头
→ 这就是"小弟推老大"
```

## 四、龙头首阴战法

### 什么是龙头首阴？

```
龙头第一次收阴线 = 首阴
必须是龙一（不是龙二龙三）的首阴
不是普通的阴线，是龙头连板后的第一根阴线
```

### 首阴的操作

```
首阴出现 → 次日：
  如果低开/平开 → 可以买入（约-7%的位置）
  如果高开 → 执行一板半到两板半的卖出策略

首阴阳线反包 → 反包当天卖出（不是买入！）

核心口诀：
  龙一首阴反包卖掉
  首阴战法用在龙一身上才有用
```

### 案例1：长青科技

```
长青科技是运输设备里的龙一
首阴出现方式特殊：用一根K线走了两根K线的活
  → 先跌8%（盘中瀑布杀）→ 尾盘反包
  → 这一天既是阴线也是阳线
  → 次日直接走，不等了
```

### 案例2：正和生态

```
正和生态（环境保护龙一）
  前一天放量跳空 → 预期一板半
  结果低开 → 给了机会 → 地天板
  地天板不能追，买点在前一天
```

### 案例3：国泰环保

```
国泰环保（20%龙一）
  首阴出现
  反包 → 反包当天卖出
  明天可以直接低开杀，也可以冲一下再杀
  两根K线合一的情况要小心
```

## 五、连板妖股的见顶判定（量价时空倒着念）

### 核心口诀

```
量价时空 → 倒着念 → 空-间-价-量
见顶判定只用最后一层：量

当连板出现后：
  时间不用看
  空间不用看（已经极致发散）
  价格不用看（不知道能打到多少）
  只看量 → 放出天量 = 见顶
```

### 天量见顶的三种后续走势

```
放出天量涨停板后：

【涨的三种】
① 开一字板1-2个 → 然后直接跌停往下灌
② 跳空板再冲一个半 → 杀3天
③ 低开高走缩量 → 再冲一次 → 跌停

【跌的两种】
④ 直接闷杀一字跌停
⑤ 低开直接跌停
```

### 量价时空的实用顺序

```
正常情况：先看空间→再看时间→再看价格→最后看量
连板妖股（已经极致）：跳过前三层，只看量
量放出天量 → 直接结束
```

### 案例：弘业期货

```
弘业期货：
  普通版放天量 → 判死刑
  直接低开杀死 → 一冲就是最后高点
  千万不要吃了这个亏
```

## 六、普通股→牛股→妖股的升级条件

### 升级路径

```
普通股
  → 有资金进场 + 有量价承接 → 牛股
  → 有妖股基因 + 连板特征 → 妖股

牛股 vs 妖股的区别：
  牛股 = 有资金，但不连板（如陕西煤业）
  妖股 = 有妖股基因 + 连板（如长青科技）
```

### 妖股基因判定

```
① 月线"红似一团火，阴似满天星"
   → 阳高阴缩，红肥绿瘦
   → 阳量高，阴量低
② 出现连板特征（2板即可定龙一）
③ 换手板（不是一字板缩量板）
④ 流通盘小（如3380万的长青科技）
⑤ 次新股有优势（头顶一片蓝天）
⑥ 周线/月线不死叉
```

### 普通股走牛的判断

**陕西煤业案例**：
```
放量（二倍量）+ 量价承接（无阴高于阳）
= 主动做多信号

高开最好（开盘高于昨日收盘）
空间测算：一浪×1.5/2/3/4算三浪主升
周线阴牌战法配合
```

### 新股/次新股的特殊价值

```
新股上方没有套牢盘（头顶一片蓝天）
出现脉冲量/移动量后有做多机会
关键：成交量是历史上最大
  放出这个量的K线最低价=关键支撑位
  不跌破此支撑 → 可以阳包阴做多
```

## 七、龙头交接规则

### 龙头倒下，新龙上位

```
老龙头5板封不住 → 下一个4板的变成新龙头
如开开实业（5板）倒下 → 长青科技（4板）接棒

龙头交接的条件：
  新龙头必须有连板基础（至少2板）
  新龙头和老龙头在同一大方向板块内
  交接发生在老龙头断层那一天
```

### 2023年8月的龙头格局

```
开开实业（5板）— 当前主板龙一
长青科技（4板）— 次新股，有妖气，新龙头候选
     ↓
如果开开实业明天板不住了
长青科技如果能低开高走+缩量
→ 接棒成妖 → 可走10+板
```

## 八、盘中操作要点

### 买点
- **龙一首阴**：低开-7%左右买入
- **板后二号位**：早盘回杀时上车
- **阳包阴**：放量阳覆盖阴线
- **空间位**：用15分钟/5分钟空间测算

### 卖点
- **放天量卖出**：见顶信号不用等
- **反包卖出**：龙一首阴反包当日走
- **一板半**：放量板+跳空 = 一板半结束
- **创新高=卖出**（周线不支持时尤其如此）

### 风控
```
破底斩立决（跌破风控位就砍，不犹豫）
买入阳线最低价做风控
换手板做妖股要有挂掉的准备
```

## 九、综合实战框架

### 大盘判断
```
证券集体涨停 → 台子已搭好 → 看好后市
中间的阴线调整 = 春雨，不是冬天来了
```

### 板块判断
```
涨停家数最多→龙头判断（身位）→龙头首阴/反包
板块是否快熄火？→ 看龙一连板天数
```

### 妖股判断
```
两大基因（中线翻倍基础/月周历史天量）
连板特征（换手板+次新+流通盘小）
月线红肥绿瘦
```

### 见顶判断
```
只看量：放出天量就走
```
