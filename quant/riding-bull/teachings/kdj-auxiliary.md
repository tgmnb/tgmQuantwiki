# KDJ辅助技巧（负值买入法 + 20以下金叉）

> KDJ是辅助工具，配合空间技术、MACD浪型使用效果更好。两种买法：**J值负值买入**（抢反弹）和**20以下金叉**（波段起涨点）。

---

## 一、J值负值买入法

### 前提条件

**核心要求：下跌必须顺滑。**

> 顺滑 = 下跌过程中**全是阴线**，中间没有大阳线打断。

### 三种可接受形态

| # | 左侧上涨 | 下跌 | 评分 |
|:-:|:--------:|:----:|:----:|
| 1️⃣ | 顺滑上涨（MACD一浪） | 顺滑下跌 | ⭐⭐⭐⭐⭐ 最佳 |
| 2️⃣ | 涨得不顺滑 | **顺滑下跌** | ⭐⭐⭐⭐ |
| 3️⃣ | 连续涨停板 | 连续跌停板+顺滑 | ⭐⭐⭐ |

> **跌得顺滑比涨得顺滑更重要！**

### 第一步：确认死叉位置

```
死叉当天，J值必须 > 50
  → 大于50说明下跌空间够大
  → 小于50则下跌空间不足，不操作
```

### 第二步：计算买入日

| 情况 | 买入日 | 说明 |
|:----:|:------:|:------|
| **正常** | **负值第3天** | 买在开盘价/收盘价 |
| **极致** | **负值第5天** | 如果正值只走1-2天就变负，则容易走5天 |

> 负值天数从J值转为负数第一天开始算。

### 买入方式

```
① 开盘价买入（主力仓位）
② 盘中跌到4.3%位置加仓（次级仓位）
③ 收盘价买入（如果没跌到4.3%）
```

### 卖出

- 短线：当天冲7%就卖出，或次日冲高卖
- 波段：配合MACD一浪回踩买入 → 吃三浪主升浪

---

## 二、20以下金叉买入法

### 核心

```
KDJ的K值和D值（不看J值）在20以下金叉
  → 短期反弹或波段起涨点
```

### 使用级别

| 级别 | 用途 |
|:----:|:------|
| **日线** | 波段反弹/起涨 |
| **60分钟** | 短线反弹（配合MACD即将金叉效果更佳） |

### 配合条件

```
20以下金叉 + MACD绿柱缩短（大变小）
  + 不跌破前阴实底
  = 买入信号 ⭐⭐⭐⭐
```

---

## 三、总结

```
J值负值买入法 → 抢反弹（顺滑下跌+死叉J>50+负值第3/5天）
20以下金叉法 → 波段起涨（K/D值20以下金叉+日线/60分钟）
```

两者都是辅助工具，不能单独决策，需要配合：
- MACD十大技术理论（浪型/背离）
- 阴线平台（支撑/风控）
- 空间测算（目标位）
