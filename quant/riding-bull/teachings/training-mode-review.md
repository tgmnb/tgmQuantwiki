# 训练模式设置与复盘流程（4-7练习法）

> 通达信自带训练模式（按4-7），隐藏股票名称和时间，随机复盘历史K线，练习买卖点判断。配合换手率+涨幅榜选股复盘，效果最佳。

---

## 一、训练模式设置

### 步骤一：基本设置

```
系统设置 (Ctrl+D) → 设置三
  → "分析图中移动时显示小窗口" → ❌ 不要打勾
```

否则会弹出股票名称和时间窗口，影响练习。

### 步骤二：隐藏股票名称

```
方法1：点击左上角灰色小框 → 关闭股票名称显示
方法2：用划线工具（小笔图标）拖到左上角挡住名称
```

### 步骤三：进入训练模式

```
键盘按 4 + 7 → 右下角弹出训练模式框 → 回车
```

### 步骤四：隐藏日期和品种

在训练模式框中：

| 选项 | 作用 |
|:----:|:------|
| ✅ **隐藏当前品种** | 隐藏股票名称 |
| ✅ **隐藏日期** | 隐藏K线日期 |
| 拖到左下角 | 挡住剩余名称显示 |

完成后只显示K线和价格，不显示任何影响判断的信息。

### 功能键

| 按钮 | 作用 |
|:----:|:------|
| **下一根K线** | 手动前进一根 |
| **买入** | 练习买入（可全仓/部分） |
| **卖出** | 练习卖出 |
| **自动播放** | 自动前进K线 |
| **随机** | 随机跳转到不同时间段 |
| **重新训练** | 从头开始 |

## 二、复盘选股流程

> 每天收盘后用2-4小时复盘，比盘中盯盘4小时更有价值。

### 第一步：按换手率排序

```
按60 → 回车 → 点"换手"列标题
  → 从高到低排序
  → 翻到换手率6-7%为止（约15页）
  → 逐一快速浏览
```

### 第二步：按涨幅排序

```
点"涨幅"列标题 → 从高到低
  → 翻到涨幅6%为止
  → 与换手率筛选结果互补
```

### 第三步：建立自选股池

```
① 换手率7%以上的股票 → 加入新建板块（如"今日换手"）
② 涨幅6%以上的股票 → 加入同一板块
③ 删除新股（C开头）和688（科创板）
④ 删除ST股票
⑤ 最终保留约300-400只股票
```

### 第四步：逐一复盘

```
对自选股池中的每只股票：
  ① 看月线MACD（三线红？）
  ② 看周线MACD（是否推升？）
  ③ 看日线形态（是否有买点？）
  ④ 快速判断是否可以操作
```

## 三、训练模式练习方法

### 随机选股

```
在训练模式中：
  → 用鼠标滚轮随机翻股票
  → 你不知道是什么股票、什么时间
  → 只看K线形态做判断
```

### 操作原则

```
月线级别：
  先看月线 → 再开周线 → 最后日线
  三线红（月周日均红柱）= 最佳标的
  数字操盘验证顶底

买入信号：
  放量突破/阳包阴/大绿小绿/线上阴线

卖出信号：
  顶背离/小红/放量滞涨/跳空低开
```

### 练习目标

```
通过大量随机练习，积累"盘感"：
  经历牛/熊/震荡/猴市各种行情
  验证自己的买点是否准确
  验证自己的卖点是否及时
```

## 四、复盘的核心要点

### 换手率的含义

- 换手率高 = 筹码交换活跃 = 有资金关注
- 妖股和牛股启动时，换手率通常在7%以上
- 但换手率高 ≠ 一定涨——需配合量价时空

### 复盘比盯盘重要

```
盘中4小时 → 情绪波动大，容易冲动交易
盘后4小时 → 冷静分析，系统性复盘
         → 比盘中4小时更有价值
```
