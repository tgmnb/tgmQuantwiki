# 大波段/小波段空间测算 + 系数理论

> 用均线死叉/金叉点结合百分比箱体，以及最低价×系数，来测算股价的目标位和支撑位。属于操盘手技术的空间测量体系。

---

## 一、大波段空间测算

### 原理

用**一波的最高价**和**5日线死叉30日线**的死叉点，画百分比箱体。

### 画法

```
① 找到一波的最高价
② 找到5日死叉30日的死叉点
③ 以最高价为箱顶，死叉点为箱底，画百分比箱体
④ 向下平移箱体：一格、两格、三格（三段为一组）
```

每**三段**（三个箱体高度）为一个整体。

### 用法

| 位置 | 含义 |
|------|------|
| 箱体50%位 | 支撑/压力 |
| 箱体100%位（平移一格） | 强压力 |
| 三段完成 | 大波段级别压力到位 |

### 配合验证

大波段空间测算 + 其他技术**共振**时信号最强：
- 阴线平台（上线/下线）
- 百分比
- MACD顶/底背离
- 数字操盘法则（尾数）
- 系数（1.3/1.26）

> 技术之间不会矛盾——它们同时指向同一个位置就是强共振。

## 二、小波段空间测算

### 原理

用**2日和5日均线**（2日定方向，5日定成本），配合**144均线**判断趋势。

### 画法

```
① 144均线走平上翘 = 趋势确认
② 找一波干净（圆滑）上涨走势
③ 取这一波的金叉点和死叉点
④ 以金叉点为箱底，死叉点为箱顶，画百分比箱体
⑤ 关注50%位（支撑位）
⑥ 箱体向上平移测压力
```

### 关键位

| 位置 | 含义 |
|------|------|
| **50%位** | **精准支撑位**——回踩50%是买点 |
| 100%位（平移一格） | 压力位 |
| 150%位 | 更高一级压力 |

### 口诀

```
2日定方向，5日定成本
15日线上不做空，42日线上做波段
103日线上做趋势
```

### 与大波段的区别

| | 大波段 | 小波段 |
|--|--------|--------|
| 均线 | 5日+30日 | 2日+5日+144 |
| 周期 | 中线 | 短线/超短线 |
| 精准度 | 相对粗 | 更精细 |

## 三、系数理论（1.3 / 1.26）

### 大系数和小系数

| 系数 | 数值 | 对应涨幅 | 说明 |
|------|------|---------|------|
| **大系数** | **1.3** | 30% | 基础系数 |
| **小系数** | **1.26** | 26% | 更常用，两者差4% |

### 用法一：从低点测高点（上涨目标）

```
目标价 = 最低价 × 系数
```

**案例**：红宝丽（002165）
- 最低价 6.55 × 1.26 = 7.12 → 实际最高价 7.17（误差0.05）
- 最低价 6.58 × 1.26 = 8.30 → 实际高点 8.30+（精准）

### 用法二：从高点测支撑（下跌目标）

```
支撑价 = 最高价 ÷ 系数
```

**案例**：
- 最高价 49.84 ÷ 1.3 = 38.33 → 实际支撑 38.35（精准到位）

### 使用规则

1. **必须配合其他技术使用**——系数单独使用只是参考
2. MACD顶/底背离 + 系数到位 = 强卖出/买入信号
3. 阴线平台 + 百分比 + 系数同时指向同一位 = 强共振
4. 系数到位 + 数字操盘法则尾数验证 = 精准度高

### 实战判断

```
股价涨到系数目标位附近：
  + MACD顶背离（确认顶部）
  + 小红（八字要诀：小红不能买）
  + 数字操盘尾数见顶信号
  = 卖出 ⭐⭐⭐⭐⭐
```

```
股价跌到系数目标位附近：
  + MACD底背离（确认底部）
  + 小绿（八字要诀：小绿后面有大红）
  + 数字操盘尾数见底信号
  = 买入 ⭐⭐⭐⭐⭐
```

## 四、综合共振示例

### 三步判断法

```
第一步：系数算目标（1.3/1.26 → 价格到位）
第二步：百分比箱体验证（平移箱体 → 压力位）
第三步：阴线平台/MACD背离确认（卖点或买点）
```

### 案例流程

```
最低价 6.55 × 1.26 = 7.12（系数目标）
  + 百分比箱体平移到位
  + MACD顶背离（A→B→C）
  + 大红变大红中小红
  = 无条件卖出
```
