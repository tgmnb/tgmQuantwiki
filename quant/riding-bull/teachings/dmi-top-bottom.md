# DMI指标判断顶底

> DMI指标的ADX数值用于判断指数或个股**一波相对区间的顶部或底部**。准确率约70-80%，需配合自然箱体、MACD等技术共同研判。

---

## 核心口诀

> **ADX ≥ 60，不是一波顶部，就是一波底部。**

- ADX 数值 ≥ 60（或等于60）= 信号触发
- 顶和底都会触发此信号 → **需要结合其他技术判断方向**

---

## 使用方法

### 识别顶底

当ADX上升到60以上时：
- 若股价处于上涨波段高位 → 大概率是**相对顶部**
- 若股价处于下跌波段低位 → 大概率是**相对底部**

### 主升浪确认（ADX回压再突破）

```
ADX触60 → 回压（数值下降） → 股价再次突破创新高 → 主升浪
```

> 当ADX见了60之后回压，股价再次突破时，结合MACD三浪/五浪结构，往往演变成**大级别主升浪**。

---

## 配合技术

| 配合技术 | 用途 |
|---------|------|
| **自然箱体/大系数** | 验证压力位和支撑位 |
| **MACD十大技术理论** | 判断浪形位置（三浪/五浪）和背离状态 |
| **数字操盘法则** | 尾数数字验证顶底 |
| **1.3系数** | 计算波段目标位（最低价×1.3=压力位） |

### 协同使用示例

```
ADX ≥ 60（见顶信号）
    + MACD顶背离（顶部确认）
    + 自然箱体二一位已到（空间到位）
    = 卖出信号 ⭐⭐⭐⭐⭐
```

```
ADX ≥ 60（见底信号）
    + MACD底背离（底部确认）
    + 数字操盘法则尾数验证（底部结构）
    = 买入信号 ⭐⭐⭐⭐⭐
```

---

## 多周期适用

DMI指标适用于**任何周期**：
- 日线、周线、月线
- 60分钟、30分钟、15分钟
- 5分钟、1分钟

> 下跌途中遇到DMI见底信号 + MACD底背离 → 抄底概率极高。

---

## 实战案例

### 五粮液
- 多个波段的顶部/底部ADX均触及60
- ADX回压后股价突破 → 三浪主升浪启动
- 配合MACD柱体面积判断：面积大于前方一切调整 → 三浪确认

### 上证指数
- 多次在波段顶/底部ADX达到60
- 结合自然箱体判断空间是否到位

## 880003平均股价+DMI+老鹰嘴

> **880003平均股价指数**周线配合DMI，是判断市场大顶大底的重要工具。配合60日线"老鹰嘴"形态，可靠性更高。

### 880003平均股价指数

```
代码：880003
名称：平均股价指数（通达信）
作用：反映全市场个股的平均价格水平
      不受权重股影响，比上证指数更真实

为什么用平均股价：
  上证指数被银行/石油等权重绑架
  平均股价反映的是"大多数股票"的真实状态
  判断大顶大底时，平均股价比上证更可靠
```

### 周线DMI信号

```
平均股价周线DMI出信号（ADX ≥ 60）：
  → 市场阶段底部附近
  → 不是精确买点，而是"底部区域"信号
  → 信号出现后：
      至少先横盘不跌
      配合MACD底背离 → 反弹/反转启动
      等待日线级别买点确认
```

| 信号 | 含义 | 后续操作 |
|:----:|:----:|:---------|
| 周线ADX≥60 + 价格低位 | 阶段底部信号 | 准备选股，等日线买点 |
| 周线ADX≥60 + 价格高位 | 阶段顶部信号 | 减仓或清仓 |
| 周线ADX回压后再涨 | 主升浪确认 | 加仓做多 |

### 60日线老鹰嘴

> **60日均线向下弯曲如老鹰嘴 = 看跌信号。**

```
老鹰嘴形态识别：
  60日均线原本走平或向上
  ↓
  均线末端开始向下弯曲
  ↓
  弯曲弧度类似老鹰的嘴
  ↓
  信号：看跌，至少短期承压

原理：
  60日线代表中期趋势
  60日线向下弯 = 中期趋势转弱
  即使股价还在60线上方
  老鹰嘴出现后大概率要跌
```

### 老鹰嘴的实战判断

```
确认条件：
  ① 60日线方向从走平/向上转为向下
  ② 弯曲发生在均线末端（最新一段）
  ③ 股价在60日线附近或下方

操作：
  老鹰嘴出现 → 减仓/不买
  老鹰嘴消失（60日线重新走平向上）→ 可重新入场
  老鹰嘴 + 股价跌破60日线 → 强烈看跌，清仓
```

### 综合研判框架

```
大底判断流程：
  ① 平均股价周线DMI出信号（ADX≥60）
  ② 周线MACD绿柱开始缩短（底背离雏形）
  ③ 60日线老鹰嘴消失，开始走平
  ④ 日线出现阳包阴/倍量 → 买入

大顶判断流程：
  ① 平均股价周线DMI出信号（ADX≥60）
  ② 周线MACD红柱缩短（顶背离雏形）
  ③ 60日线出现老鹰嘴
  ④ 日线出现放量滞涨 → 卖出
```

### 口诀

```
平均股价出信号，阶段底部附近到
60日线老鹰嘴，不跌也要震一震
DMI回压再突破，主升浪里赚大钱
老鹰嘴消均线平，重新入场好时机
```
