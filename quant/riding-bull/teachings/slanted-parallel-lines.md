# 斜横线与平行线（划线技术+黄金坑）

> 划线是最直观的技术辅助工具。取两个高点画斜横线（下降趋势线），取高点的阴实底/阴实顶画平行线（水平线），两条线交汇处容易出中大阳线或中大阴线。
>
> **黄金坑**：跌破前低后快速站回，配合放量/倍量/涨停板，容易走V型反转。
>
> 划线不预测未来，只告诉你**股价当前所处的位置**和**哪里容易变盘**。

---

## 一、基础设置

### 坐标要求
- **日线、前复权、量赋权**（与之前所有教学一致）
- **不要用对数坐标**！对数坐标会扭曲K线，让划线失真
- 用最普通的百分比坐标，贴近自然就是最真实

### 画线工具
通达信画线工具中的三个基本工具：
1. **斜线**（画斜横线/下降趋势线）
2. **平行线**（画水平线/阴实顶阴实底线）
3. **射线**（画延伸线）

---

## 二、斜横线（下降趋势线）

### 画法

```
取两个高点：前一个高点A + 后一个高点B
条件：B ≤ A（后高不高于前高，或低于前高）
连线：从A向右拖到B，形成一条向下倾斜的直线

规则：
  1. 连线中间不能穿插任何K线
  2. 必须干净利落，K线全部在线条一侧
  3. 一旦K线突破此线 → 趋势可能反转
  4. 一旦跌破底部 → 重新取高点画线
```

### 定义方式

股价运行过程中，**不能提前取点**。当后一个高点出现（需要3根K线确认），才能确定这个高点，再画线。

```
走到位置A（高点出现）→ 不画线（还未确认）
走到位置B（比A低的高点出现，3根K线确认）→ 画斜横线
走到位置C（突破线）→ 关注买点
走到位置D（跌破底部）→ 重新取点画线
```

### 跌破后的处理

如果画好线后股价跌破底部（阴实底），需要**重新取点**：
- 新的反弹高点 + 之前的低点 重新连线
- 不能用原来的线继续穿

## 三、平行线（水平支撑压力线）

### 画法

完成斜横线之后，画两条平行线：

```
斜横线（A→B两个高点连线）画好后，取：
  第二条线 = 第二个高点B的阴实底（支撑线）
  第三条线 = 第二个高点B的阴实顶（压力线）

即：
  斜横线：A高点 → B高点
  平行线1：B高点的阴线实体最低价（支撑位）
  平行线2：B高点的阴线实体最高价（压力位）
```

### 突破与跌破

| 情况 | 含义 | 操作 |
|:----:|:----|:----:|
| **突破斜横线** | 下降趋势被打破 | 关注买点 |
| **突破平行线（阴实顶）** | 压力变支撑 | 放量突破可买入 |
| **跌破平行线（阴实底）** | 支撑变压力 | 止损/观望 |
| **两线同时突破** | 最强反转信号 | **容易出现中阳线/大阳线** |
| **两线同时跌破** | 最强下跌信号 | 容易出现中阴线/大阴线 |

### 核心口诀

> **斜横线与平行线上，容易出中阳线、大阳线、中阴线、大阴线。**

这两条线交汇的位置就是变盘节点。

## 四、黄金坑（Golden Pit）

### 定义

```
黄金坑 = 股价跌破前期低点（阴实底）→ 快速放量站回
      ↓
        左方下跌横盘震荡出的底部
        → 被一根阴线跌破
        → 随后放量（倍量/增量资金）快速站回
        → 产生V型反转
```

### 三种级别

| 级别 | 特征 | 操作策略 |
|:----:|:----|:---------|
| **超短线黄金坑** | 1-3根K线破底后快速站回，涨停板突破 | 阳线最低价做风控，短线快进快出 |
| **波段黄金坑** | 跌破前低，5-10根K线放量站回 | 波段级别买点，阴实底做风控 |
| **大级别黄金坑** | 月线/周线级别的破底后站回 | 配合MACD级别判定，重大买点 |

### 黄金坑的判定标准

```
1. 必须有一个明确的前期底部（阴实底）
2. 股价跌破这个底部
3. 放量/倍量/涨停板站回（不能是缩量）
4. 站回后回踩不破这个底部
5. 出现阳包阴确认
```

**关键区分**：

| 形态 | 特征 | 判断 |
|:----:|:----|:----:|
| **真黄金坑** | 跌破→放量→快速站回→涨停板/大阳线突破 | ✅ 买点 |
| **假跌破** | 跌破→缩量→站不回→继续走弱 | ❌ 观望 |
| **V型反转** | 跌破→直接V型拉回（涨停板最有效） | ✅ 超强信号 |
| **普通坑** | 跌破→磨蹭→慢慢回 | ⚠️ 等阳包阴确认 |

### 黄金坑与金山谷/银山谷的关系

```
金山谷：上去→下来→再突破（形成山谷）
银山谷：上去→下来→形成谷底（比金山谷小）
黄金坑：先跌破前低→再快速站回（比金山谷更早出现）

顺序：
  黄金坑出现 → 站回 → 金山谷/银山谷形成 → 主升浪
```

黄金坑是在金山谷之前出现的快速破底信号，**比金山谷更早、更暴力**。

## 五、划线实战应用

### 应用1：上证指数划线示例

取上证的两个高点连线：
- 前一个高点 → 后一个高点 → 画斜横线
- 取第二个高点的阴实底画平行线
- 第三条线（阴实顶）画平行线

**结果**：
- 斜横线+平行线交汇处 → 预测出中阳/大阳
- "一根穿云箭不够，两根" — 两根线同时突破时最强

### 应用2：个股操盘

```
1. 找到波段的两个高点 → 画斜横线
2. 取第二个高点的阴实底 → 画平行线（支撑）
3. 取第二个高点的阴实顶 → 画平行线（压力）
4. 观察股价运行到两条线的位置
5. 线交汇处放量 + MACD配合 → 买入/卖出
```

### 应用3：黄金坑实战买点

```
① 发现跌破前低（创了新低）
② 等待放量/倍量/涨停板站回
③ 确认突破了黄金坑边缘（前低的阴实底）
④ 阳包阴 = 买点
⑤ 风控：阳线最低价 或 黄金坑边缘
⑥ 目标：空间测算（乒乓球理论50位/前高）
```

## 六、MACD第一区间选股法（本课实战案例）

### 什么是MACD第一区间？

```
MACD三大区间：
  第一区间 = MACD红柱最大、最高的一波（大红）
  第二区间 = 红柱中等（中红）
  第三区间 = 红柱最小（小红）

第一区间条件：
  MACD红柱高于左方1-2波（高于一切）
  是动能的巅峰区域
```

### 第一区间选股模型

```
① MACD第一区间（大红/中红）
② 有增量资金入场（放量/倍量/涨停板）
③ 出现阳包阴
④ 周线/月线至少一个是红柱
⑤ 用空间技术测目标位

成功率极高的模型：大红+增量+阳包阴+周月红
```

### 实战案例

该课中展示的实战案例全部使用此模型：

| 股票 | MACD区间 | 信号 | 结果 |
|:----:|:--------:|:----|:----:|
| 财信发展 | 第一区间+周线不金叉+月线阳线 | 阳包阴 | 涨停板 |
| 利尔化学 | 第一区间（周线未金叉偏弱） | 尾盘阳包阴买入 | 次日涨停 |
| 利民股份 | 第一区间+周线红柱底背离 | 阳包阴买入 | 涨停板 |
| 威孚高科 | 第一区间+月周线红 | 阳包阴买入 | 次日回调→第三日涨停 |
| 如意集团 | 第一区间+资金进场 | 阳包阴买入 | 涨停+冲高 |
| 通鼎互联 | 第一区间+黄金坑+增量入场 | 阳包阴买入 | 涨停板 |

**风控统一**：跌破买入阳线的最低价无条件离场。

## 七、量价时空的本质（本课纠正的重要认知）

### 价格引领一切，不是量

> **错误认知**：放量了就会涨。
> **正确认知**：价格（空间）动了，量才会跟上。不是量引领指标，是**价格引领指标**。

```
价格不动 → 放量也没用 → 均线不动、MACD不动 → 全无作用
价格一活 → 量自然会跟 → 均线发散、MACD金叉 → 全都活了

所以：空间为王，量是从属
         ↓
先看空间 → 再看时间 → 再看价格 → 最后看量
```

**实战意义**：
- 不要看到放量就冲进去
- 先看价格形态是否具备上涨空间
- 价格到位了，量自然来
- 价格不到位，放天量也不涨

## 八、划线核心要点总结

### 斜横线
- 两点取高点（后高≤前高）
- 中间不穿插K线
- 跌破底部重新画

### 平行线
- 阴实底（支撑）和阴实顶（压力）
- 突破/跌破出大K线

### 黄金坑
- 跌破→放量站回=V型反转
- 倍量/涨停板站回最强
- 结合MACD大绿小绿使用

### 划线+MACD配合

```
划线告诉位置（在哪儿）
MACD告诉级别（多大行情）
量能告诉强度（多大概率）
```
