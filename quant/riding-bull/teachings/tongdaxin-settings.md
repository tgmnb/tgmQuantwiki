# 通达信软件正确设置

> 操盘手技术的软件基础。设置错误 → 所有技术指标失真。**前复权+量复权+百分比坐标系**是三大核心。

---

## 一、复权设置

### 前复权 vs 不复权

| 模式 | 均线 | 指标 | 适用场景 |
|:----:|:----:|:----:|:--------|
| **前复权** ✅ | 连续，有效 | 真实 | **趋势/波段/所有技术** |
| 不复权 ❌ | 断开，失真 | 失真 | 超短线（仅看量价） |

> **必须用前复权。** 不复权模式下均线断开、指标失真，所有技术分析无效。

### 成交量也必须复权

```
系统设置 (Ctrl+D) → 设置一
  → "分析图中成交量复权" → 打勾
  → 确定
```

否则主图复权、成交量没复权 → 量价不匹配 → 分析错误。

**确认方法**：页面显示"前复权" + "量复权" = 设置正确。

---

## 二、坐标系设置

### 正确选择

```
点右键 → 主图坐标
  → ✅ 普通坐标系 或 ✅ 百分比坐标系
  → ❌ 不要用对数坐标（会压缩百分比空间）
```

百分比坐标系用于百分比与对偶技术，普通坐标系用于日常看盘。

### 设置三：不要打勾

```
系统设置 → 设置三
  → "主图指标不影响坐标高低值" → ❌ 不要打勾
```

打勾会压缩百分比空间，导致空间测量错误。

---

## 三、设置四：分时图显示涨跌停价格

```
系统设置 → 设置四
  → "分时图显示涨停板价格" → ✅ 打勾
```

这样分时图上会显示涨停价和跌停价，便于交易决策。

---

## 四、指数对比的问题

### 三问法

当别人用股票对比指数时，先问三个问题：

```
① 指数会停牌吗？ → 不会
② 指数会除权吗？ → 不会
③ 指数会送股/分红吗？ → 不会
```

### 时间错配

```
股票停牌（1个月~2年）→ 指数不停牌
  → 期间指数涨/跌，股票未参与
  → 复牌后可能出现与指数走势截然不同
```

**结论**：指数对比只能对比**某一段特定时间**，不能全程对比。

## 五、页面验证（别人分享屏幕时）

```
一眼扫过去确认两点：
  ① 是否前复权？
  ② 量是否复权？
  
两者都复权 = 对的
任一不复权 = 错的 → 技术分析失真
```

## 六、量价时空的软件体现

```
① 价格（K线图） → 空
② 均线（时间周期） → 时
③ 成交量 → 量
④ MACD指标 → 价+动能

软件页面 = 量价时空的完整呈现
```
