# 量价时空（阴阳理论）

> 操盘手技术的哲学基础。将成交量(量)、价格(价)、时间(时)、空间(空)四个要素按阴阳分类，判断股票所处的阶段。

---

## 四要素分类

| 要素 | 阴阳 | 特征 |
|:---:|:----:|------|
| **量**（成交量） | **阳** | 可见，可量化，昨日量推演今日量（最多2倍） |
| **价**（价格） | **阳** | 可见，涨停板/跌停板肉眼可见 |
| **时**（时间/均线） | **阴** | 飘忽不定，无法预知何时停牌/退市 |
| **空**（空间） | **阴** | 涨多高跌多深未知，虚无缥缈 |

> **量+价 = 阳（实）**，**时+空 = 阴（虚）**。阴阳结合才是完整体系。

---

## 阴阳的运用

### 时间（均线）的阴阳

均线方向决定时间的阴阳：
- 均线**向上** = **时间为阳** → 可以入场
- 均线**向下** = **时间为阴** → 不宜入场

### 空间（页面）的阴阳

把股票走势页面用中轴线一分为二：

```
        阳 ↑ （三点钟~十二点钟）
─────── 中轴线 ───────
        阴 ↓ （六点钟~三点钟）
```

### 核心口诀

> **仅顶右上角，创新高，还有新高。**

买股票要买在**三点钟之上的位置**（上升段的早期和中期），不要买在十二点钟（已经涨过头）。

---

## 时空与量价的关系

### 时空 > 量价 = 看涨

时空（虚）凌驾于量价（实）之上。当**时间和空间站稳了量价的位置**，股价将出现质变。

### 脚踩巨量翻倍涨

```
巨量（放量）的位置 = 分歧点（有人买有人卖）
时间（均线）把巨量踩在脚下
  → 分歧变一致
  → 弱转强
  → 股价翻倍
```

**操作**：股价站稳历史巨量对应的最高价时 → 买入。

### 天量见天价

> **口诀：巨量不涨先退场。**

放量是应该涨的。如果放量了但股价涨不动（价格不承接）→ 有人在卖 → 由强转弱 → 先退场。

---

## 量承接 vs 价承接

### 量承接（上涨时）

| 状态 | 含义 | 操作 |
|------|------|------|
| **放量+价格上涨** | 量承接正常，买盘强 | 持股 |
| **放量+价格不涨（滞涨）** | **量不承接**，有人在卖 | ⚠️ 卖出 |

> 上涨时：有卖单（压盘）反而会涨——有对手盘，主力可以往上推。

### 价承接（下跌时）

| 状态 | 含义 | 操作 |
|------|------|------|
| **下跌+无买单（缩量）** | 价格承接，没人接盘 | 跌不动，十字星 |
| **下跌+有托单（放量）** | **价格不承接**，有人接盘 | ⚠️ 还会跌，越托越跌 |

> 下跌时：有托单反而会跌——主力借托单出货。

### 奇妙规律

```
上涨→有卖单→越大越涨（有对手盘可推）
下跌→有托单→越托越跌（主力借机出货）
上涨→无卖单→不涨（无量空涨难持续）
下跌→无托单→不跌（无量十字星）
```

## 选股应用

### 月线选股——量价时空共振

```
步骤① 空间：跌幅够大（有空间）
步骤② 时间：均线横盘走平（时间够久）
步骤③ 价格：由小变大（从不活跃→活跃）
步骤④ 量：超越前方一切量（有资金入场）
步骤⑤ MACD：还没走过大红（有潜力）
```

**案例**：某股月线MACD已经12年没走过大红 → 空间时间都够 → MACD即将走大红 → 选股目标。

> 暴涨过后的股票不要参与——用一年的上涨，可能需要十年消化。
