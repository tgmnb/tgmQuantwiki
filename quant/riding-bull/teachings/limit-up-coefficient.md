# 板后系数（涨停板后抄底系统）

> 成功率95%的超短线系统。涨停板后，在**三个固定价格位**分批买入，冲高卖出。配合周线MACD红柱可接近100%成功率。

---

## 一、三个价格位

### 10%涨跌幅（主板）

| 位置 | 比例 | 仓位 | 作用 |
|:----:|:----:|:----:|:------|
| **一号位** | **7.14%** | 3成 | 赚小钱 |
| **二号位** | **4.3%** | 3成 | 赚大钱 |
| **三号位** | **1.43%** | 4成 | 解套 |

### 20%涨跌幅（科创板/创业板）

| 位置 | 比例 |
|:----:|:----:|
| 一号位 | 14.3% |
| 二号位 | 8.6% |
| 三号位 | 2.86% |

---

## 二、挂单规则

### 价格修正

```
计算出一号位/二号位/三号位价格后：
  ① 避开整数和5、0的关口
     - 避免7.50 → 挂7.51或7.52
     - 避免7.55 → 挂7.56或7.57
  ② 加手续费
     - 股价5元 → 加3-5分
     - 股价7元 → 加5-7分
     - 股价10元 → 加7分
     - 股价更高 → 加1毛
```

---

## 三、九种不能买的涨停板

| # | 类型 | 原因 |
|:-:|:----|:------|
| 1️⃣ | **一字板** | 买不进，成交概率低 |
| 2️⃣ | **T字板** | 同上 |
| 3️⃣ | **跳空高开4个点以上** | 悬空，无承接 |
| 4️⃣ | **MACD顶背离（小红）** | 动能衰竭 |
| 5️⃣ | **144均线向下** | 趋势向下 |
| 6️⃣ | **下跌第一区间的涨停** | 下跌趋势中的反弹 |
| 7️⃣ | **放历史巨量/脉冲量** | 量能异常，出货嫌疑 |
| 8️⃣ | **带长下影线的涨停** | 分歧大 |
| 9️⃣ | **单边上涨尾盘板** | 尾盘偷板，无实力 |

### 加上两个核心规则

| 类型 | 次日开盘 | 操作 |
|:----:|:--------:|:----:|
| **缩量涨停** | 买**低开** | 低开开盘买入 |
| **放量涨停** | 买**高开** | 高开回踩买入 |

---

## 四、成功率提升

### 日线条件

涨停板必须**切断5日线和10日线**（或至少5日线）。

### 周线条件 ⭐⭐⭐⭐⭐

```
周线MACD红柱（不论零轴上还是零轴下）
  + 日线涨停板符合条件
  = 成功率接近100%
```

## 五、可买品种与回避品种

| ✅ 可做 | ❌ 不做 |
|:-------:|:--------:|
| 盘子适中的个股 | **银行**——盘子太大 |
| 券商 | **高速公路**——股性差 |
| 保险 | **黄金**——受消息面影响大 |
| | **石油/高铁**——大盘股 |

---

## 六、时间限制

- **涨停板后9天内**可操作板后系数
- 超过9天 → 不再适用板后系数
- **买第一次回落**（成功率95%）
- 不买第二次回落

### 卖出时机

```
涨停板次日冲高 → 85%的股票在第1-2个30分钟见顶
  → 不创新高 → 卖出
→ 用计算好的仓位，一单即可卖出
```

---

## 七、仓位计算

```
方法：向后数15天
  → 去掉最高成交量、去掉最低成交量
  → 取中间值（阳量和阴量均可）
  → 该中间值 × 150 = 可买股数
  → 股数 × 股价 = 可动用资金
```

这个资金量可以**一单卖出**（在盘中轻松卖出）。
