# 上下影线定位技术

> 用影线K线的最高价和后续某日收盘价画百分比箱体，平移测支撑和压力。影线**放量越大**，箱体越有效。

---

## 一、定义

### 上影线

| 类型 | 取价方式 |
|:----:|:--------:|
| **上影线是阴线** | 取最高价 → **第3天**收盘价画百分比箱体 |
| **上影线是阳线** | 取最高价 → **第2天**收盘价画百分比箱体 |

### 下影线

| 类型 | 取价方式 |
|:----:|:--------:|
| **下影线是红色K线** | 取最低价 → **第3天**收盘价画百分比箱体 |
| **下影线是阴线** | 取最低价 → **第2天**收盘价画百分比箱体 |
| **十字星** | 看对应成交量颜色——量是阴则定义为阴线，量是阳则定义为阳线 |

### 使用场景（下影线）

- **熊市和震荡市**使用
- 要求从高点跌到低点 **跌幅超过25%以上** 最佳
- 必须有**涨停板**出现——资金为王，没资金进来不操作

---

## 二、操作方法

### 画箱体

```
① 找到一个放量的上影线（阴线或阳线）
② 取最高价 + 后续第2/3天收盘价
③ 画百分比箱体（以此两点为箱体上下边界）
```

### 测支撑压力

```
原始箱体 → 向上平移 → 找压力位
         → 向下平移 → 找支撑位
```

### 50%位是关键

箱体平移后，50%位（半分位）往往是强支撑或强压力。

---

## 三、配合体系

| 配合技术 | 作用 |
|:--------:|:----:|
| **三段九格** | 三格为一段，三段九格走完变盘 |
| **数字操盘** | 尾数验证顶底 |
| **阴线平台** | 支撑/压力的交叉验证 |
| **MACD背离** | 确认买卖点 |

> 越近期的影线箱体越准确；不同技术指向同一位置 = 强共振。

---

## 四、实战注意

- 买到影线箱体压力位附近 → 需要震荡整固，很煎熬
- **跳空高开过压力位** → 强势，可突破
- 放量越大，箱体越有效
- 区间越近越准（股价翻倍后旧箱体会失效）
