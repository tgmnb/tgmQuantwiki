# 下横下战法（凯旋战法/对偶技术）

> 源自徐翔创立的超强势股短线战法。专做**下跌→横盘→下跌**结构后的反弹，目标收益20%-50%。核心逻辑：**跌一涨一（对偶），跌多少最终涨回多少。**

---

## 一、前提条件（必须满足）

### 1️⃣ 强势股：3-4个以上连续涨停板

| 类型 | 说明 | 优先级 |
|:----:|------|:------:|
| **换手板** | 有量有成交，实体板 | ⭐⭐⭐⭐⭐ **最佳** |
| **一字板** | 无量涨停 | ⭐⭐⭐ 次之 |

> 涨停板是资金进场信号，没有这个前提 → 后面对偶无效。

### 2️⃣ 完整性

股价前期趋势必须是**上升通道/震荡偏上**。长期下跌的低位票不适用。

---

## 二、标准结构：下跌→横盘→下跌

### 图形

```
① 急跌（3-5根下杀）
② 横盘（窄幅波动，可横成一条线，也可有小反弹）
③ 再下跌（大阴线/跌停板破位）
```

### 第二波下跌的要求

- 必须是大阴线破位（6%以上或跌停板）
- 破位次日 = 买点窗口

---

## 三、MACD确认条件

### 核心：大绿→小绿（先大后小）

```
MACD绿柱：
  第一波下跌：大绿（绿柱放大）
  第二波下跌：小绿（绿柱明显缩短，缩到前期一半以下）
             ↕
  进入第三区间 → 可以做多了
```

### ❌ 不能买的情况

```
先小又大（绿柱缩小后又放大）→ 不能买，下跌动能未释放完
```

### ⚠️ 周线要求

周线MACD为红柱（或即将变红柱）→ 大级别支撑。

---

## 四、买点

### 大资金

跌停板位置直接建仓（因为跌停板才有大筹码）。

### 小资金

破位次日，三种买入方式：

| 方式 | 说明 |
|:----:|------|
| **开盘价买入** | 次日开盘直接买 |
| **盘中4.3/5.7位买入** | 盘中下跌到前一波涨幅的4.3%或5.7%位置 |
| **收盘价买入** | 尾盘2:50确认后买入 |

> 很多时候破位次日直接涨停 → 一定要果断。

---

## 五、卖出：对偶原理

### 核心公式

```
下跌 = 一（基准值）
未来上涨 = 一的 0.5倍 / 1倍 / 1.5倍 / 2倍
```

### 操作流程

```
第一步：测量第一波下跌的幅度（从最高到最低）
  例：跌了32% → 基准一 = 32

第二步：计算卖出关口
  0.5倍 = 16%  （第一压力）
  1.0倍 = 32%  （第二压力）
  1.5倍 = 48%  （第三压力）
  2.0倍 = 64%  （第四压力）
  ...

第三步：观察开盘信号
  涨到某个关口时：
    ✓ 高开 → 强势，可持有观察下一关
    ✗ 低开 → 弱势 → 卖出！
```

### 低开卖出法（最核心的卖出细节）

```
股价涨到对偶目标位附近：
  次日低开 → 不论赚多少 → 卖出
```

---

## 六、三种趋势下的预期

| 趋势 | 类型 | 预期收益 | 特征 |
|:----:|:----:|:--------:|------|
| **上升趋势** | **反转** | 翻倍级（100%） | 最强，可做多轮对偶 |
| **震荡趋势** | **波段** | 30-50% | 中等 |
| **下降趋势** | **反弹** | 10-20% | 最弱，短线快走 |

### 上升趋势的翻倍操作（案例）

```
第一周（买入→卖出）：
  买入：下横下破位次日
  卖出：涨到一×1.5（48%利润）
  时间：约7天

第二周（再次买入→卖出）：
  买入：突破前高后次日开盘
  卖出：涨到一×2（再赚近50%）
  时间：约7天

合计：两周约100%利润
```

---

## 七、完整操作流程

```
① 找强势股：3-4个以上连续换手涨停板
② 等结构：下跌→横盘→下跌（破位大阴线）
③ MACD确认：大绿→小绿（先大后小，第三区间）
④ 周线确认：MACD红柱
⑤ 买入：破位次日（开盘/盘中/收盘）
⑥ 测量基准：跌了多少（记下百分比一）
⑦ 卖出：涨到一×0.5、一×1、一×1.5...
       → 各关口看开盘：低开就卖
```

## 实战案例

### 柘中股份
- 前提：4个连续换手涨停板 → 上升趋势
- 下横下结构完成 → 跌停板破位
- MACD：大绿→小绿B点确认
- 基准一 = 32%
- 卖点：32%(涨停)→48%(低开→卖出，赚48%)
- 突破后再次买入 → 再赚56% → 两周翻倍

### 宝色股份
- 前提：4个涨停板 → 震荡趋势
- 下横下结构 → 小绿成立
- 基准一 = 40%
- 20%位置卖出 → 赚约10%

### 新华都
- 一字板前提 → 下横下结构
- 基准一 = 44%
- 22%→44% → 从-4%到+40%，赚44%

### 廊坊发展 / 利君股份 / 洲际油气
- 均适用此法 → 短期15-24%收益
