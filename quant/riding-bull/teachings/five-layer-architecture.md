# 技术骨架论（五层分析架构）

> **战法是肉，架构是骨。** 单一战法（倍量切、起跑线、日线翻倍、周线翻倍、阴线平台）之所以时灵时不灵，是因为脱离了架构。只有在正确的架构中填入战法，才能奏效。
>
> 本页面是骑牛登泰山整个技术体系的**元框架**—自上而下五层分析，顺序不可颠倒。

---

## 一、为什么需要架构？

### 战法 vs 骨架

```
战法（肉）：
  倍量切 ✓ → 怎么有时好用有时不好用？
  起跑线 ✓ → 为什么这个位置能买，换个位置就不能？
  日线翻倍 ✓ → 买了不涨，是技术错了吗？

骨架（骨）：
  方向 → 均线状态 → 级别+浪型 → 价格 → 量能
  ↓
  把战法填进去 → 每个战法都对上号 → 成功率大幅提升
```

**核心口诀**：架构对了，战法必然对。战法依从于架构，不是架构依从于战法。

### 没有架构的表现

- "这个技术上次好使，这次怎么不好使了？"
- "亏了不敢用，等敢用了又不好使了"
- "学了一堆战法，不知道怎么组合"
- 看票没有层次，全靠感觉

## 二、五层分析架构（自上而下，顺序不可颠倒）

```
第一层：方向（均线方向）
    ↓
第二层：均线状态（发散与收拢）
    ↓
第三层：级别 + MACD浪型（月/周/日）
    ↓
第四层：价格（阳价 vs 阴价 + 八字要诀）
    ↓
第五层：量能（右量过左量 + ABC量）
```

### 第1层：方向（均线方向）

股市只有三个方向：

| 方向 | 特征 | 操作含义 |
|:----:|:----|:---------|
| **下跌** | 144线向下，均线空头排列 | 观望，不参与 |
| **震荡** | 144线走平，股价上下串 | 可操作但级别小，买在下位 |
| **上攻** | 144线向上，均线多头排列 | 重点参与区间 |

**三个级别**的均线行情：

| 行情级别 | 起涨均线 | 行情性质 |
|:--------:|:--------:|:---------|
| **25日线行情** | MA25走平上翘 | 小行情（80%概率） |
| **60日线行情** | MA60走平上翘 | 中级行情 |
| **144日线行情** | MA144走平上翘 | **大行情**（配合浪型可走4-5板） |

**操作铁律**：方向不对不做。144向下时，任何战法都不好使。

---

### 第2层：均线状态（发散与收拢）

均线只有两种影响股价走势的状态，不分涨跌：

| 状态 | 定义 | 含义 |
|:----:|::----|:-----|
| **发散** | 均线间距扩大，5/10/25/60/144分开 | 趋势延续中，**不在发散末端追买** |
| **收拢（粘合）** | 均线间距缩小，两条以上靠拢 | 变盘前兆，**收拢后发散的方向决定行情** |

**均线粘合的三种模式**（任意三条均线粘合均可操作）：

```
模式A：5日 + 10日 + 25日 粘合  → 小级别操作
模式B：5日 + 10日 + 60日 粘合  → 中级操作
模式C：25日 + 60日 + 144日 粘合 → 大行情操作
```

**最佳粘合结构**：大级别均线在下（支撑），小级别均线在上（攻击），或者小级别穿透大级别。

---

### 第3层：级别 + MACD浪型

#### 级别判定

在正确方向 + 收拢状态的基础上，看MACD级别：

| MACD级别 | 对应操作 | 行情级别 |
|:--------:|:--------:|:--------:|
| **月线金叉** | 中长线大行情 | 最大 |
| **周线金叉** | 波段行情 | 中等 |
| **日线金叉** | 短线/反弹 | 最小 |

#### 浪型位置

MACD两种浪型：**三浪和五浪**（简化版只看三浪即可）。

```
浪型规则：
  上水回踩 → 走三高（三浪主升）
  不上水的金叉 → 级别小，谨慎

核心买点：三浪位、五浪位
```

**上水回踩走三高** = MACD上零轴 → 回踩不破 → 走第三浪主升。

#### 八字要诀配合

| MACD柱 | 含义 | 操作 |
|:------:|:----|:----:|
| 大绿 | 下跌主浪 | 观望 |
| 小绿 | 下跌衰竭 | 准备观察买点 |
| 小红 | 反弹级别 | 快进快出 |
| 大红 | 主升浪级别 | 重点持有 |

---

### 第4层：价格（阳价 vs 阴价）

在方向→收拢→级别都对上的前提下，看价格形态：

#### 价格反转条件

```
阳价 > 阴价  →  阳线能够覆盖阴线，价格开始反转
```

具体表现：
- 小阳线、中阳线开始出现
- 带上影线但收盘不弱
- 阴线已不足以吞没阳线
- **一阳穿多线**（一根阳线穿透多条均线）

**关键判断**：价格形态必须和前三层共振。如果均线方向向下（第1层错），再好的阳包阴也没用。

#### 八字要诀嵌套

```
大绿 → 小绿 → 阳价>阴价 → 倍量切/阳包阴 → 买入
                     ↓
              八字要诀判定：
              大绿里的小红 = 反弹，不可反转
              大绿里的大红 = 主升，可以持有
```

**实战示例**：小绿中出倍量切 + 阳价覆盖阴价 + 均线粘合向上 + 月线/周线级别金叉 = 必涨。

---

### 第5层：量能（右量过左量）

最后一层才是量。**不要先看量，先看前四层。**

#### ABC量分析

```
下跌过程：
  左方量（A量）= 套牢盘
  右方量（B量）= 新资金

B量 > A量 → 右量过左量 → 上涨不限量
B量 < A量 → 量能不足 → 需要C量放量才能涨
```

**三个级别**的量能分析（同前四层一样分级）：

| 量能级别 | 含义 |
|:--------:|:-----|
| 月线放量 | 大级别资金进场 |
| 周线放量 | 中级资金进场 |
| 日线放量 | 短线资金进场 |

**连续放量**：月周日同向放量 → 可以连续拉升。无量 → 无法连续。

## 三、完整推演流程（五层走一遍）

### 拿到一只股票，看什么？

```
Step 1: 缩小 → 看月线
  ① 方向对吗？（144向上/走平/向下？）
  ② 粘合了吗？（均线收拢还是发散？）
  ③ MACD级别？（月线金叉？上水了吗？）
  ④ 价格？（有没有阳价覆盖阴价？）
  ⑤ 量能？（右量过左量了吗？）

  月线不对 → 不用浪费时间，放弃

Step 2: 放大 → 看周线
  重复①-⑤，判定周线级别是否支持操作

Step 3: 日线找买点
  用同样的五层法筛选日线，找到符合所有条件的买点
```

### 实战口诀

```
一判方向（144向上是门票）
二看粘合（收拢了才能变盘）
三定级别（月/周/日各有大小）
四审价格（阳盖阴才是反转）
五验量能（右量过左量才起飞）
```

### 随意翻股验证

随便翻一只股票，用五层法走一遍：

```
打开月线：
  → 方向对了吗？（144向上？）
  → 粘合了吗？（5/10/25/60/144间距？）
  → MACD上水了吗？（零轴上金叉？）
  → 价格形态？（大绿小绿？阳覆盖阴？）
  → 量能？（右量>左量？）

打开周线：重复

打开日线：重复，找买点
```

**所有股票都必须符合这个走势**，无一例外。

## 四、大级别改变小级别 vs 小级别改变大级别

### 核心规律

```
大级别向上（月线144向上）：
  → 可以带动周线、日线向上
  → 日线的调整只是回调，不是反转
  ↓
大级别向下（月线144向下）：
  → 必须先改变日线 → 再改变周线 → 才能改变月线
  → 日线的反弹只是反弹，不是反转
```

### 实战意义

| 月线方向 | 日线操作策略 |
|:--------:|:------------|
| 月线向上 | 日线回调=加仓机会 |
| 月线走平 | 日线可以做波段，到阻力位走 |
| 月线向下 | 日线只能做反弹，不反转，50位就走 |

## 五、实战应用示例

### 示例：某股票月线走平粘合

```
第1层：月线144走平 → 方向中性（可操作）
第2层：均线粘合（5/10/25/60靠拢）→ 变盘前兆
第3层：MACD零轴下金叉 → 级别偏弱（上水才行）
第4层：小阳线开始出现 → 价格在企稳
第5层：量还不够大 → 可能需要C量放量

结论：月线级别行情还早，只能做周线级别的波段
```

### 示例：一字板战法也需要架构

一字板开板随便进（板后系数技术）：
- 架构对（方向向上、均线粘合、级别对、价格形态好、量能足）→ **一字板开板才能送钱**
- 架构不对 → 一字板开板就是接盘

## 六、与单一战法的关系

### 蓝色线 = 起跑线
起跑线是架构对了之后，在收拢末端画的那根线。没有架构，起跑线是画不准的。

### 倍量切
倍量切是架构的第4-5层（价格+量）的体现。但必须先过1-3层：
```
这个倍量切：
  → 方向对吗？（第1层）
  → 收拢对吗？（第2层）
  → 级别对吗？浪型对吗？（第3层）
  → 在几浪？是顶背离吗？第几区间？（MACD细节）
```

### 日线翻倍/周线翻倍
日线翻倍的基础是：
- 144走平上翘（第1层方向对）
- 均线收拢（第2层）
- MACD浪型正确（第3层）

## 七、学习路径建议

1. **先学架构**（本篇）— 建立系统性思维的骨架
2. **再学中级** — 方向判断、均线收拢发散
3. **再学高级** — MACD浪型、空间测算、量价配合
4. **最后填战法** — 倍量切、起跑线、板后系数等

> 单一技术就像肉长在骨架外面，一碰就散。架构对了，所有战法自然归位。
