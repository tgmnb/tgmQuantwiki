# 神奇的两点半

> 定价线技术的分支。取**下午2:30的黄色均价线价格**作为未来的支撑位/压力位。适用于大盘和个股，以大盘判断为主。

---

## 核心定义

**适用K线**（必须是放量的）：

| # | K线类型 | 说明 |
|:-:|---------|------|
| 1️⃣ | 放量的红色长上影线 | 上影线越长越好 |
| 2️⃣ | 放量的阴线长上影线 | 阴线长上影也有效 |
| 3️⃣ | 放量的大阳线/中阳线 | 6个点以上或涨停板 |

> **不放量的K线不成立两点半技术。** 放量可以是单根放量或阶段性放量。

### 取价方法

1. 打开该K线当日的**分时图**（切换到普通坐标系）
2. 将十字光标放在**下午2:30**位置
3. 读取**黄色均价线**的价格（不是白色即时线）
4. 这个价格 = 神奇的两点半定价线

### 口诀

> **回踩到神奇两点半，立马翻身向上。**

---

## 使用方法

### 大盘用法

| 动作 | 条件 | 备注 |
|------|------|------|
| **买入** | 价格回踩两点半线 | 正负5个点范围内均可 |
| **卖出** | 价格跌破两点半线 + 反抽确认 | 收盘价跌破才算破 |
| **持有** | 价格在两点半线上方运行 | 不破即趋势延续 |

> 上证指数使用时可取整数关口（如3082→3080），正负5个点都可接受。

### 个股用法

| 动作 | 操作 |
|------|------|
| **买入** | 价格回踩两点半线，加**0.07元**（七分钱）确保成交 |
| **卖出** | 价格到两点半线，减**0.07元**确保成交 |
| **突破买入** | 放量翻上两点半线 + 缩量回踩不破 = 启动信号 |

---

## 核心特性

### 1. 极端走势

两点半位置经常产生**极端走势**：
- 跳空高开（突破）
- 跳空低开（跌破）
- 岛型反转也常产生在两点半位置

### 2. 配合MACD

```
两点半回踩 + MACD第一区间/零轴上金叉
  = 买入信号 ⭐⭐⭐⭐⭐

两点半 + MACD顶背离 + 死亡勾
  = 极端走势（跳空低开） → 卖出
```

### 3. 配合其他技术

- **阴线定价线**：两点半是定价线的分支，需要与阴线平台互相验证
- **数字操盘法则**：配合尾数判断顶底
- **资金箱体/起爆线**：突破起爆线 + 两点半回踩 = 强买点

## 实战案例

### 上证指数

| 日期 | K线类型 | 两点半价格 | 后续走势 |
|------|---------|-----------|---------|
| 中阳线 | 放量中阳 | 3082点 | 多次回踩3082立马翻身向上 |
| 中阳线 | 放量中阳 | 3127(取3125) | 回踩后上涨 |
| 长上影 | 放量长上影 | 3171点 | 4-5次回踩支撑，大幅反弹 |
| 中阳线 | 放量中阳 | 3206点(取3205) | 回踩3205后反转 |
| 大阳线 | 放量大阳 | 3305点 | 回踩支撑后上涨 |

### 风华高科
- 大阳线（9%）→ 两点半价格 14.23元 → 加7分14.30元挂单 → 精准抄底
- MACD三浪配合 → 一波主升浪

### 冠农股份
- 放量长上影 → 两点半 6.28元 → 最高6.30元 → 减7分精准卖在最高点

### 西藏发展
- 阶段放量长上影 → 两点半 15.05元 → 缩量回踩后涨停突破

### 绿景控股
- 放量阳线 → 两点半 9.80元 → 最高价9.80元精准触顶

## 风控设置

可设置**两道风控线**：
1. 第一道：两点半价格（跌破反抽卖出）
2. 第二道：更低位置的另一根两点半价格（无条件止损）

> 脉冲量（单日放量）的K线两点半效果较差，**堆量**（连续放量）效果更好。
