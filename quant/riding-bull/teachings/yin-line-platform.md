# 阴线平台技术（定价线）

> **操盘手只看阴线不看阳线——"看阴不看阳，太阳易上当。"**
> 阴线平台 = 用阴线的开盘价/收盘价画出的压力和支撑线。最核心的定价技术。

---

## 一、阴线平台三大要素

### 1️⃣ 普通阴线平台（最基本的定价线）

| 方向 | 用法 |
|------|------|
| **涨了一波** | 找这波最高的阴线 → 取它的**开盘价** = **压力位** |
| **跌了一波** | 找这波最低的阴线 → 取它的**收盘价** = **支撑位/风控位** |

**买入信号**：
- 回踩不破 → 买点
- 跳空高开 → 买点
- 首次回踩买进，第二次回踩还有反弹

**卖出信号**：
- **跌破** → 无条件卖出
- 跳空低开 → 卖点
- 跌破判定：一半以上K线实体跌破才算破，冒头不算

> 适用于指数和个股。从5178到6124等大级别点位都可用阴线平台验证。

### 2️⃣ 连阴线（连阴线先走远，耐心等待伏击点）

**定义**：连续3根及以上阴线形成的压力区（2根少见，3根为主）。

**最佳形态**：**放量的连阴线**——量越大，效果越明显。

**伏击点（四种）**：
1. **跳空高开**过连阴位 → 买
2. **涨停板**过连阴位 → 买
3. **倍量**过连阴位 → 买
4. MACD **三浪**配合 → 确认

> **第一次摸连阴位不要买**，需要等待回调确认或放量突破。

### 3️⃣ 单日大阴顶

**定义**：单独一根放巨量的阴线——往往预示着大黑马。

操作方法与连阴线类似：必须过了这个位置才算有效。

> **连阴线+单日大阴**是黑马启动的常见前兆。

## 二、A/B/C点对比法（结构判断）

> 核心：下跌横盘下跌 / 上涨横盘上涨 两个标准结构，进行A/B/C点对比。

### 标准结构图

```
下跌 → 横盘 → 下跌（买点结构）
上涨 → 横盘 → 上涨（卖点结构）
```

### A/B/C点对比法

```
A点（左侧参照）→ 股价下跌至低点，放量
B点（右侧对比）→ 股价再次下跌，与A点对比
C点（确认点）   → 小阴小阳时可能出现，大阳大阴时无C
```

### 判定规则

| 现象 | 含义 |
|------|------|
| **放量不跌破**A点 | **强**——主力洗盘不是出货 |
| 放量不跌破 + 次日**跳空高开** | **极强**——反转信号确认 |
| 跳空高开 + 收阳线包掉阴线 | **底分型确认**,反转开始 |
| A点→B点→股价越来越高 | 越走越强，持股 |
| B点量 > A点量 + 不跌破 | 解放套牢盘，为了涨 |

### 过左峰先示弱

股价触及左峰压力位时，先**示弱回踩**再突破是真突破。
- 示弱（回踩）→ 突破 → 买入
- 直接高开冲过去 → 容易回落

## 三、斜横线与平衡线

> 用阴线的**开盘价**画趋势线，辅助判断支撑和压力。

### 三种线型

| 类型 | 画法 | 用途 |
|------|------|------|
| **上斜横线** | 连接两个以上波段顶部阴线的开盘价 | 上行趋势支撑/压力 |
| **下斜横线** | 连接两个以上波段底部阴线的开盘价 | 下降趋势压力/支撑 |
| **平衡线** | 阴线开盘价的水平线 | 水平支撑/压力 |

### 绘制要求

- 至少**三个点共振**（两个画线点 + 后续验证点）才稳定
- 放巨量的阴线开盘价画线意义更大
- 突破一条线后，回踩变成支撑位

### 配合144均线

```
144均线走平上翘 + 突破斜横线 + 回踩不破
    = 三浪启动（绝佳买点）
```

## 四、实战判定流程

### 看一只股票的全流程

```
1️⃣ 看均线趋势（144方向向上）
2️⃣ 看MACD趋势（零轴上二次金叉最佳，周线红柱支撑）
3️⃣ 看K线大小（小阴小阳→大阳出现=开始活跃）
4️⃣ 看成交量（阳量高于左边=有增量资金入场）
5️⃣ 看结构（下跌横盘下跌完成=买点结构）
6️⃣ 看阴线平台（放量不跌破=强，跌破=卖）
```

### 放量不跌破的判断

```
放量阴线出现 → 这么大的量 → 为什么不跌破支撑？
  答案：它不是出货，是为了洗盘。→ 持股或买入
```

## 实战案例

### 上证指数
- 5178点单边回落 → 第一波支撑 = 跌一波最低阴线收盘价
- 阴线平台在周线级验证：涨一波→回踩→突破→反转

### 拉芳家化
- 连阴线位置 → 第一次摸回调 → 倍量突破 → 拉升

### 卓研科技
- 连阴位 + 量承接 → 涨停板突破 → 三浪主升

### 迈克生物
- 连阴线位 → 跳空高开 → 一浪二浪三浪 完美买点

### 银河科技
- 单日大阴 → 大黑马形态 → 跳空高开突破 → 回踩不破
