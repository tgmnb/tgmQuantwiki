# 百分比与对偶（空间密度/顶底相加/异动量战法）

> 操盘手技术的灵魂。股市源代码 = **0.5和1** 的关系。以百分比测量空间，以对偶预判涨跌比例，以**异动量**（移动量）作为画龙点睛的买卖信号。

---

## 一、百分比：顶底相加中间不变

### 原理

股市是一个圆形：0°→180°→360°。每格 = **12%**（12+24+36+...=180）。

### 核心口诀

> **顶底相加，中间不变。**

- **中间不变**：12%、24%、36%...这些百分比数字固定不变
- **顶底相加**：0%（顶）＋ X%（底）= X%，但实际操作中需要灵活判断

### 画法

```
① 点右键 → 主图坐标 → 百分比坐标系
② 鼠标定在起始K线下方，按住左键拖到终点K线右方2-3根
③ 一松手，自动显示0%、12%、24%、36%...等位置
```

### 应用

```
从最高到最低跌了32% → 记下32
  → 16% (0.5) 是第一关注位
  → 32% (1.0) 是第二关注位
  → 48% (1.5) 是第三关注位
```

## 二、对偶：0.5和1的关系

### 核心公式

```
跌了X% → 未来反弹：
  第一目标：X × 0.5（一半）
  第二目标：X × 1.0（等长）
  第三目标：X × 1.5
  第四目标：X × 2.0
```

### 示例

```
上证从3587跌到2440，跌了32%
  → 反弹0.5 = 16% → 第一压力
  → 反弹1.0 = 32% → 第二压力（精准到位）
```

> 对偶比百分比简单，但百分比更精确。

## 三、异动量（移动量）—— 画龙点睛

### 定义

在 **0.5和1的位置** 一定会出现**异动量**（放量）。

### 异动量 vs 脉冲量

| 类型 | 特征 | 后续 |
|:----:|:----|:----:|
| **异动量** ✅ | 第一天放量+**第二天继续放量**（补量） | 正向，可买入 |
| **脉冲量** ❌ | 第一天放量+**第二天缩量** | 负面，不能买 |

### 买入规则

```
① 0.5或1位置出现异动量
② 第一天放量 → 不买（等）
③ 第二天继续放量 → 确认异动量
④ 收盘价突破异动量K线的最高价/收盘价 → 买入
```

### 风控

```
异动量K线的最低价 = 风控位
跌破 → 无条件卖出
```

### 独立使用

> 异动量可以**单独使用**——不需要其他指标配合，本身就是买卖信号。

## 四、K线组合卖出法（脖子/腰/腿肚子）

### 连板后的卖出判定

```
涨停板作为基准：
  ① 骑着头顶收盘（收盘价 ≈ 涨停收盘价） → ✅ 不卖，继续持有
  ② 跌到脖子（3分之1位） → ⚠️ 关注
  ③ 跌到腰（2分之1位） → ❌ 卖出
  ④ 跌到腿肚子（跌破涨停实体） → ❌❌ 必须卖
```

### 核心规则

| 位置 | 收盘状态 | 操作 |
|:----:|:--------|:----:|
| 头顶 | 收在涨停收盘价之上/附近 | 持有 |
| 脖子 | 跌到上半部 | 警惕 |
| **腰** | **跌破2分之1** | **卖出** |
| **腿肚子** | **跌破涨停实体大部分** | **必须卖** |

## 五、实战应用流程

### 一波完整的操作

```
① 从最高到最低画百分比 → 记下跌幅X%
② 计算0.5和1的位置
③ 在0.5和1位置等异动量出现
④ 异动量确认后 → 突破买入
⑤ 风控 = 异动量最低价
⑥ 涨到对偶目标位（0.5/1/1.5） → 观察：
    - 出异动量 → 继续持有
    - 不出 → 卖出
```

### 顶底相加的陷阱

```
跌了32% → 你按32%算目标
  但实际可能顶底相加为36%
  → 如果在32%抄底 → 会再跌4% → 死亡陷阱
  → 必须等异动量出现再买 → 避免陷阱
```
