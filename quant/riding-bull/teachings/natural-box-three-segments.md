# 自然箱体与三段九格（1.3系数箱体）

> 箱体测量体系包含四种：**自然箱体**、**小波段箱体**、**大波段箱体**、**1.3大系数箱体**。以自然箱体和1.3大系数箱体使用最为频繁。

---

## 一、自然箱体（50%放量买点法）

### 核心原理

> **任何一波下跌的2分之1（50%）位置，一定会放量。**

这个放量称为**异动量**，结合MACD的三浪结构，就是买点。

### 买点流程

```
① 找一波下跌的自然高点和自然低点
② 画百分比箱体 → 找到50%位
③ 在50%位附近必定出现放量（异动量）
④ 结合MACD三浪结构 → 三浪位置买入
```

### 50%位放量的三种形式

| 形式 | 说明 |
|:----:|------|
| **放量阳线/阴线** | 典型的异动量，量能明显放大 |
| **跳空高开** | 即使量不大，也算异动量 |
| **倍量** | 放量超过前一日2倍，最强的异动信号 |

### 买点确认

```
50%位放量 → 一浪（不买，不清晰）
          → 二浪回踩
          → 三浪启动 → 买入 ⭐⭐⭐⭐⭐
```

> 三浪比一浪更清晰稳定，是首选买点。

### 测幅（上涨目标位）

```
股价突破箱体高点后：
  将箱体向上平移 →
    第一格（100%位）
    第二格（200%位）
    第三格（300%位）= 三段完成，多为变盘点
```

一般走**三格**，大多数最远三格（最多），三段九格完成多为变盘。

## 二、1.3大系数箱体

### 原理

> 用一波的**最低价 × 1.3** 得出箱体高度，向上平移测量压力。

### 画法

```
最低价 × 1.3 = 箱体高度
     ↓
以最低价为底，最低价×1.3为顶 → 画出一个箱体
     ↓
将箱体向上平移 → 每格高度 = 最低价×1.3
     ↓
三格为一段，三段为九格
```

**示例**：
- 最低价 6.96 × 1.3 = 9.05（箱体高度）
- 箱体：6.96 ~ 9.05
- 平移第一格：9.05 ~ 11.14
- 平移第二格：11.14 ~ 13.23
- 平移第三格：13.23 ~ 15.32（三段完成）

### 用法

| 格数 | 位置 | 说明 |
|:----:|:----:|------|
| 第1-3格 | 前三格 | **最准确**，常用段 |
| 第4-6格 | 中三格 | 准确度下降 |
| 第7-9格 | 后三格 | 压力极大，往往见顶 |

> 前三格最准确，后三格往往形成最终顶部。

### 配合MACD

```
股价运行到某格50%或100%位：
  + MACD出现A→B→C顶背离
  = 卖出信号 ⭐⭐⭐⭐⭐
```

## 三、三段九格体系总览

| 箱体类型 | 基准 | 用途 | 精度 |
|:--------:|:----:|:----:|:----:|
| **自然箱体** | 一波下跌的高→低 | 找50%放量买点 + 测涨幅 | 较准（5%误差） |
| **1.3大系数箱体** | 最低价×1.3 | 测波段压力位 | 波段级，需配合MACD |
| **小波段箱体** | 2日/5日线 | 短线测压 | 精细 |
| **大波段箱体** | 5日/30日线 | 中线测压 | 较粗 |

### 三格为一段 三段为九格

```
第一段（格1-3）：初始拉升段
第二段（格4-6）：中途运行段
第三段（格7-9）：末端冲刺段 → 变盘
```

---

## 四、多重箱体共振

> 不同箱体类型指向同一位置 → 强共振，从不矛盾。

**示例**：
```
自然箱体平移第三格 = 压力位
1.3系数箱体平移第三格 = 同一价位
  + MACD顶背离
  + 数字操盘尾数见顶
  = 无条件卖出 ⭐⭐⭐⭐⭐
```

---

## 五、风控

- **自然箱体买点的风控**：50%位放量阳线的最低价，或一波下跌的底部阴线平台
- 跌破 = 无条件离场
- 风控位置越小越好

---

## 实战案例

### 国光电器
- 一波下跌 → 50%位5.92放量 → 一浪回踩 → 三浪买入 → 箱体向上平移测目标

### 新黄浦
- 50%位5.55放量 → 三浪买入 → 走了1.5格 → DMI+MACD见顶信号卖出

### 乐山电力
- 最低价4.5 × 1.3 = 5.85 → 箱体平移 → 50%位精准测到压力
