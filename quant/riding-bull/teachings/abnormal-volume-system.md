# 异动量与成交量分析体系

> 异动量是操盘手技术里**唯一认可的量能形态**。其他所有量（地量、缩量、增量）都是分支或衍生，只有三种真正的基本量：**倍量、脉冲量、组合量**。
>
> **核心认知**：下跌不需要量，上涨才需要。量是价格拐点处的油门——该踩油门时要踩，不该踩时别乱踩。

---

## 一、异动量的三种基本形态

操盘手技术中，成交量只有一种——**异动量**。它包含三个分支：

### 1. 倍量（最基本）

```
定义：今日成交量 = 昨日成交量 × 2

常态：2-3天上涨
用法：倍量切做超短线，赚2-3天的利润
性质：主动做多信号
```

### 2. 脉冲量（最突兀）

```
定义：在一片平静的量能中，突然放出历史巨量
      像心电图上的脉冲信号

常态：**下跌**（脉冲量常态是跌）
特殊：可能升级为移动量（组合形态）
口诀：脉冲量一出现，就要小心

天量见天价：
  放出天量 → 见了天价回老家
  但是！如果回踩三一不破 → 可升级为黑马/龙头
```

### 3. 组合量/移动量（最可靠）

```
定义：连续放量（2根以上），量能逐步放大
      不是单根脉冲，是持续性放量

性质：组合量 = 移动量 = 最好的异动量
特征：放的量一天比一天高，或者至少持平
      不能是缩量阳

用法：移支量的最低价 = 风控位
      突破移动量的最高价 = 买点
```

### 三者的关系

```
倍量（基础）
  ├── 跳空高开（倍量的分支）
  ├── 涨停板（倍量的分支）
  └── 阶段性放量（倍量的分支）
      ↓
脉冲量（单根巨量，最差） → 可能升级 → 组合量/移动量（最好）
```

## 二、移动量的风控体系（核心）

### 移动量的通用风控规则

```
买进后，以该移动量K线的最低价做风控：
  未跌破 → 持有
  跌破2分之1位 → 减仓
  跌破最低价 → 无条件卖出

如果是组合量（多根）：
  看最后一根放量K线的最低价
  如果最后一根是包含关系（上影/星线）→ 看前一根
```

### 移动量的买卖点

**买入**：
```
① 移动量最低价不破 + 阳包阴 = 买点
② 突破移动量最高价（站稳2分之1位）= 买点
③ 缩倍量回踩移动量不破 = 买点
④ 下跌缩倍量 + 阳包阴过移动量顶 = 买入时机
```

**卖出**：
```
① 跌破移动量最低价 → 无条件卖出
② 跌破2分之1位 → 减仓
③ 移动量被跌破后反抽不过 → 清仓
```

### 移动量风控的逐级推移

```
随着股价上涨，移动量逐步上移：
  每一波放量 → 产生新的移动量
  新的移动量底 = 新的风控位
  旧的移动量自动失效

图示：
  位置A 放量 → 风控A
  位置B 放量 → 风控B（取代A）
  位置C 放量 → 风控C（取代B）
  
股价始终以最新的移动量做风控
```

## 三、异动量的实战应用

### 口诀体系

```
① 出现异动别慌张，涨势回踩看二一
② 跌破破底必须过顶上
③ 缩倍量便是好股，要收藏
④ 量能去解放左边的量，不是价格
⑤ 下跌缩倍量 + 缩量低开是加分项
```

### 应用1：涨势回踩看二一

```
当股价上涨过程中出现异动量：
  涨势回踩 → 看2分之1位
  不破2分之1 → 支撑有效，可介入
  破了2分之1 → 支撑失效，等下一个
```

### 应用2：破底过顶上

```
移动量被跌破（破底）后：
  等待股价反弹 → 突破移动量的最高价（过顶）→ 买入
  过顶那天的最低价 = 新风控位
  
这就是"破底之后过顶上"
```

### 应用3：缩倍量便是好股

```
当股价出现异动量后：
  第二天缩量到一半以下（缩倍量）
  ↓
  缩倍量意味着：
    - 抛压迅速衰竭
    - 主力控盘度高
    - 后面大概率上涨
  ↓
  收藏起来，等阳包阴或过顶上买入
```

### 应用4：右量过左量另类用法（量能突破）

```
一般右量过左量看价格，但还有一种用法：
  右方的量能直接突破了左方大阴线的量能高度
  （即使价格还没到，量能先到了）

含义：
  量能已经突破了左方的套牢盘区域
  未来股价只要站稳这个价格 → 主升浪

案例：双良节能（600481）
  左方天量阴线 → 右方量能突破了这个量的水平线
  → 预示未来会有行情
```

## 四、量价关系比喻（汽车+油门理论）

> 用开车来理解量价关系，最直观

### 基本设定

```
车 = 股价（主体）
油门 = 成交量（动力）
路 = 趋势方向
```

### 三种路况

| 路况 | 行为 | 量价关系 |
|:----:|:----|:---------|
| **上坡**（上涨） | 必须踩油门 | 上涨必须放量 |
| **平路**（震荡） | 轻微给油即可 | 维持量或缩量 |
| **下坡**（下跌） | 不用踩油门 | 下跌不需要量 |

### 关键判断

```
放量 = 遇到阻力
  上涨途中放量 → 遇到抛压，正常
  下跌途中放量 → 遇到支撑，可能要变盘
  平白无故放量（脉冲量） → 此地无银三百两

爆缸信号（放量滞涨）：
  放了很大的量，股价却没涨多少
  = 油门踩到底了车不动
  = 价格不匹配量 → 要掉下来了

下跌不需要量：
  下跌是惯性的，不需要踩油门
  下跌过程中放量 = 遇到阻力了
  遇到阻力后缩量再放量 → 就要拐头
```

### 实践的指导意义

```
量是价格的温度计，不是领先指标
量用来权衡价格所处位置能不能拐头

正确顺序：
  价格 → 量（量跟随价格）
  不是量 → 价格！
```

## 五、瀑布杀（三种形态）

### 定义

```
瀑布杀 = 早盘突然急跌，像瀑布一样下杀
```

### 形态1：技术面没问题的瀑布杀

```
特征：
  周线月线都没问题
  早盘突然瀑布杀
  盘中又收回来（收出下影线）

操作：
  这种瀑布杀不是卖点，是买点
  到了关键支撑位（如三号位）可直接买入

案例：宁波港
  早盘瀑布杀到3.65 → 技术形态没坏 → 买入
```

### 形态2：老鼠仓瀑布杀（最经典）

```
特征：
  盘中出现一个极低价（如跌停价）
  但分时图上**根本看不到这个价格**
  一闪而过，瞬间拉回

判断方法：
  打开分时图看当天的最低点
  如果分时上没有出现过这个价格
  → 老鼠仓瀑布杀 → 未来牛股信号

选股方法：
  编写指标：选择下影线超过7%的股票
  盘中最低价跌到过-7%以下，收盘在-3%以内
  → 筛选出老鼠仓股票

经典案例：
  某股最低价2.66（按前收盘计算跌停价）
  但分时图上从未出现过2.66
  只有9:33一瞬间的自买自卖
  → 未来从3块涨到8块
```

### 形态3：连阴线后瀑布杀

```
特征：
  出现连阴线后
  早盘瀑布杀
  配合移动量/连阴位做风控

关键时间：9:33（神奇九点三十三）
  老鼠仓瀑布杀往往发生在9:33
  跟神奇的两点半类似
```

### 瀑布杀的风控

```
买入后看当天的下影线最低价
  不跌破 → 持有
  跌破了 → 瀑布杀无效，止损
```

## 六、底部尖角波

### 定义

```
尖角波 = 分时图上突然上冲然后回落
          形成尖角形状的波形

底部尖角波 = 发生在底部区域的尖角波
```

### 底部 vs 顶部尖角波

```
底部尖角波（好的）：
  分时上一柱擎天然后回落
  代表主力在底部有单子吃
  预示着后面有行情
  后边配合MACD可买入

顶部尖角波（不好的）：
  顶部出现尖角波
  代表主力在出货
  后面看跌
```

### 尖角波选股

```
条件：
  ① 底部区域出现尖角波
  ② 量能放大（移动量）
  ③ 阳包阴确认
  ④ MACD配合

案例：创益通（300703）
  底部出现多个尖角波
  阳包阴位置买入
  抓到一波40%+的行情
```

## 七、量承接价承接升级版（徐翔看盘要素）

> 2015年4月上海听来的看盘要素。核心：**价格去解放左侧套牢盘**。

### 核心逻辑

```
涨跌自有阴阳：
  阳必须要大于阴（阳线去覆盖阴线）
  用阳线解放左方的阴线套牢盘

用阳解放 vs 用阴解放：
  阳线解放（阳量覆盖阴线全部）= 100%解放 → 强势 → 后边没有压力
  阴线解放（阴量覆盖阴线50%）= 50分解放 → 弱势 → 后边有压力需调整
```

### 50分解放规则

```
当股价上涨到左方阴线位置：
  50分解放 = 阳线打到了阴线的2分之1位
  100分解放 = 阳线完全覆盖了阴线全部

50分解放后：
  上面还有50%的压力
  需要再放量才能完全过

完全解放后（100分解放）：
  该位置没有压力了
  股价可以顺畅上涨
```

### 解放的三种方式

```
过压力位/连阴位的三种方式：
  ① 低开高走 — 经典方式，在盘中解放套牢盘
  ② 跳空高开 — 直接跳过去（但多数会回踩确认）
  ③ 线上阴线/阳包阴 — 回调不破支撑后突破
```

### 量承接与价承接

```
量承接 = 新资金进来接住抛压（半山腰买入法）
价承接 = 价格在支撑位不破（底部买入法）

量承接的特征：
  放量后价格不跌
  新资金进来接住了抛压盘
  半山腰位置买入

价承接的特征：
  股价回到支撑位不破
  缩量找支撑
  底部位置买入
```

### 上影线解套时间

```
分时细节：
  如果某天出现长上影线
  这天上影线部分在高位停留了约40分钟
  = 主力给了40分钟时间给套牢盘解套
  = 这个位置已经没有压力了
```

### 右侧高点判定

```
当右方的高点高于左方的高点 = 看好的积极信号
  高点逐步往上：底部抬高 → 看新高
  高点基本持平：震荡 → 待突破
  高点逐步往下：弱势 → 不碰
```

## 八、综合实战框架

### 步骤1：看趋势
```
144方向向上/走平 → 可以做异动量
144方向向下 → 异动量成功率低
```

### 步骤2：找异动量
```
找到倍量/脉冲量/组合量 → 确定风控位（最低价）
```

### 步骤3：等买点
```
阳包阴不破风控位 → 买入
突破移动量最高价 → 买入
缩倍量回踩不破 → 买入
```

### 步骤4：设风控
```
跌破移动量最低价 → 无条件卖出
跌破2分之1 → 减仓
```

### 步骤5：推新量
```
股价上涨后出现新的异动量 → 风控上移到新量
```
