# 均线收拢与发散（弓理论/填满战法）

> 所有股票的涨跌都经过**收拢**（均线粘合）和**发散**（均线张开）两个阶段。收拢是"拉满弓"，发散是"射箭"。**买在刚刚打开的那一刻。**

---

## 一、核心原理

### 两种走势

```
下跌横盘下跌 → 收拢（均线粘合）→ 发散向下
上涨横盘上涨 → 收拢（均线粘合）→ 发散向上 ✅
```

**横有多长竖有多高**的真相：不是指横盘时间，而是指**收拢的紧密程度**。

### 弓理论

```
均线粘合（5/10/25线拧成一股绳）
  = 弓拉满
  = 蓄势待发

均线张开（发散向上）
  = 箭射出
  = 主升浪开启
```

### 关键均线

| 均线 | 作用 |
|:----:|:------|
| **5日线** | 短线成本 |
| **10日线** | 短线趋势 |
| **25日线** | **第二生命线**——股价必须站上 |
| **144线** | 大趋势方向——必须走平上翘 |

---

## 二、收拢的过程

### 标准流程

```
一浪上涨 → 二浪回调（开始收拢）
  → 反复涨跌，将5/10/25线搓在一起
  → 25线靠近144线
  → 5/10/25线之间缝隙最小化（填满）
  → 同时开口朝上
  → 收拢完成
```

### 填满（弓拉满）

> 要把5日线、10日线、25日线这三条线粘合在一起，使它们之间尽量不要有缝隙。

粘合得越紧 → 力度越大 → 涨幅越大。

### 老鸭头形态

```
横盘很久 → 突然抬头（试压力）
  → 再蹲下（继续收拢）
  → 再涨（正式发散）

这个"抬头再蹲"的动作，就是为了把几条线捋平。
```

---

## 三、发散（买点）

### 买入时机

> **买在刚刚打开过左峰的那一刻。**

不是买在已经发散很久的位置，而是买在收拢完成、**刚刚开始发散**的瞬间。

### 四个买入条件（日线翻倍）

```
144均线走平上翘
股价冲高回落，144正负10%
MACD双线上零轴
过左峰（涨停/跳空/倍量/线上阴线）
```

### 新手选股三原则

```
① 三线红（月/周/日MACD均为红柱） + 开口朝上
② 三线都在水上的（零轴上）最佳
③ 144必须明显走平上翘
```

---

## 四、卖出信号

发散到极致后，均线间距过大 → 回调。

```
均线间距过大（发散过度）
  + 顶背离 / 小红
  = 卖出信号
```
