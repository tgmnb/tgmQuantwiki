# 倍量切（启动K线/正切与反杀）

> 一根中阳/大阳线同时切断5日和10日均线，且当日成交量是前一日2倍——称为**倍量切**。它是一波主升浪的**启动K线**（起爆K线）。但灵魂不在K线本身，而在**均线的发散与收拢**。

---

## 一、倍量切的定义

### 形态条件

| 条件 | 要求 |
|:----:|------|
| **K线** | **中阳线或大阳线**（涨停板最佳），切断5日线和10日线 |
| **成交量** | 当日量是昨日的**2倍以上**（倍量） |

### 命名

```
倍量 = 成交量是昨日2倍
  切 = K线切断5MA和10MA
```

---

## 二、核心灵魂：均线的发散与收拢

### 为什么倍量切如此重要？

> 均线发散 = 市场成本分散 → 鬼招手（不能买）
> 均线收拢 = 市场成本集中 → 即将变盘

```
均线收拢（成本集中）→ 倍量切出现 → 主升浪启动
                        ↓
均线发散 → 鬼招手 → 反杀出现 → 下跌开始
```

### 三种均线格局

| 格局 | 大级别均线位置 | 预期 |
|:----:|:--------------:|:----:|
| **上涨格局** | 年线/半年线/60线**在下方**拖着 | 倍量切=主升浪启动 |
| **震荡格局** | 均线**走平** | 倍量切=波段行情 |
| **下跌格局** | 年线/半年线/60线**在上方**压着 | 反杀=大跌开始 |

### 大级别 vs 小级别

| 级别 | 均线参考 | 行情规模 |
|:----:|:--------:|:--------:|
| **大级别** | 60日线 + 144日线附近 | 大波段/主升浪 |
| **小级别** | 25日线附近 | 小波段 |

> 大级别的倍量切都产生在60线与144线附近。小级别的在25线附近。

---

## 三、正切与反杀

### 正切（上涨信号）

```
阳线切断5MA和10MA + 倍量 + 大级别均线在下方
  = 一波上涨启动
```

### 反杀（下跌信号）

```
阴线（不一定倍量）切断5MA和10MA
  + 大级别均线在上方压着
  = 一波下跌开始
```

> 反杀不需要倍量——只要阴线切断了5/10MA，就是下跌前兆。
> 反杀如果还带倍量 → 杀伤力更大。

### 鬼招手口诀

> **有钱不买鬼招手，买上之后夜夜愁。**

均线发散（五指张开）的状态下，不要买——等收拢后再等倍量切。

---

## 四、倍量切的实战流程

### 上涨买点

```
第一步：检查均线收拢程度
  → 大级别均线（60/144/年线）在脚下
  → 小级别均线（5/10/25）已收拢粘合

第二步：出现倍量切
  → 阳线切断5MA和10MA
  → 成交量是昨日2倍

第三步：确认
  → MACD红柱（正向）
  → 结合MACD三浪结构买入

第四步：目标
  → 自然箱体/1.3系数测目标
  → MACD顶背离卖出
```

### 下跌卖出（反杀卖出）

```
第一步：均线发散（鬼招手）
第二步：出现反杀K线（阴线切断5MA和10MA）
  → 立即卖出/减仓
```

## 五、实战案例

| 股票 | 倍量切位置 | 大均线状态 | 后续 |
|------|:----------:|:----------:|:----:|
| ST继龙 | 倍量切断5/10MA | 年线半年线在下方 | 涨40-50% |
| 金盛机电 | 倍量切+过左峰 | 年线半年线支撑 | 一波主升浪 |

| 阳浦医疗 | 反切+顶背离 | 大均线在上方 | 下跌 |
