# MACD十大技术理论

> 骑牛登山MACD系统的基础框架。十大理论分四组：**三种区间 + 两种浪形 + 两种拐离 + 三种背离**。所有个股和大盘的MACD推演都必须组合运用这十大理论。

---

## 零轴基础

**零轴决定多空方向：**
- 零轴上方 = 多方区，股价强势
- 零轴下方 = 空方区，股价弱势
- **零轴上死叉不可怕**（高位死叉 ≠ 见顶）
- **零轴下金叉并不好**（低位金叉 ≠ 反转）
  - 口诀："底背离之后跌得更深了，顶背离之后涨得更高了"

零轴为"水面"：DIF/DEA 上水 = 进入多头领地，下水 = 进入空头领地。

---

## 一、三种区间（区间理论）

区间 = 根据 MACD 红绿柱与黄白双线的相对位置，判断动能强弱的三级分类。

### 第一区间：强势区间

| 方向 | 条件 |
|------|------|
| 做多第一区间 | 黄白双线在零轴**以上**，**红色柱体穿透了白色线（DIF）和黄色线（DEA）** |
| 做空第一区间 | 黄白双线在零轴**以下**，**绿色柱体穿透了白色线和黄色线** |

> **本质**：动能最强。做多时买进意愿最旺盛，做空时卖出意愿最旺盛。
> 操作：**买点出现在三浪或五浪的第一区间**。

### 第二区间：中势区间

| 方向 | 条件 |
|------|------|
| 做多第二区间 | 红色柱体**离开了白色线（DIF）**，动能开始衰减 |
| 做空第二区间 | 绿色柱体离开了白色线，做空动能衰减 |

> **本质**：动能减弱。如果继续强，柱体应继续穿透白线；脱离了说明买/卖欲望降低。

### 第三区间：弱势区间

| 方向 | 条件 |
|------|------|
| 做多第三区间 | 红色柱体**脱离了黄色线（DEA）**，进入弱势区 |
| 做空第三区间 | 绿色柱体脱离了黄色线 |

> **本质**：强→中→弱的终端。红色脱离黄线 → 股价即将由强转弱，**提示卖出**。
> 第三区间的衰竭预示着卖出信号。

### 关键注意

- 零轴以下的柱体穿透黄白线，**不能称为区间**，只能称为"反弹"
- 只有 **零轴以上**的穿透才算有效做多区间
- 操作窗口通常在第一区间（强势区），第三区间是离场信号

---

## 二、两种浪形（波浪结构）

浪形 = MACD 黄白双线穿越零轴的波浪结构，判断行情处于哪个阶段。

### 浪形一：三浪结构（3浪）

```
零轴下方 → 上水(1浪) → 回踩不破零轴(2浪) → 上升(3浪)
```

- 1浪：从零轴下方首次上水
- 2浪：回踩不破零轴（黄色线 DEA 最好不跌破零轴）
- 3浪：再次上升
- 零轴为分界点，二浪最好不破零轴线

### 浪形二：五浪结构（5浪）

```
摸水 → 回踩 → 上水 → 回踩 → 走高
```

- 1→2→3→4→5
- 四浪最好不破一浪高点
- 一浪、二浪、三浪、四浪、五浪依次展开

### 买点规则

- **买在3浪或5浪的第一区间**
- 看不懂日线看周线，看不懂周线看月线
- 所有股票都走 1-2-3 或 1-2-3-4-5

---

## 三、两种拐离（乖离修复）

拐离 = 黄白双线因开口过大或离零轴过远，必然发生的"回拉修复"行为。

### 拐离一：开口拐离

> **开口大了就要收。**

- 白色线（DIF）和黄色线（DEA）开口过大 → 必然收拢
- 上升途中开口大 → 回勾修复 → 再次张开形成第三次金叉
- 判断大盘或个股都会用到，非常重要

**实战案例**：上证指数60分钟线，开口一大→往上一勾→上去了。

### 拐离二：空间拐离

> **离零轴太远了，失去支撑，需要回拉。**

- 黄白双线远离零轴 + 下方还有柱体 → 空间拐点
- 跌得非常狠时的"一收就会涨"
- 涨得高了后的回勾

**两个拐离的区分**：
- 开口拐离 = 两线之间开口过大 → 收敛
- 空间拐离 = 两线离零轴太远 → 回拉

> 涨得高了回来，千万别在回勾时卖了——**开口拐离≠趋势反转**。

---

## 四、三种背离（背离形态）

背离 = 股价走势与 MACD 指标不一致。

### 背离一：高度背离

**顶背离**：A点股价低 → B点股价创新高，但 B 点的红柱**高度**低于 A 点
- 用10成力量拉高→再用小力量拉创新高→**诱多出货**
- 底背离反之：B点股价创新低，但绿柱高度低于 A 点 → 做空力量衰竭

### 背离二：面积背离

**顶背离**：A点面积大 → B点股价创新高（或与A同高），但红柱**总面积**小于 A 点
- 同价位（同位背离）：同样10块钱，红柱面积应大致相同，面积小了 = 上攻无力

**底背离**：A点面积大 → B点股价创新低，但绿柱总面积小于 A 点

### 背离三：线型背离

**顶背离**：A点白线高 → B点股价创新高，但白色线（DIF）高度低于 A 点
- 通常与高度背离、面积背离**同步出现**
- 三者全部背离时 = **明确的卖出信号**

### ⚠️ 特例：三打白骨精

> K线连创3高，MACD连创3低 → **必暴跌**

形态特征：
- A点（左侧参照）：白色线高度为基准
- B点（1打）：股价高于A，白线低于A
- C点（2打）：股价高于B，白线低于B  
- 股价连创3高，MACD白线连创3低 → 经典三打白骨精

**实战案例**：西藏发展，K线三高+MACD三低 → 后续暴跌。

## 五、实战应用：顶背离卖出法

> 来自另一期教学视频的补充精讲。解决"顶背离看到了但不知道哪一天卖"的问题。

### 判定流程

1. **确立A点**：选定左侧红柱高点作为参照，以**最高价**画水平线向右
2. **看随后的3-4天**：柱体明显缩短 → A点成立
3. **确认信号确立（B点）**：
   - **第一天**：红色K线（阴线不算）的**收盘价 > A点最高价** → 信号第一天
   - **第二天**：红色K线的收盘价**再次** > A点最高价（不是大于第一天的收盘价）→ 信号第二天

### 卖出规则

| 信号 | 卖出概率 | 条件 | 操作 |
|------|---------|------|------|
| ① 第一天（1） | ~60% | 必须是大阳线或涨停板 | 可减半仓或观望 |
| ② 第二天（2） | **~85%** | 任何阳线（不分大小）均卖出 | 全仓清掉 |
| ③ 第三天（3） | 10-15% | 极少出现 | 有更好，没有不强求 |

> **核心规则**：出①且是大阳线→可以不等②直接卖。②来的时候不管什么阳线全部清仓。2:50收盘价确认就全仓清掉——精确到天。

## 总结

| 组 | 内容 | 数量 |
|----|------|------|
| 区间 | 强势区间 / 中势区间 / 弱势区间 | 3 |
| 浪形 | 三浪结构 / 五浪结构 | 2 |
| 拐离 | 开口拐离 / 空间拐离 | 2 |
| 背离 | 高度背离 / 面积背离 / 线型背离（含三打白骨精） | 3 |
| **合计** | | **10** |

> 这十种基础形态每天都会用到，大盘和个股无时不在。
