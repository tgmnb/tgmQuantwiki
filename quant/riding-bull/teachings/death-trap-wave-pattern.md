# 死亡陷阱波浪形态（金山谷/银山谷）

> 一种经典的主力洗盘→拉升的波浪形态。下跌→横盘→急跌（V型反转）→回压→突破。因"老手新手都会死在这个位置"而得名"死亡陷阱"。

---

## 标准图形（上涨版本）

```
① 下跌 ──→ ② 长期横盘 ──→ ③ 突然急跌（挖坑）
                                      ↓
                              死亡陷阱（假摔）
                                      ↓
                         ④ V型反转（急速回拉）
                                      ↓
                         ⑤ 回压（可破/不破前低）
                                      ↓
                         ⑥ 突破 → 买入
```

> **核心特征**：下跌用了很长时间，V型反转只需要**一两根K线**。

---

## 操作流程

### 识别

1. **下跌**：股价从高位下来
2. **横盘**：长时间的横盘震荡（高低点不重要）
3. **急跌挖坑**：突然直线下跌（死亡陷阱）—— 老手新手都死在这里
4. **V型反转**：一口气快速拉回（一根K线）
5. **回压**：再次回调（可以跌破前低，也可以不破）
6. **突破**：突破横盘平台高点 → **买入**

### 买入点

突破横盘平台高点时买入 → 一波主升浪。

### 测幅（自然箱体）

```
从横盘底部到顶部画自然箱体
  → 向上平移，目标位 = 箱体高度
  → 配合数字操盘法则验证顶底
```

## 金山谷/银山谷

死亡陷阱形态也称"金山谷/银山谷"：

- **银山谷**：首次V型反转后的回压低点
- **金山谷**：突破横盘后的主升浪高点

### 洗盘方式

主力洗盘有两种手法：
1. **用时间洗**：长时间横盘消磨耐心，让散户觉得"不行了，换股"
2. **用幅度洗**：急跌挖坑，让散户恐慌割肉

> 散户心理：跌了→解套→赶紧卖→卖完涨停→"不能买了，涨太多"→拍大腿

## 黄金钩（相关波形）

> 黄金钩是死亡陷阱在MACD上的对应形态，属于MACD两种乖离（开口乖离/空间乖离）的一种应用。

**形态**：
```
股价创新高 → 回压不破 → 再次创新高
MACD白线：A点高 → B点低 → C点更低（线型顶背离）
```

- 黄金钩 = 上涨末段的诱多（又多的勾）
- 死亡钩 = 黄金钩之后的反杀（见顶信号）

**判定**：
- 黄金钩在MACD上表现为**线型顶背离**的变种
- 回压不破前低 = 强势；跌破 = 弱势

## 实战案例

### 券商板块
- 下跌→横盘→急速回拉→回压→突破 → 经典死亡陷阱图形

### 天风证券
- 下跌震荡→急跌→一根K线V反→回压→突破

### 合肥城建
- 下跌→横盘→一字板跌停→快速回拉→回压→突破
- 买入后单波盈利约40%

### 京达股份
- 横盘→断崖下跌→V反→回踩→突破 → 3-4个涨停板

### 中船防务
- 下跌断崖→回拉（创新低迷惑）→上行回压→突破

### 国贸股份
- 下跌横盘→破位→上行回压→突破 → 一波主升浪

### 青龙管业
- 长时间下跌→长时间横盘→急跌→瞬间回拉→回压→突破

### 岭南控股
- 慢跌→急跌→上行回压→小山包→突破

## 与八字要诀的关系

死亡陷阱的下跌段对应八字要诀的**下跌横盘下跌**结构：
- 下跌横盘 = 大绿
- 急跌末段 = 小绿 → 买点
- V反初段 = 小红
- 突破 = 大红
