# MACD八字要诀：抓大放小 · 抓小放大

> 源自徐翔口述的MACD波段操作技术。八个字概括股市运作规律：**抓大放小，抓小放大**。

---

## 核心思想

MACD红绿柱体的大小对比，决定买卖时机。

| 口诀 | 含义 |
|------|------|
| **抓大放小** | 买大红的柱体，放弃小红的柱体（小红不能买，小红后面有大绿） |
| **抓小放大** | 买入小绿的位置（或刚开始小红的位置，小绿后面有大红） |

> **红色柱体缩小 = 卖出信号。**

---

## 两个经典图形

### 图形一：上涨横盘上涨 → 抓大放小

```
上涨 → 横盘 → 上涨
        ↓
 大红 → 小红 → 卖出
```

- **大红**：买入区域
- **小红**：不能买，是卖出信号（小红后面有大绿）
- 买点：**高度超过小红的柱体**

配合区间、浪形、背离综合判断。

### 图形二：下跌横盘下跌 → 抓小放大

```
下跌 → 横盘 → 下跌
        ↓
 大绿 → 小绿 → 买入
```

- **小绿**：买入区域（小绿后面有大红）
- **大绿**：观望，不做

---

## 实战判定

### 怎么判断"大"与"小"？

同一波上涨/下跌结构中，对比相邻的柱体高度：
- 同一波中的红柱：如果越来越矮 → 小红 → 卖出
- 同一波中的绿柱：如果越来越矮 → 小绿 → 买入

**跨波对比**：中间隔了一波红/绿后，新的柱体高度明显低于前一波 → 背离。

### 经典买入形态（徐翔口述）

```
4+涨停板 → 急速下跌 → 跌停板第二天买入
```

条件：
1. 前面有 **4个以上的涨停板**
2. 涨停板过后急速下跌
3. 在**跌停板第二天**的下跌位置买入（追涨停板模式）
4. MACD呈**底背离**图形

### 经典卖出形态

```
大红 → 小红 → 卖出
```

同一结构中，红柱由大转小 → 无论什么位置 → 卖出。

## 死亡陷阱

> 下跌横盘创新低 → 实为假摔 → 散户被洗出 → 主力快速拉升

**特征**：
- 长时间下跌横盘
- 破位创新低（散户死亡陷阱）
- 极短时间内快速拉升回前高以上
- 可视为黄金坑/金山谷突破形态

**判定**：突破买入，不突破不是买点。

## 实战案例（转录原文要点）

### 哈投股份
下跌横盘创新低 → 死亡陷阱 → 两根K线全部解封获利 → 还没涨完（还有新高）

### 中国电信
下跌横盘 → 打掉底 → 快速拉升（短时间）→ 黄金坑+金山谷+银山谷

### 东方精工
大绿小绿 → 小红大红 → 小红不能买 → 下跌横盘破位 → 大绿小绿 → 刹车换挡 → 一字板 → 涨到自然箱体二一位

### 三角轮胎
小红不下水 → 底部不破高抬高 → 突破买入 → 顶背离不破底买进

### 艾迪尔
下跌横盘下跌 → 刹车 → 换挡不破底 → 突破 → 上涨
