# Uncovering Symmetry Transfer in Large Language Models via Layer-Peeled Optimization

**ArXiv ID:** [2605.12756v1](https://arxiv.org/abs/2605.12756v1)
**PDF:** [https://arxiv.org/pdf/2605.12756v1](https://arxiv.org/pdf/2605.12756v1)
**Published:** 2026-05-12
**Categories:** math.OC, cs.AI, stat.ML, math.OC
**Authors:** 

## Abstract

Large language models (LLMs) are pretrained by minimizing the cross-entropy loss for next-token prediction. In this paper, we study whether this optimization strategy can induce geometric structure in the learned model weights and context embeddings. We approach this problem by analyzing a constrained layer-peeled optimization program, which serves as a mathematically tractable surrogate for LLMs by treating the output projection matrix and last-layer context embeddings as optimization variables
