# Learning Agent-Compatible Context Management for Long-Horizon Tasks

**ArXiv ID:** [2605.30785v1](https://arxiv.org/abs/2605.30785v1)
**PDF:** [https://arxiv.org/pdf/2605.30785v1](https://arxiv.org/pdf/2605.30785v1)
**Published:** 2026-05-29
**Categories:** cs.AI, cs.AI
**Authors:** 

## Abstract

LLM agents increasingly face long-horizon tasks such as web search and deep research in real-world applications, where accumulated context can cause long-context degradation and reasoning failures. Prior work mitigates this through context management with agent-side context control or fixed strategies such as summarization, which require training the agent itself for adaptation - making it impractical for closed-source agents and ignoring that different agents may require different strategies.
