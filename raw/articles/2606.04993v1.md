# Code Lifespan Survival Analysis (CLSA): Predicting the Survival of Source Code Lines Using AST-Aware Mining

**ArXiv ID:** [2606.04993v1](https://arxiv.org/abs/2606.04993v1)
**PDF:** [https://arxiv.org/pdf/2606.04993v1](https://arxiv.org/pdf/2606.04993v1)
**Published:** 2026-06-03
**Categories:** cs.SE, cs.SE
**Authors:** 

## Abstract

Context: Predicting which source lines will be deleted - and when - matters for maintenance, technical debt, and review prioritization. Existing MSR approaches work at file or method granularity, masking individual-statement risk. Objective: We introduce Code Lifespan Survival Analysis (CLSA), the first framework to model code survival at individual-line granularity. CLSA treats each line as a right-censored subject and estimates deletion risk from structural, contextual, and temporal covariates
