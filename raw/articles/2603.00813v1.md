## Abstract

De Finetti's optimal reinsurance is a set of contracts, one for each risk in a portfolio, that caps the retained aggregate variance to a pre-specified level while minimizing total expected loss. The premiums are determined using the expected value principle, and the safety loading is allowed to vary with the risks.
