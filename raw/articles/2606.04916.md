# Worker Utility as Hysteresis: A Preisach Model of Transaction Acceptance in Gig Labour Markets

**ArXiv ID:** [2606.04916](https://arxiv.org/abs/2606.04916)
**PDF:** [https://arxiv.org/pdf/2606.04916](https://arxiv.org/pdf/2606.04916)
**Published:** 2026-06-04
**Categories:** Machine Learning (cs.LG)
**Authors:** Piotr Frydrych

## Abstract

Worker utility is not observed -- only its consequence is. Each gig transaction produces a single bit: accepted or rejected. We argue this structure points directly to the Preisach hysteresis model as the natural representation of latent worker preferences. The Preisach operator models aggregate output as an integral over a population of binary threshold elements -- precisely the structure that emerges when heterogeneous workers each carry a private acceptance wage.
