# MACD 民间方言与实战战法

> **说明**：本页面收集中文交易社区（B站/股吧/同花顺等）流传的 MACD "方言"用法，即非标准教科书、民间实战经验总结的战法。部分用法相互矛盾，优劣需自行验证。

---

## 一、水上水下判断法

**来源**：股吧/散户圈最广为流传的基础用法

### 核心规则
| 区域 | MACD 位置 | 含义 | 操作 |
|------|-----------|------|------|
| **水上** | DIF 和 DEA 都在 0 轴以上 | 多头市场 | 只做多，不做空 |
| **水下** | DIF 和 DEA 都在 0 轴以下 | 空头市场 | 只做空，不做多 |
| **水面** | 在 0 轴附近黏合 | 方向不明 | 观望 |

### 方言补充
- **"水上金叉买"**：水上金叉（MACD 线在 0 轴以上金叉）是强势买入信号
- **"水下死叉抛"**：水下死叉是弱势卖出信号，应离场
- **"水上一点绿"**：在水上区域出现小绿柱（调整），是加仓机会而非卖点

## 二、"买小卖小，缩头缩脚"（最重要方言）

**来源**：B站多个 10万+ 播放量视频的核心口诀，被认为是 MACD 柱状图的精华

### 八字真言解释

**"买小"** = 买**小红柱**（绿转红的第一根小柱体）
- 行情大幅下跌后，MACD 绿柱缩短但还未金叉时，第一根明显缩短的**小红柱**是抢反弹买点
- 之后可能出现"缩头"（红柱又缩小），这是"卖小"

**"卖小"** = 见**小绿柱**就卖
- 行情上涨一段后，第一根开始缩短的小红柱（还没变绿），就先卖出一部分
- 这叫"缩头"——红柱缩小

**"缩头"** = 红柱从大变小
- 上涨中红柱开始缩小，预示短期调整，卖出信号

**"缩脚"** = 绿柱从大变小
- 下跌中绿柱开始缩小，预示反弹可能，买入信号

### 操作口诀
```
绿转红第一根 → 买（小绿柱变短的开始）
红转绿第一根 → 卖（小红柱变短的开始）
```

### 核心逻辑
- 不等金叉死叉，在柱体**刚刚收缩**时就行动
- 优点：比金叉/死叉**提前 1-3 根 K 线**预警
- 缺点：震荡市中假信号多，容易被反复"打脸"

### 实战要领
1. 区分大势：上升趋势中的"缩头"只是减仓，趋势不破不走
2. 绿柱缩小后再次放大（不缩到零）：说明趋势极强，是"空中加油"
3. 连续出现小绿柱却不翻红：说明是阴跌，不抄底

## 三、空中加油（强势续涨形态）

**来源**：股吧/散户圈广泛流传的强势股战法

### 形态特征
```
上涨 → MACD 水上死叉（调整）→ 绿柱很小/迅速翻红 → 继续上涨
```

### 两种类型

**标准空中加油**：
- 上涨一段后在水面附近死叉
- 绿柱面积小且时间短（2-3 根）
- 快速二次金叉，价格不破关键均线

**变种——"拒绝死叉"**：
- 水上死叉后绿柱极短
- 拒绝下穿 0 轴
- 第二天直接阳包阴

### 识别要点
| 条件 | 说明 |
|------|------|
| 前一波涨幅 | 至少 10-15% 以上 |
| 调整幅度 | DIF 回调不超过前波涨幅的 30% |
| 绿柱面积 | 明显小于前一波红柱面积 |
| 成交量 | 调整时缩量，再启动时放量 |

---

## 四、拒绝金叉 / 拒绝死叉

**来源**：与"空中加油"配套，被认为是主力行为的体现

### 拒绝死叉（看涨）
- MACD 水上死叉后：**绿柱极短**，几乎看不到就重新金叉
- 第二天阳线拉升，是加速上涨的前兆
- **主力控盘**：故意不死透，测试抛压后快速拉升

### 拒绝金叉（看跌）
- MACD 水下金叉后：**红柱极短**，很快又死叉
- 是弱势反弹出货的标志
- 遇到这种形态应**快速离场**

### 方言口诀
> "拒绝死叉买，拒绝金叉卖"
> "金叉不死，退步抽身"

## 五、三板斧（MACD 短线战法）

**来源**：裸 K 方程式等多个 B站 UP 主（数百万播放量的教程）

### 三板斧定义（不同流派说法不一）

**版本一（短线三板斧）**：
1. **第一斧**：MACD 0 轴附近金叉 → 买入
2. **第二斧**：回踩 0 轴不破 → 加仓
3. **第三斧**：顶背离 → 离场

**版本二（MACD+均线三板斧）**：
1. DIF 和 DEA 在 0 轴上黏合（蓄力）
2. 放量拉升，DIF 向上发散
3. DIF 和 DEA 远离 0 轴，顶背离时离场

### 实战要点
- 适用于**日线级别短线**（持有多日至 2 周）
- 需要配合成交量：拉升时放量，调整时缩量
- 止损设在 MACD 水下死叉的位置

## 六、背离的民间细化（四种隐藏背离）

**来源**：B站"江浙陈某"——"MACD 高进阶，背离有四种分类，隐藏类背离操盘手掌握的也不多"（2.9万播放）

### 标准背离（常规背离）
| 类型 | 价格 | MACD | 信号 |
|------|------|------|------|
| 顶背离 | 创新高 | 高点低于前高 | 卖出 |
| 底背离 | 创新低 | 低点高于前低 | 买入 |

### 三种隐藏背离

**1. 隐藏顶背离**
- 价格创新高，但 MACD 高点**低于前高**（正常顶背离）
- 隐藏版：价格创新高，但 MACD 的**红柱面积**小于前高
- 信号更强：说明虽然价格涨，但动量已经在萎缩

**2. 隐藏底背离**
- 价格创新低，但 MACD 低点**高于前低**
- 隐藏版：价格创新低，但 MACD 的**绿柱面积**小于前低
- 抄底成功率更高

**3. 交叉背离**
- 价格创新高/低，但 MACD 的**交叉点**低于/高于前一个交叉点
- 发生在震荡行情中

### 方言："背离还背离的破解"
- 第一次背离可能失败（反弹一下继续原趋势）
- 第二次背离：成功率大幅提升
- 第三次背离：几乎确定反转

## 七、红绿柱面积相关民间用法

> 参见：[[macd-area-and-zero-return]]

**补充方言观察**：

- **"红肥绿瘦"**：上涨中红柱明显大于绿柱（多方强）
- **"绿肥红瘦"**：下跌中绿柱明显大于红柱（空方强）
- **"台阶式红柱"**：红柱一波比一波高 → 趋势延续；红柱一波比一波矮 → 趋势减弱
- **"拒绝缩到零"**：趋势极强时柱体收缩但不穿越零轴，然后继续放大

## 十、周期共振战法

**来源**：多个 B站视频强调（"MACD 周期共振结合三买精确买点" 1万播放）

### 核心逻辑
- **周线定方向**：周线 MACD 水上/水下决定大方向
- **日线找买点**：日线 MACD 金叉在周线方向内是"二买/三买"
- **30 分钟/60 分钟找精确入场点**

### 操作步骤
1. 周线 MACD 在 0 轴上方 → 只做多
2. 日线 MACD 回调到 0 轴附近再次金叉 → 二买
3. 30 分钟 MACD 金叉确认 → 精确入场

### 方言："MACD 周线战法，顶配版"（7200+ 播放）
- 专门做周线级别 MACD 的二次金叉/背离
- 持仓周期：数周到数月

## 十一、分时图 MACD 用法

**来源**：B站"一指连阳"的 MACD 分时图用法（1200+ 播放）

### 特点
- 用于**日内 T+0 操作**
- 在分时图上叠加 MACD（参数需调小，如 5, 20, 4）
- 利用 MACD 的"缩头缩脚"判断盘中高低点

### 操作
- 分时 MACD 出现金叉 → 买入
- 快速出现死叉 → 卖出
- 特别适合**ETF、指数基金**的日内波段

## 风险提示

1. **方言用法未经严格回测**：多数来自经验总结，未做系统性数据验证
2. **参数敏感性**：非标准参数的 MACD 效果因市场和品种而异
3. **滞后性本质**：所有 MACD 用法都存在滞后，无法预测突发性行情
4. **量化验证建议**：用 Python 对感兴趣的战法做历史回测（如 [[macd-area-and-zero-return]] 中的方法）
