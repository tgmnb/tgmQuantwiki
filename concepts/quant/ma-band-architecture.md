# 自适应均线防守带策略架构

## 概述

自适应均线防守带策略是一个量化交易系统，通过一组移动平均线（MA）构成的网格来识别有效的价格支撑/阻力区域。相邻的均线组成"防守带"——当价格回调到较短均线（左侧均线）时触发入场信号；如果价格跌破较长均线（右侧均线），则认为防守失败并离场。

**核心创新**在于**自适应均线带选择**：哪些MA配对在当前市场最有效是随时间变化的。系统通过历史触碰/穿透统计持续重新评估均线带的有效性。

## 核心概念

### MA网格生成

```python
periods = [5, 6, 7, 8, 9, 10, 11, 13, 15, 17, 19, 21, 24, ...]
# start=5, ratio=1.10, ceil取整, max=720
```

**原理**：对数间距在短周期提供细粒度（均线带间距窄、信号频繁），在长周期提供粗粒度（趋势信号更显著）。

### 防守带定义

一个防守带由两个相邻MA周期定义：
- `left_ma`：较短周期（入场触发线）
- `right_ma`：较长周期（止损线）

N个MA周期产生N-1个防守带。

### 触碰与穿透事件

**做多方向（回调入场）：**
- 触碰：`前一根收盘 > left_ma[t-1] AND 当前最低 <= left_ma[t]`
- 穿透：`当前最低 <= right_ma[t]`（触碰的同时）

**做空方向（反弹入场）：**
- 触碰：`前一根收盘 < left_ma[t-1] AND 当前最高 >= left_ma[t]`
- 穿透：`当前最高 >= right_ma[t]`（触碰的同时）

### 穿透比例与断点强度

```
穿透比例 = 穿透次数 / 触碰次数
断点强度[i] = 穿透比例[i-1] - 穿透比例[i]
```

`断点强度`越大，说明从前一个带到当前带，穿透概率出现明显下降——当前带是更有效的防守带。

### 三层选择系统

| 层级 | 更新频率 | 数据范围 | 目的 |
|------|---------|---------|------|
| 全量统计 | 每月/每周 | 当前时间点之前全部数据 | 计算所有带的累计触碰/穿透 |
| 长期池 (Top 20) | 每周 | 全量统计 | 按断点强度或MFE评分选最优带 |
| 短期池 (Top 5) | 每天 | 长期池内，限定近期窗口 | 选出当前实际使用的交易带 |

**选择器演进：** 见 [[ma-band-selection]]

### 入场逻辑

做多回调入场需满足：
1. 趋势过滤：最长3根MA必须多头排列 (MA_C > MA_B > MA_A)
2. 回调检测：`前收盘 > left_ma[t-1] AND 最低 <= left_ma[t]`
3. 右侧MA斜率：`right_ma[t] > right_ma[t-1]`（MA必须上行）
4. 回调深度限制（可选）：`(left_ma - low) / close < max_pullback_depth_pct`
5. ATR门槛（可选）：波动率须低于阈值分位数
6. 斜率过滤（可选）：右侧MA须连续N根上行
7. 拥堵过滤（可选）：窗口内最多N个信号
8. 最小止损距离：`abs(入场价 - 止损价) / 入场价 >= min_stop_distance_pct`

### 离场逻辑

| 离场类型 | 触发条件 | 执行方式 |
|---------|---------|---------|
| 止损 | `最低 <= 止损价`（做多）/ `最高 >= 止损价`（做空） | 下一根开盘价/5分钟均价 |
| 时间离场 | 持仓 ≥ 5根K线 AND `收盘 <= 入场价`（做多） | 下一根开盘价 |
| 均线带反转 | `left_ma <= right_ma`（做多）/ `left_ma >= right_ma`（做空） | 下一根开盘价 |
| 数据结束 | 到达最后一根K线 | 按最后价格平仓 |

### 移动止损

当价格站上新的有效均线带：
- 做多：`收盘[t-1] <= new_left_ma[t-1] AND 收盘[t] > new_left_ma[t] AND new_right_ma[t] > 当前止损`
- 止损**上移**到 `new_right_ma[t]`（永不下降）
- 做空：镜像逻辑，止损**下移**

止损类型：`ma_left`, `ma_right`, `trail_pct`, `trail_atr`, `trail_fixed`

### 仓位计算

```
单位风险 = abs(入场价 - 止损价)
风险金额 = 权益 × 风险比例
数量 = 风险金额 / 单位风险
名义价值 = 数量 × 入场价
if 名义价值 / 权益 > 最大杠杆: 封顶到最大杠杆
```

### 成本模型

单边费率：`0.08%`（可配置）
- 入场成本：`数量 × 入场价 × 费率`
- 离场成本：`数量 × 离场价 × 费率`
- 总计往返：`2 × 费率 = 0.16%`
