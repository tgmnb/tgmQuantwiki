# 大A选股回测框架 (select-stock-pro)

> 邢不行-策略分享会（26分享会）官方选股回测框架，用于A股量化选股因子/策略的研究、回测与参数优化。
> 框架路径：`/mnt/c/quant/select-stock-pro_v2.3.3/`
> 原始教程：[[quant/bbs/股票/教程/【26分享会小白攻略】大A选股回测框架V2.X及网页版使用指南]]

## 核心原则

- **只改配置和因子，不动框架代码**
- 环境不支持就改环境，不许动框架

```

│   └── config.py          ← 网页版 config（葫芦ID/apiKey/数据路径）
```

## Config 参数详解

### 回测配置

| 参数 | 类型 | 说明 |
|------|------|------|
| `start_date` | str | 回测开始日期，"YYYY-MM-DD" |
| `end_date` | str 或 None | None 表示最新数据日期 |
| `performance_mode` | "ECO"/"BAL"/"MAX" | ECO 最稳但慢，MAX 快但吃内存 |
| `real_backtest_mode` | bool | True=真实模式（停牌/涨停/跌停模拟） |

### 策略配置

```python
strategy_list = [
    {
        "name": "策略名",            # 对应策略库中的文件名
        "hold_period": "3D",         # 持仓周期
        "offset_list": [0,1,2,3,4],  # offset 列表
        "select_num": 10,            # 选股数量（int 或 float 比例）
        "cap_weight": 1,             # 多策略资金权重
        "rebalance_time": "open",    # 换仓时机
        "factor_list": [...],        # 选股因子列表
        "filter_list": [...],        # 过滤因子列表
    }
]
```

### 持仓周期

| 级别 | 可选值 |
|------|--------|
| 日级别 | 2D, 3D, 4D, 5D, 6D, 7D, 8D, 9D, 10D |
| 周级别 | W, 2W, 3W, 4W, 5W, 6W, W53 |
| 月级别 | M |

最常见：`3D`、`5D`、`W`

### offset_list

offset 数量与持仓周期绑定（如 W 有 5 个 offset [0-4]）。
- 小资金：选择其中 1-3 个
- 大资金：全选实现 **offset 全覆盖**
- 写法：`[0, 1, 2, 3, 4]` 或 `list(range(0, 5, 1))`

### 换仓时机 (rebalance_time)

| 模式 | 值 | 说明 |
|------|-----|------|
| 隔日换仓 | `'close-open'` | offset结束日收盘卖出 → offset开始日开盘买入 |
| 开盘日内换仓 | `'open'` | 当天开盘先卖后买（**推荐**，同股不重复买卖） |
| 定时换仓 | `'1000-1000'` | 按 5 分钟粒度指定换仓时间 |
| 尾盘 | `'close'` | 不建议使用 |

> 基础策略建议用 `'open'`。若上下 offset 选中同一只股票，日内换仓不会重复买卖，节省滑点+手续费。

### 选股因子 (factor_list)

```python
('因子名称', True, None, 1)
```

| 字段 | 说明 |
|------|------|
| 第一位 | 因子名称，必须与因子库文件中一致 |
| 第二位 | 排序方式，True=从小到大，False=从大到小 |
| 第三位 | 因子参数，None=无参数 |
| 第四位 | 权重，多因子复合时用于计算排名权重 |

### 过滤因子 (filter_list)

```python
('成交额Mean', 5, 'pct:>=0.05', True)
```

| 字段 | 说明 |
|------|------|
| 第一位 | 因子名称 |
| 第二位 | 因子参数（如 5=过去5天均值） |
| 第三位 | 过滤规则：`pct:>=0.05`=保留95%尾部剔除5%；`rank:>1`=保留排名>1；`val:>1_0000_0000`=保留值>1亿 |
| 第四位 | 排序方式 |

### 分批进场目标仓位

建议用默认值，2026年新策略再进一步介绍。

### 选股数量

- **整数**：如 10 就是选 10 只
- **小数**：如 0.1 就是全市场 10%（≈500 只）

小资金起步选 1-3 只，大资金 5 只起步，30 只也不多。

### 板块过滤

支持过滤创业板、科创板、北交所。默认过滤北交所。根据自己证券交易权限调整。

```python
days_listed = 250     # 拉黑上市不足250天的新股
total_cap_usage = 1   # 仓位 1=满仓，50/100=半仓
```

### 模拟交易配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `initial_cash` | 10万 | 初始资金。太小可能买不起高价股（100股起），建议按实际实盘资金设置 |
| `c_rate` | 1.2/1000 | 手续费（双向），建议设高一些包含滑点 |
| `t_rate` | 印花税率 | 卖出时征收（北交所更高） |
| `stock_timing_order_price` | 5（分钟） | 择时后模拟成交时间偏移 |

### 凌烟阁个股择时

```python
stock_timing_list = [
    {
        "name": "个股择时_布林带",     # 信号库中的文件名
        "params": {...},               # 阈值参数
        "factor_list": [...],          # 依赖的因子列表
        "weight": 1,                   # 多择时策略权重
        "period": "1H",               # 择时计算周期
    }
]
```

**凌烟阁1.2（V2.4.0+）** 新增收盘价替代机制，防止休市前后收盘价与开盘价价差过大导致择时偏差：

| 模式 | 值 | 说明 |
|------|-----|------|
| 模式1 | `'close'` | 保持收盘价（原始） |
| 模式2 | `'next_open'` | 每小时收盘价用下小时开盘价替代 |
| 模式3 | `'auto'` | 仅休市时用下小时开盘价替代 |

> ⚠️ 凌烟阁1.2截至2026-05-11仅支持回测，不支持实盘。

## 回测流程

### 一键回测（推荐）

直接跑 `回测主程序.py`，自动完成全部 5 步：

1. 整理数据 → 2. 计算因子 → 3. 选股 → 3.5 个股择时 → 4. 实盘模拟

输出：资金曲线图（蓝=策略、橙=沪深300、绿=中证500、黄=回撤）。

### 分步回测

在数据和因子不变的情况下，只改选股数量/板块过滤/初始资金/手续费等参数，可只跑后三步（step3→step3.5→step4），节省时间。

各步骤均支持 `boost = True`（并行加速），性能不够报错则改为 `boost = False`（串行）。

## 网页版使用

网页版入口：`web/main.py`（Flask 应用），右键运行后点击链接打开浏览器。

**5步走**：启动网页版 → 导入策略包 → 设置参数 → 一键回测 → 分析结果

**策略管理**：
- 支持创建/复制/删除/导出 config 配置
- 支持策略组合（多策略按权重分配资金）
- 支持目录分类管理 config
- 支持保存自研模板（`web/data/cfg/`）
- "启用"按钮将网页版配置覆盖到框架 `config.py`

**个股择时配置**：
- 启用后可在网页版配置择时策略名称/参数/权重/周期
- 可新增多条择时策略自由组合

## 框架使用原则

1. **不动框架代码**：配置文件 (config.py) 和因子文件可以改，core/program/tools 等核心代码不许动
2. **环境优先适配**：环境不支持就改环境（装包/换Python版本），不改框架
3. **Git 管理版本**：框架版本用 Git 管，配置版本运行时 copy 进结果目录
4. **回测资金建议用实际实盘资金**，避免资金规模不匹配导致部分空仓
